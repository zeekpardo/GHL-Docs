Boosting sales and attracting potential customers to your offers can be enhanced with the use of strategic discounts. One effective method to provide these discounts is through coupons. Here's a step-by-step guide on how to set up and use coupons in memberships within .

## **Key Features:**

Exclusive Offers and Discounts: Empower users with the ability to access special deals and price reductions during the checkout process.

Versatility: Coupons can be integrated into various sections, be it regular offers, one-click offers, or in-app promotions.  

**_Please Note:_**_Coupons will only work for Version 2 Offers that aren’t Free Offers._

## **Setting Up the Coupons Feature:**

### **Enabling Coupons:** 


Navigate to Memberships on the left > Courses at the top > From the dropdown select Offers:  

![](https://assets2.modalsupport.net/1770281908589-image6.png)


Edit your offer and find the "Allow checkouts to access coupons" section under the Price category. Flip the switch to activate the coupon option for that offer.  

![](<https://assets2.modalsupport.net/1763755152615-Captura de pantalla 2025-11-21 145905.png>)

### **Visual Confirmation:**


Post-activation, previewing the offer will now display an additional input field prompting users to enter their coupon code.

![](<https://assets2.modalsupport.net/1763755186201-Captura de pantalla 2025-11-21 145938.png>)

##   
**Creating a New Coupon:**

-   Go to payments, and then to the coupons section.

![](<https://assets2.modalsupport.net/1763755204979-Captura de pantalla 2025-11-21 145959.png>)

-   Click on "Create coupon".
-   Label your coupon and assign it a memorable code (e.g., SUMMER20).
-   Choose between a percentage discount or a fixed amount.
-   Set the coupon's start date. While an end date is optional, it can be useful for limited-time promotions.

![](https://assets2.modalsupport.net/1770281940487-image4.png)

There are additional customization options:

-   Limit the total number of times a coupon can be redeemed.
-   Restrict the coupon to certain products/offers only.
-   Limit to one use per customer
-   Also apply to recurring/future payments, if applicable

##   
**Redeeming the Coupon:**

Users will find the coupon redemption process straightforward:

As they reach the checkout page, an "Enter your coupon code" field will be present.  Upon entering the valid code and clicking 'apply', the respective discount will be subtracted from the total payable amount.

![](<https://assets2.modalsupport.net/1763755351126-Captura de pantalla 2025-11-21 150223.png>)

## **FAQs**




**Question:** How do I enable the coupon feature for memberships?
**Answer:** To enable the coupons feature, go to Memberships > Courses > Offers, edit your offer, and find the "Allow coupons in offer" section under the Price category. Flip the switch to activate the coupon option for that offer.




**Question:** What should I do if the coupon toggle is not visible?
**Answer:** If the coupon toggle isn't visible, navigate to the Labs section under Settings. Scroll down to find the membership coupons option and activate it.




**Question:** Can coupons be used with all offers?
**Answer:** Coupons can only be used with Version 2 Offers and are not available for Free Offers.




**Question:** How can I create a new coupon?
**Answer:** Go to Payments > Coupons and click on "Create Coupon." Label your coupon, assign it a code (e.g., SUMMER20), select either a percentage discount or a fixed amount off, and set the start date. You can also set an optional end date for limited-time promotions.




**Question:** Are there customization options available when creating a coupon?
**Answer:** Yes, you can customize the coupon by limiting the total number of redemptions or restricting the coupon to specific products.




**Question:** How do users redeem a coupon?
**Answer:** During checkout, users will see an "Enter your coupon code" field. They can enter the code and click 'apply' to have the discount subtracted from the total payable amount.




**Question:** Can I limit how many times a coupon can be used?
**Answer:** Yes, when creating a coupon, you have the option to limit the total number of times it can be redeemed.




**Question:** Can I set a coupon to apply only to specific products?
**Answer:** Yes, you can restrict the coupon to be used only on certain products during the setup process.




**Question:** What types of discounts can I offer with coupons?
**Answer:** You can offer either a percentage discount or a fixed amount off with your coupons.




**Question:** Will users be able to see a coupon input field when they preview the offer?
**Answer:** Yes, once the coupon feature is activated for an offer, users will see an input field for entering their coupon code during the preview of the offer.




**Question:** Is there an option to set a coupon to expire?
**Answer:** Yes, you can set a start date and an optional end date for your coupon to limit its availability.




**Question:** Where do I manage my coupons?
**Answer:** Coupons are managed in the Payments section under Coupons, where you can create new coupons, edit existing ones, and view their performance.




**Question:** Can I apply a coupon to recurring payments?

**Answer:** Yes, this is an option when setting up your coupon.


**Question:Is it possible to prevent a single customer from using the same coupon multiple times?

**Answer:** Yes, within the coupon customization options, you can choose to limit the coupon to one use per customer.




**Question:** What happens to the price if a customer enters a valid code?

**Answer:** Once the customer clicks "Apply," the system automatically calculates the discount (either percentage or fixed amount) and subtracts it from the total payable amount in real-time before they complete the checkout.
