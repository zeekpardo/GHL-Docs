Email marketing remains one of the most effective ways to engage with your audience and boost business growth. In this guide, we will delve into the Email Builder elements within its Inline Editor, empowering you to streamline your email creation process and create compelling campaigns that resonate with your audience.

### Advanced Editing Options

The inline editor comes equipped with advanced editing options. The 'Clear Format' feature lets you strip any existing formatting from a section of the text, a handy tool when pasting text from external sources.

-   **Clear Format:** Removes existing formatting, useful when pasting text from external sources.
-   **Custom Values:** Adds a personalized touch by incorporating recipient-specific details.
-   **Trigger Links:** Enables interactive elements, simplifying user actions within the email.
-   **ContentAI:** Utilizes artificial intelligence to generate engaging email content quickly.

### Managing Elements

The Inline Editor simplifies text management and formatting:

-   **Font Styles, Sizes, and Colors:** Users can customize font styles, adjust sizes (from 8px to 96px), and modify text colors.
-   **Text Formatting Options:** Bold, italic, underline, and strikethrough features enhance text formatting.
-   **Hyperlinks and Alignment:** Users can add hyperlinks and control text alignment and line height.

The 'Manage Elements' function gives you control over each element within your email. 

Moreover, each element can be edited independently, allowing you to change its properties.

### Spacer Element

The Spacer element in the email builder allows you to easily add adjustable vertical or horizontal space between content blocks. You can control the exact height or width for precise spacing, leading to cleaner and more professional designs.

![](https://assets2.modalsupport.net/1773258540801-image3.png)

**Important Notes:**

-   The Spacer element is available in both new and existing templates.
-   It does not add any content; it only provides visual space in your email layout.

### Conditional Element

You can personalize email campaigns even further by adding conditional visibility to elements within a single email. This allows you to display specific content only to contacts who meet certain conditions, streamlining your campaign process. 

Go to the Visibility tab 

Enable Conditional Element. 

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXe50BsIHDglJ8XCTiohdBSsCWgKBqBr7L_CvsGrD7SmaSsfqkWTv-TEAoJUu9kQd-vSOvfo5GITz-pJG58zuiKmvFxvse5WWG0TQBnk2dKO8-QIbqmXHp5mN3VMq-TBikhUekFKoBhUXkTzI58goqnl3f4?key=-hU005FD7ezm8ozc8aM4nQ)

Conditions can be set based on various contact fields like single-line text, numbers, radio selects, and drop-down options. 

**Note:** Certain custom fields, such as multi-select fields, are not supported.

**“Null” as a Value:** You can set a conditional element's value to null in the email builder, enabling more targeted content delivery based on missing contact data. For instance, you would like to display elements only when specific contact fields are empty.

**Note:** Simply writing the word NULL would not work; the field needs to be kept empty.

## FAQs




**Question:** What conditions can I set for an element?
**Answer:** You can set conditions based on single-line text fields, numbers, radio selects, or single-select drop-down fields.




**Question:** Can I apply conditions to every field type?
**Answer:** No, currently only specific field types like single-line text, numbers, and radio selects are supported.




**Question:** Will contacts who don't meet the conditions still see the element?
**Answer:** No, only contacts who meet the set conditions will see the element.




**Question:** Can I undo or revert changes in the editor?
**Answer:**  Yes. You can **revert to a previous version** of your email using the version history feature, or manually undo recent changes during your editing session.




**Question:** How do I preview or test my email before sending?
**Answer:** Use the **Preview Template** and **Test Email** options in the editor to see how your email will appear to recipients and test interactive elements like trigger links.




**Question:** How can I use the "Null" value for a conditional element?

**Answer:** To use a "Null" value, you must leave the field empty rather than typing the word "NULL." This allows you to display elements specifically when contact fields are empty.
