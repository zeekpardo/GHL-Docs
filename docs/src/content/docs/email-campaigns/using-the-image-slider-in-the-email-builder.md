**Using the Image Slider in the Email Builder**

The email builder includes an Image Slider element, enhancing the toolkit for users aiming to design dynamic and visually engaging emails. This feature simplifies the process of creating and integrating image sliders, offering an enriched user experience and heightened engagement in email campaigns.

### Key Benefits of the Image Slider Element in the Email Builder

1.  **Visual Appeal**The Image Slider allows for the integration of dynamic images in a sequence, making your email campaigns visually engaging. This can increase recipient interaction and help communicate messages more effectively through eye-catching visuals.
2.  **Easy Setup and Customization**With a simple drag-and-drop interface, adding and configuring the Image Slider in emails is straightforward. You can add images directly from the media library or via URLs, making it flexible and easy to use for various types of email content.
3.  **Improved User Experience**By allowing images to be presented in a slider format, the feature helps create a more organized and clean layout. Users can focus on one image at a time while still being able to navigate through multiple images with ease.
4.  **Customizable Design**The Image Slider is fully customizable to suit your design preferences. You can adjust padding, margins, and even set the corner radius for the images. Additionally, the inclusion of thumbnails beneath the slider helps with navigation and adds a visual preview for the recipient.
5.  **Navigation Controls**With built-in options to add navigation arrows, recipients can easily move through the image slides. The slider supports user interaction, which enhances the overall engagement with the email content.

## Setting up the Image Slider:

1- Go to the Marketing tab:

-   Hover over the email section and navigate to campaigns or templates.
-   Click on the '+ New' button on the top right and select 'Blank'.

![](<https://assets2.modalsupport.net/1764328349902-unnamed - 2025-11-28T191210.457.jpg>)

2- In the builder:

-   Click on the element icon on the top left.
-   Drag the image slider element and drop it where you want it in the email.

### Configuring Your Slides:

**Adding the slides:**

-   You can add a slide for each image and set the order. Add the images to the slides in the same sequence in which you want them to appear in the email.

![](<https://assets2.modalsupport.net/1764328380879-unnamed - 2025-11-28T191245.477.png>)

**Adding Images to the slides:**

-   Add images to each slide. You can either add the URL of the image or upload the image from the media library by clicking on the image icon.
    -   Note: You need to click on each slide separately to add an image to the respective slide.

![](<https://assets2.modalsupport.net/1764328401301-unnamed - 2025-11-28T191310.274.png>)

-   The image slider element also supports custom values for media elements. To achieve this, simply access your builder, choose the media element (video, image, logo) to customize. Enter a custom value in the element’s source field to dynamically link content like images or videos.

![](<https://assets2.modalsupport.net/1764328420740-unnamed - 2025-11-28T191330.516.png>)

**Padding and Corner Radius:**

-   Set the padding and margin of the image slider and the corner radius of the images.

**Enable Thumbnail:**

-   Toggle on this option if you want to show thumbnails of the images beneath the image slider. This helps clients with pagination.
-   You can also set the thumbnail radius and thumbnail width if this option is enabled.

![](<https://assets2.modalsupport.net/1764328447879-unnamed - 2025-11-28T191353.394.png>)

**Include Arrows:**

-   Add arrows to the image slider to help clients navigate to the next or previous image.
-   You can also select the arrow icons from the available options.

![](<https://assets2.modalsupport.net/1764328465080-unnamed - 2025-11-28T191416.195.png>)

## FAQs 




**Question:** What is the Image Slider element in the email builder?
**Answer:** The Image Slider element is a feature in the email builder that allows users to create dynamic and visually engaging emails by integrating a sequence of images in a slider format. This enhances the visual appeal of emails and encourages more interaction from recipients.




**Question:** How do I add the Image Slider element to my email?
**Answer:** To add the Image Slider, navigate to the Marketing tab, select the email section under campaigns or templates, click '+ New' and choose 'Blank'. In the builder, click the element icon on the top left, drag the Image Slider element, and drop it into the desired location in the email.




**Question:** How can I add images to the Image Slider?
**Answer:** You can add images to each slide by clicking on the specific slide and then either uploading images from the media library or adding image URLs. Ensure you add the images in the sequence you want them to appear in the email.




**Question:** Can I adjust the padding and corner radius of the Image Slider?
**Answer:** Yes, you can customize the padding and margin of the Image Slider as well as set the corner radius of the images within the slider to fit your design preferences.




**Question:** What is the purpose of enabling thumbnails in the Image Slider?
**Answer:** Enabling thumbnails provides a mini preview of each slide below the Image Slider, helping recipients navigate through the images easily. You can also adjust the thumbnail radius and width when this option is enabled.




**Question:** How do I add navigation arrows to the Image Slider?
**Answer:** To add navigation arrows, toggle on the arrow option within the slider settings. You can select the desired arrow icons from the available options, which will help recipients move forward or backward through the image slides.




**Question:** Is there a limit to the number of slides I can add to the Image Slider?
**Answer:** There is no specified limit in the documentation, so you can add as many slides as needed for your email, keeping in mind the overall size and load time of the email.




**Question:** Can I control the order of the slides in the Image Slider?
**Answer:** Yes, you can set the order of slides by adding the images in the desired sequence during the configuration of your Image Slider.




**Question:** What image formats are supported for the Image Slider?
**Answer:** Typically, standard image formats like JPEG, PNG, and GIF are supported, but it’s recommended to check the platform’s guidelines for any specific format restrictions.




**Question:** Can I use URLs for adding images, or do I need to upload them?
**Answer:** You have the option to either upload images from the media library or use direct URLs for the images you wish to display in the Image Slider.




**Question:** Will all email clients support the Image Slider?
**Answer:** Most modern email clients will render the Image Slider properly, but **some older or restrictive clients** (e.g., certain versions of Outlook) may display only the **first image**. Always test your emails across devices and platforms for the best results.  




**Question:** How does the Image Slider benefit email campaigns?

**Answer:** It enhances user experience and engagement by presenting multiple images in a compact, interactive format, keeping recipients interested.




**Question:** Can the Image Slider element support dynamic content like videos or logos?
**Answer:** Yes, the Image Slider element supports custom values for media elements, allowing you to dynamically link content like images, videos, or logos by entering a custom value in the element’s source field.




**Question:** Is it possible to use the Image Slider to show a sequence of logos or videos?

**Answer:** Absolutely. While it is primarily used for images, the slider supports various media elements, including **videos and logos**, which can be linked directly or through custom values.
