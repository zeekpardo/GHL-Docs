**How to Use the Email Builder for Email Templates**  

When using the _email builder_, an **email template** is a preset layout into which you can insert images, videos, and text elements to create a customized template for future emails. 

Please note that an **email template** will not be the final email sent to contacts; it is just a layout placeholder for you to use as a foundation for future emails. The actual email that you will send out is called an **email campaign**. 

We recommend only using placeholder text and custom values when creating an **email template** in the builder. The unique and customized text should be added once you send out an **email campaign** (that uses the **template** as its foundation).

Navigate to Marketing on the left > Emails at the top > Select Templates and Click **\+ New** to create a new template.

**Code Editor:**

In the Code Editor, you can use HTML to modify and customize the email with code.

**Plain Text Editor:**

The Plain Text Editor allows you to create plain-text emails. This streamlined editor is designed for crafting simple, text-based emails without the complexity of HTML or design elements. 

**Includes:**

-   **Simple Text Formatting:** Basic tools for text formatting such as font size, bold, italics, and headers.
-   **Basic Text Types:** Options for paragraphs, headings, and other text structures.
-   **Link Insertion:** Ability to add clickable links to text.
-   **Image Embedding:** Option to embed images within the email.
-   **Text Alignment:** Tools for aligning text and adjusting spacing.
-   **Bulleted Lists:** Ability to create and format bulleted lists.
-   **Custom Values and Trigger Links:** Support for custom values and trigger links to personalize emails.
-   **Preview Options:** Preview emails on desktop, tablet, and mobile devices.
-   **Sender and Subject Line Settings:** Adjust sender details and subject lines.
-   **Testing Capabilities:** Test emails before sending to ensure they appear as expected.

### FAQs




**Question:** How do I choose the right editor for my email template?
**Answer:** Use the Code Editor for advanced customization with HTML, the Plain Text Editor for simple, text-based emails, and the Design Editor for a visual, drag-and-drop experience.




**Question:** Can I use custom values in my email templates?
**Answer:** Yes, custom values can be used to personalize your email content. Make sure to add these values when creating your email campaign, not in the template.




**Question:** How do I preview my email template before sending?
**Answer:** Use the preview option found in the settings (three dots) to view how your email will appear on different devices.




**Question:** What should I do if I want to make changes to a pre-made template?
**Answer:** Open the template in the Design Editor, make your changes, and save it as a new template or overwrite the existing one.




**Question:** How can I ensure my emails are not too formal and maintain a personal touch?
**Answer:** Use the Plain Text Editor for a more personal feel and avoid overloading your emails with too many design elements.




**Question:** How does the AI feature help with email templates?
**Answer:** The AI tool can generate content suggestions, helping speed up email creation while maintaining quality and relevance.




**Question:** What happens if there is a syntax error with my custom values?

**Answer:** If a syntax issue is detected in your custom values, a warning will appear at the top of the email template. You can click on this warning to be directed to the specific area that needs correction.




**Question:** Can I save specific elements to be used and synced across other parts of my account?

**Answer:** Yes, you can use Universal Elements. This feature allows you to save individual elements such as buttons, forms, and navigation menus so they automatically sync across all your pages, funnels, blogs, and e-commerce sections.




**Question:** Can I paste text from external documents into the email builder without messing up the formatting?
**Answer:** Yes. The email builder features an "auto-cleansing" function for text. When you copy and paste from external sources, the system is designed to support a seamless transfer while automatically removing any pre-applied styling to ensure your template remains clean and consistent.
