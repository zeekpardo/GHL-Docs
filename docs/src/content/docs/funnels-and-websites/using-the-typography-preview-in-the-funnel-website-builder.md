## Typography Preview

has an integrated live typography preview feature, eliminating the guesswork in font selection. With access to over 1500+ fonts from Google Fonts, users can see a real-time preview of how text appears in different styles directly in the font selection dropdown menu. This streamlines the design process and empowers users with immediate visual feedback.

## How It Works: A Step-by-Step Guide

Here's how you can enhance your website-building experience with 's live typography preview:

**Dive into the Builder:** Access the Builder in Sites > Funnels > Select the funnel, and you’ll find yourself in a much more intuitive interface. Everything you need is right where it should be, sleek and accessible.

![](<https://assets2.modalsupport.net/1762263131381-unnamed - 2025-11-04T213147.320.png>)

**Experiment with Typography:** Go to the Typography settings. As you search through the extensive list of available fonts, the live preview feature will display how each font looks right then and there. There is no need to apply it first—see it in action as you browse.

## Why It Matters

**Efficiency:** The immediate preview of fonts slashes the time spent on selecting the perfect typography, making the process faster and more efficient.

**Better Decision-Making:** Designers and marketers can make more informed decisions about branding and visual communication strategies.

**User Experience:** By reducing the steps involved in the process, the overall user experience is significantly improved, which is always a core objective in good design practices.

### FAQ




**Question:** Can I preview multiple fonts at once using the live typography preview feature?
**Answer:** No, the live typography preview allows you to see one font at a time as you scroll through the list. Each font will be displayed in real-time, helping you quickly evaluate and choose the right one for your design.




**Question:** Are all Google Fonts available for preview in the Builder?
**Answer:** Yes, the integrated typography preview feature includes access to over 1500+ Google Fonts, allowing you to explore a vast selection of font styles directly within the Builder.




**Question:** Does the live typography preview feature work on all devices?
**Answer:** The live typography preview is designed to work seamlessly within the Builder interface across various devices. However, the experience might vary slightly depending on screen size and browser compatibility. It’s recommended to use a modern browser for the best results.




**Question:** Can I apply a font directly from the preview?
**Answer:** Yes. Once you see a font you like in the live preview, you can apply it directly to your text in the Builder.




**Question:** Does using many different fonts from the preview slow down my page?

**Answer:** While you can preview as many as you like, we recommend using only 2–3 different font families per page. This keeps your design consistent and ensures your website loads quickly for your visitors.




**Question:** If I change a font in the preview, does it update every page in my funnel?
**Answer:** If you update the font in the **Global Typography** settings, it will update every page within that specific funnel. If you change the font for a specific "Text Element," it will only change that one instance.




**Question:** Can I upload my own custom "Brand Font" if I don't see it in the 1500+ Google Fonts?

**Answer:** Yes! While the live preview is built for the Google Fonts library, you can upload custom font files (like .ttf or .woff2) in the **Settings > Custom CSS** or **Media Library** (depending on your version). Note that custom uploaded fonts may not show the same real-time preview in the dropdown as the integrated Google Fonts.
