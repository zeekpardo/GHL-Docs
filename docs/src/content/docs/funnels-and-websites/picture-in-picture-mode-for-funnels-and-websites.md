**Picture in Picture Mode for Funnels and Websites**

The PiP mode is an innovative addition to 's funnel/website builder. It allows a hosted video to continue playing in a miniaturized view as the user scrolls down the page. The video elegantly minimizes to a small window positioned in the bottom right corner of the browser, providing a continuous viewing experience without interrupting the user's navigation through the site.

## How Does it Work?

Upon enabling this feature, whenever a user watches a video and scrolls past it, the video automatically transitions into PiP mode. This creates a floating video window that follows the user as they continue to explore the page. It eliminates the need for manual activation of PiP mode, offering a seamless and engaging user experience.

## How to Use?

To take advantage of this feature, follow these simple steps:

### Add a Video Element:

Navigate to your funnel/website builder within

![](https://assets2.modalsupport.net/1763477519756-2.png)

Select the section where you wish to add your video.  
Click on 'Add Element' and choose the 'Video' option.

![](https://assets2.modalsupport.net/1773261717778-image4.png)

### Select Hosted Video:

Once the video element is placed, click on it to access the settings.  
In the video settings, select the option to use a 'Hosted Video'.

![](https://assets2.modalsupport.net/1763477648737-4.jpg)

Upload or select the video you wish to use from your library.

![](https://assets2.modalsupport.net/1763477677036-6.png)

### Toggle PiP Mode on Scroll:

Toggle the option that says "Enable PiP mode on scroll" within the General settings in the builder.  
Once enabled, the PiP feature will activate automatically when a user scrolls past the video.

### FAQ




**Question:** Can PiP mode be used with videos hosted on external platforms like YouTube or Vimeo?
**Answer:** No, the PiP mode in 's funnel/website builder is specifically designed for hosted videos. You need to upload or select a video from your library within to use this feature.




**Question:** Can I customize the position of the PiP window on the screen?
**Answer:** Currently, the PiP window is fixed to the bottom right corner of the browser. Customization of the PiP window’s position is not available at this time.




**Question:** How do I disable PiP mode for a specific video?
**Answer:** To disable PiP mode, scroll through General settings for the video element within the Website or Funnel Builder and toggle off the "Enable PiP mode on scroll" option. This will prevent the video from minimizing into PiP mode when a user scrolls past it.




**Question:** In which specific tools can I use this PiP feature?

**Answer:** This feature is available for use within both the Funnel Builder and the Website Builder inside the platform.  




**Question:** Can I use PiP mode for multiple videos on the same page?

**Answer:** Yes. Each hosted video on the page can have PiP mode enabled individually. Only the video that the user scrolls past will transition into the floating window.
