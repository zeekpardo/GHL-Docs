**How to add multiple chat widgets**

Adding multiple chat widgets can significantly enhance user engagement by providing tailored communication options on various landing pages. This feature allows for different designs and customizations, ensuring an interactive user experience.

## Setting Up Multiple Chat Widgets

### Step-by-Step Guide

### Step 1: Access the Chat Widget Settings

-   **Navigate to Sites**:
    -   Click on **Sites** in the main menu.
    -   Select **Chat Widget** from the submenu.

### Step 2: Create a New Chat Widget

-   **Initiate Widget Creation**:
    -   Click on **Create New Chat Widget**.

![](https://assets2.modalsupport.net/1766485750468-unnamed.png)

-   **Select Chat Type**:
    -   Choose from the following options:
        -   **All-in-One Chat**
        -   **SMS/Phone Chat**
        -   **Live Chat**
        -   **WhatsApp Chat**
        -   **Facebook Chat**
        -   **Instagram Chat**
        -   **Voice AI**

![](<https://assets2.modalsupport.net/1766485770810-unnamed \(1\).png>)

-   **Note**: The current setup will become the default chat widget.
-   **Note:** You can use the **All in One chat** option to eliminate the need for multiple widgets and manage all interactions in one place.

### Step 3: Customize the Chat Widget

-   **Design and Customization**:
    -   Customize the widget design to match your landing page aesthetics.
    -   Adjust settings as needed for optimal user interaction.

![](<https://assets2.modalsupport.net/1766485793027-unnamed \(2\).png>)

-   **Save and Implement**:
    -   Save your settings to implement the new chat widget on your site.

![](<https://assets2.modalsupport.net/1766485813190-unnamed \(3\).png>)

### Step 4: Integrate Chat Widgets with Funnels or Websites

-   Select Chat Widget in Funnels/Websites:
    -   In your funnel or website settings, select the chat widget you want to use for specific pages.

![](<https://assets2.modalsupport.net/1766485832769-unnamed \(4\).png>)

-   **Support for WordPress**:
    -   For WordPress sites, the default chat widget will be supported.

## Why Use Multiple Chat Widgets?

-   **Diverse Communication Options**: Different landing pages may require distinct communication channels.
-   **Enhanced Customization**: Tailor each widget to match the design and functionality of specific pages.
-   **Improved Performance**: Streamlined code for better performance and hassle-free updates.

**FAQs**


**Question:What are the benefits of using multiple chat widgets?
**Answer:** Multiple chat widgets offer diverse communication options, allowing different landing pages to use different channels (e.g., SMS, live chat, WhatsApp). This enhances user engagement, improves customization, and streamlines communication based on the specific needs of each page.


**Question:How do I set up multiple chat widgets?
**Answer:** Follow these steps:

-   Go to Sites in the main menu and select Chat Widget from the submenu.
-   Click on Create New Chat Widget.
-   Choose the chat type (SMS/Phone Chat, Live Chat, or WhatsApp Chat).
-   Customize the widget’s design and settings to fit the landing page.
-   Save and implement the widget on your site.


**Question:Can I use different chat widgets on different landing pages?
**Answer:** Yes, you can select a specific chat widget for each funnel or website page. Simply choose the widget you want to use from the settings of the respective page or funnel.


**Question:How do I customize the appearance of the chat widgets?
**Answer:** You can design and customize the chat widget to match the aesthetics of your landing page. Adjust the colors, size, placement, and other elements to ensure it integrates smoothly with the page’s design.


**Question:Can I integrate chat widgets with my WordPress site?
**Answer:** Yes, the default chat widget will be supported for WordPress sites. You can set up and implement the widget just like on any other website.




**Question:** Is there a way to manage all my communication channels without creating multiple separate widgets?

**Answer:** Yes. You can use the **All-in-One Chat** option. This eliminates the need for multiple separate widgets by allowing you to manage all types of interactions in one central place.




**Question:** Which chat widget becomes the default one when I have multiple versions created?

**Answer:** In the system, your current or most recent setup is automatically designated as the default chat widget. This is particularly important for integrations like WordPress, which specifically support the default widget.
