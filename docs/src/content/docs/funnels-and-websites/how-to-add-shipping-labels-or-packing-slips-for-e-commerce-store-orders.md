**How to add Shipping labels or packing slips for E-commerce store orders**

The ability to print shipping labels and packing slips directly from your store’s order management system helps you handle orders with ease, ensuring you have all the necessary information in one place.

### **Advantages**

You can generate and print shipping labels and packing slips for any e-commerce store order. Whether you need labels for entire orders or just specific items, this feature provides the flexibility you need. The labels and slips will include important details such as:

-   **Order ID**
-   **Order Date**
-   **Store Name**
-   **Shipping and Billing Address**
-   **List of Products** with:
    -   Product Variants
    -   Product Images
    -   Quantities
-   **Store Address** (Sender's Address)

#### **How to Use**

Here’s how you can start using the print shipping labels and packing slips feature:

**1-Go to Your Orders**:

-   Navigate to **Payments > Orders** in your store.

**2-Select an Order**:

-   Choose any order from the list that you want to print a label or packing slip for.

**3-Print the Shipping Label**:

-   Click the **"Print Shipping Label"** button

You can print shipping labels from two main places:

**1- Order Details Page**:

-   On the order details page, click the **"Print Shipping Label"** button to download a PDF with all the order information. This PDF will include both unfulfilled and fulfilled orders on separate pages.

#### **Important Notes**

-   **E-commerce Orders Only**: You can only generate shipping labels and packing slips for e-commerce store orders.
-   **Multiple Fulfillments**: If an order has multiple partial fulfillments, each one will list the items on a separate page when you generate shipping labels.
-   **Address Fields**: Right now, the billing and shipping addresses are the same on the shipping labels. This will change once the billing address field is added to the checkout page.
-   **Packing Slips Access Location:** Packing slips can also be generated from the order details page or during fulfillment using the same “Print Shipping Label” button. This allows users to download a PDF with order details organized by fulfillment status.

### FAQs




**Question:** Can I print shipping labels for non-e-commerce store orders?
**Answer:** No, shipping labels and packing slips can only be generated for e-commerce store orders.




**Question:** What happens if I have multiple fulfillments for the same order?
**Answer:** If an order has multiple partial fulfillments, each one will be listed on a separate page when generating the shipping labels.




**Question:** Why are the shipping and billing addresses the same on the labels?
**Answer:** Currently, the shipping and billing addresses are the same on the shipping labels. However, this will be updated once the billing address field is added to the checkout page.




**Question:** Can I print labels for specific items only within an order?
**Answer:** Yes, when fulfilling an order, you can select the specific items and quantities for which you want to generate the shipping label.




**Question:** Can I print a packing slip or shipping label for a partially fulfilled order?
**Answer:** Yes, from the Orders page, click the three-dot menu next to a partially fulfilled order to download a shipping label or packing slip specific to that fulfillment.




**Question:** Can I print packing slips in the same way as shipping labels?
**Answer:** Yes, packing slips are generated using the same process and interface as shipping labels, and include all order details such as item variants, images, and quantities.




**Question:** Do the shipping labels include all necessary order information?
**Answer:** Yes, labels and packing slips include Order ID, Order Date, Store Name, Shipping & Billing Address, Products with variants and images, quantities, and sender’s address.


**Question:Can I print shipping labels for canceled or refunded orders?
**Answer:** Shipping labels and packing slips are intended for active e-commerce orders. Canceled or refunded orders may not be eligible for label generation.  




**Question:** Can I generate a label at the same time I am marking an order as shipped?

**Answer:** Yes. **During fulfillment**, after you select the items and quantities being shipped, you can click the "Print Shipping Label" button to generate a document specifically for those items.




**Question:** If I ship an order in two separate boxes, can I get two different packing slips?

**Answer:** Yes. If an order has multiple partial fulfillments, the system will generate each one on a separate page of the PDF, allowing you to include the correct slip in each specific box.
