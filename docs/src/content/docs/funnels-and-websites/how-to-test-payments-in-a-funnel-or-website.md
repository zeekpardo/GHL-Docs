In this article, we will show you how to switch your Funnel or Website into test mode. This way, you can test your products and automation associated to ensure they’re working properly.

## Part 1: Putting the Funnel/Website into test mode. 

-   First, navigate to your Sites and select the funnel on which you would like to test the payment mode.
-   Go to Settings.
-   Then, select the toggle button to switch to Test mode.

### FAQ




**Question:** How do I switch my Funnel or Website into test mode?

**Answer:** Navigate to your Sites and select the funnel or website you want to test. Go to Settings and then toggle the switch to activate test mode.




**Question:** What should I do after putting my Funnel or Website into test mode?

**Answer:** Visit your live page or preview and use any of the provided test card details to submit an order form and verify that payments and automation are functioning correctly.




**Question:** Can I use real credit card information while testing?

**Answer:** No, for testing purposes, use only the provided test card details to ensure that your payment systems and automation are working correctly without processing actual transactions.




**Question:** What happens after I complete a test order using the test card details?

**Answer:** After submitting a test order using the provided test card details, you can verify if the payment processing and automation are working correctly. If any issues are found, you can troubleshoot and make adjustments as needed. The test orders are not real transactions, so no money will be charged.




**Question:** Can I use test mode on multiple funnels or websites at the same time?

**Answer:** Yes. Test mode is applied individually, so you can enable it on specific funnels or websites without affecting others.  




**Question:** What happens if I forget to turn off Test Mode?
**Answer:** If you leave Test Mode on, real customers will not be able to buy your products. Their real credit cards will be declined, and your business will not collect any payments. **Always double-check that the toggle is OFF before driving traffic to your page.**




**Question:** What specific CVC and Expiry Date should I use when entering test card information?
**Answer:** For most test cards (like Visa or Mastercard), you can use any 3-digit number (such as 123) for the CVC and any future date (such as 12/34) for the Expiry Date. If you are testing an American Express card, you should use any 4-digit number for the CVC.
