You can add a Facebook Pixel event directly to your forms, surveys, and calendars. This pixel will then fire upon a submission of the form, survey, or calendar. This is a default event that registers with Facebook as 'Submit Application' for a Form or Survey Submission and 'Schedule' for a Calendar Submission 

## **What you'll need:**

-   Your Facebook Pixel ID

### **Step 1:** Form Pixel

-   Navigate to the Form Builder.
-   Select Your Form.
-   Click Settings on the top center > Input your Pixel ID into the Pixel Section.
-   Integrate Form and Save.

![](https://assets2.modalsupport.net/1765391088875-image4.jpg)

### **Step 2:** Survey Pixel

-   Navigate to the Survey Builder.
-   Select Your Survey.
-   Click Settings on the top Center
-   Input your Pixel ID into the Pixel Section.
-   Integrate Survey.
-   Save to confirm your changes.

![](https://assets2.modalsupport.net/1765391126289-image5.jpg)

### **Step 3:** Calendar Pixel

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXe0Dwq9Uii4SNuRacJ8L2t6tLNf0mhvW3V-NO6JvQZZmyTrGTSK7_v9VFr764MbsZnsBmI1iK702igujk9q7p2Qu5Wd69HktFavXeUaxSrXeqJnBgMGD8GvmC-G4fcNIQcIiafkc3Jc4lJRRL-HYWoKc6dm?key=y8jfEsOWsv7Y8rRWPGwd-Q)

-   Navigate to Settings > Calendar.
-   Click in to edit the Calendar (Pencil Icon)
-   Input the Pixel ID into the Calendar.
-   Click Save.

## **FAQ’s**




**Question:** What events does the Facebook Pixel register for forms, surveys, and calendars?
**Answer:** For forms and surveys, the Facebook Pixel registers the event as 'Submit Application' upon submission. For calendar bookings, it registers the event as 'Schedule' when an appointment is made.  




**Question:** Can I use the same Facebook Pixel ID for multiple forms, surveys, and calendars?
**Answer:** Yes, you can use the same Facebook Pixel ID across multiple forms, surveys, and calendars. This allows you to track all submissions and bookings under a single Pixel, providing consolidated data for analysis.  




**Question:** What should I do if the Facebook Pixel is not firing correctly?
**Answer:** If the Facebook Pixel is not firing correctly, double-check that the Pixel ID is correctly entered in the Pixel section of the form, survey, or calendar settings. Ensure there are no typos and that the changes have been saved properly. You can also use Facebook’s Pixel Helper tool to troubleshoot and verify the Pixel setup.  




**Question:** Do I need to make any changes on Facebook after setting up the Pixel in forms, surveys, or calendars?
**Answer:** No additional changes are required on Facebook once the Pixel ID is correctly set up in your forms, surveys, or calendars. The Pixel will automatically start tracking events as users interact with your submissions.  




**Question:** Can I track specific actions taken on forms, surveys, and calendars using the Facebook Pixel?
**Answer:** Yes, by integrating the Facebook Pixel with your forms, surveys, and calendars, you can track specific actions like form submissions, survey completions, and calendar bookings. These actions will be registered as predefined events on Facebook, which can be used for analytics and retargeting.  




**Question:** What happens if I change my Facebook Pixel ID?
**Answer:** If you change your Facebook Pixel ID, you will need to update it in the Pixel section of each form, survey, and calendar where it was previously entered. Save the changes to ensure the new Pixel ID is correctly linked and tracking events.  




**Question:** Is there a limit to the number of forms, surveys, or calendars I can connect to a single Facebook Pixel?
**Answer:** There is no specific limit to the number of forms, surveys, or calendars you can connect to a single Facebook Pixel. You can use the same Pixel ID across all relevant sections of your site to consolidate tracking and reporting.  




**Question:** How do I know if my Facebook Pixel is working correctly after integration?
**Answer:** To verify if your Facebook Pixel is firing correctly after integration, you can use the Facebook Pixel Helper tool. This browser extension will show you if the Pixel is firing on your forms, surveys, and calendars. If the Pixel is not firing, check your Pixel ID in the respective sections and ensure that you’ve saved the changes properly.  




**Question:** What is required before I can begin the setup process for my forms, surveys, or calendars?

**Answer:** You will need to have your Facebook Pixel ID available to input into the settings for each specific tool.  




**Question:** Is the Pixel ID applied to the whole site or individual elements?

**Answer:** This process allows you to add the Pixel ID directly to **individual** forms, surveys, and calendars. This ensures the event fires specifically when those elements are submitted, rather than just tracking a general page view.




**Question:** Where exactly do I find the Pixel ID field within the Form or Survey Builder?
**Answer:** To add your Pixel ID, you need to open your specific form or survey in the builder, click on the **"Settings"** tab located at the **top center** of the screen, and then input your ID into the designated Pixel section.
