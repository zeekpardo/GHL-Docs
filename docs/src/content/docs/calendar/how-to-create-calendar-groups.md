**How to create calendar groups**

Calendar groups offer a consolidated view where users can select from multiple calendars. This feature is useful for scenarios where clients need to choose from different scheduling options. Calendar settings are designed to streamline user availability management and facilitate integrations with tools like Google Calendars, Outlook, Zoom, and Stripe. These settings are particularly useful for Grouping Calendars, where user availability and primary calendar connections play a crucial role.

Additionally, some settings may impact Event Calendars, and the calendar’s “week view” can also be adjusted here. This article will guide you through the Calendar Settings interface and key considerations on how to effectively add calendars to groups. 

### How to Group a Calendar

To utilize calendar groups, you will need to create a group by selecting the create group button.

![](https://assets2.modalsupport.net/1773257600551-image6.png)

In the column for groups, you can find existing group names next to a folder icon. This will indicate that it is a grouped calendar. You can also view all existing groups in the ‘groups tab’

![](https://assets2.modalsupport.net/1773257622163-image14.png)

![](https://modal-inbox-chat-attachments.s3.us-east-2.amazonaws.com/1762366047629-image.png)

In order to move a calendar to a group, you will need to have a group. You can create a group by clicking ‘New Group’  

![](https://assets2.modalsupport.net/1773257641797-image2.png)

Provide the group name, description, template (select either the Classic or Neo templates for the Group view), and the group URL.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeeuGbNqs9k2KR-glKfBXu7Mq5f0pOsvx2sfXdxUzF_nBAJX2t2GOuqOU22RBn-rGHKwWNdciv75_DogEly8MYloZ6gzUyXnVUnGxA8A_35-K5_Xv1s4wTgo5O1A85U6fos3_-Q9Lceo1FCNSvGm_TQHMVY?key=740C7HX4J9S4YykfqWqAvw)

Once your calendar group is set up, you can easily embed it on various platforms like WordPress, Wix, or Squarespace. Obtain the embed code or direct link for the calendar group from the group settings tab by copying it for the individual calendar or group of calendars and paste it into the HTML section of your website. If you encounter display issues, such as incorrect height, adjust the iframe code by adding a height parameter (e.g., height: 750px)."

![](https://assets2.modalsupport.net/1773257659853-image16.png)

**Managing Calendar Groups**

This allows users to organize and control multiple calendars by grouping them. You can easily deactivate or reactivate an entire group for efficient management.

**How to Use:**

**1- Deactivate a Group**: Click the three dots next to the group, select **Deactivate all calendars in group**, and click **Save**.

![](https://assets2.modalsupport.net/1773257700051-image8.png)

**2- Reactivate a Group**: Click the three dots, select **Activate all calendars in group**, and click **Save**.

![](https://assets2.modalsupport.net/1773257721430-image9.png)

**Important Notes:**

-   Deactivating a group disables all calendars within it.
-   Reactivating restores all calendars within the group.

### Calendar Group Deletion

Users can choose to either retain all calendars and their appointments while deleting only the group or delete the group along with all associated calendars and appointments. This offers greater flexibility in managing calendar data.

**How to Use:**

1- Click on the three dots next to the calendar group you wish to delete.

2- Select Delete.

3- In the pop-up, check the box if you want to delete the calendars as well. Leave it unchecked to retain the calendars and their appointments.

4- Click Confirm.

![](https://assets2.modalsupport.net/1773257829271-image1.png)

### FAQs




**Question:** What are calendar groups and how do they work?
**Answer:** Calendar groups allow users to consolidate multiple calendars into one view, making it easier for clients to choose from different scheduling options. Users can create, organize, and manage these groups, which can include calendars from tools like Google Calendars, Outlook, Zoom, and Stripe.  




**Question:** How do I create a calendar group?
**Answer:** To create a group, click the "Create Group" button, provide a group name, description, and URL, then select a template. Once set up, you can embed the group on platforms like WordPress, Wix, or Squarespace using the provided embed code.  




**Question:** How do I manage or rearrange calendars within a group?
**Answer:** You can rearrange the order of calendars in the group by clicking on the menu dots and selecting “Rearrange Calendars.” You can also search for groups and edit settings using the group tab.  




**Question:** Can I delete a calendar group without removing the associated calendars and appointments?
**Answer:** Yes, when deleting a group, you can choose whether to delete the group only or delete the group along with all associated calendars and their appointments by checking or unchecking the relevant box in the pop-up.  




**Question:** How can I integrate my calendar with Zoom or Google Meet?
**Answer:** You can add custom meeting locations, including integrations for Zoom, Google Meet, or phone links, under User Preferences. This ensures that all scheduled meetings are equipped with the correct meeting links.  




**Question:** Can I automate processes like reminders or confirmations for calendar appointments?
**Answer:** Yes, you can set up workflows to automate processes such as sending reminders, confirmations, and follow-up messages based on the status of booked appointments, helping to streamline the calendar management process.




**Question:** Will changing the group order affect my website embed?
**Answer:** Yes, the order in which calendars appear inside the group will also reflect in the embedded version on your website.
