Outlook calendar integration is a great feature with a Teams Calendar. This allows you to create and sync calendar invites for Outlook, to sync back to .

**_NOTE_:** Only Outlook.com, Office 365, live.com, and Hotmail calendars are supported. You can also have only one (1) Outlook account connected to one user in at a time.

### Step 1: Connecting to the Outlook Integration

-   Navigate to Settings > Profile.
-   Scroll down to “Calendar Settings” and click the “Connect” button next to Outlook Calendar.

### Step 2: Configure the Calendar to use Outlook Integration

-   Navigate to Settings > Profile.
-   Under “Calendar Configuration” go to “Primary Calendar”.
-   Click edit.
-   Select your Outlook account and find your calendars. Select the one you want to read/write.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcORue-K3ZPn1ibgRzSuEz6OmQW9lbm2ZPS-cuMujELLlTseZNI0QQblNbP2LOWm5PyqRhnHzG3VrSwcKgNsoFM3v_QymeEZeL-lgi_cAV1zgTz3Ja2TF7ulKJfCjvcT0TNYACEpr3a9xL9tSvO2zwU7-tu?key=poLVoH5kJj249xJT0S_fEQ)

-   Your **primary Calendar** will read all the calendar events from Google/Outlook and write all the appointments to Outlook/Google.
-   **Check for Conflicts**: It only reads calendar events from Google/Outlook.

-   Navigate to “Check for Conflicts” and click on the edit button.
-   Here you can select the calendar from which you want to check for conflicts.

**Please Note:**

If appointments are not syncing over from Outlook or to Outlook, please **Edit the Primary calendar**, choose **Do not sync events to the primary calendar** \> Save > Then reselect the same calendar. This should resolve the syncing issue.

### Step 3: Managing Email Notifications for Third-Party Calendars

Users have enhanced control over communication preferences for third-party calendars, allowing them to manage invitation and appointment update emails more effectively.

### How to Use

**Navigate to Calendar Settings:** Access your calendar settings from the dashboard.

**Locate Notifications & Policies:** Find and click on "Notifications & Policies."

![](https://assets2.modalsupport.net/1773939703815-image3.png)

**Enable or Disable Notifications:** Check or uncheck the box labeled "Allow Google Calendar to send invitation or update emails to attendees" to enable or disable notifications from the respective third-party calendars.

**FAQ's**




**Question:** Is Outlook integration compatible with Event Calendars?
**Answer:** No, Outlook integration is not supported for Event Calendars. However, it is fully compatible with Round Robin, Collective, and Service Calendars.




**Question:** Which Outlook calendars are supported for integration?

**Answer:** The integration supports Outlook.com, Office 365, live.com, and Hotmail calendars. Only one Outlook account can be connected to one user in at a time.




**Question:** Is it possible to use Google Calendar and Outlook Calendar simultaneously?
**Answer:** Yes, both Google Calendar and Outlook Calendar can be used at the same time. You can set either as the primary calendar and configure it to check for conflicts with the other.




**Question:** Can appointments be written to both Google and Outlook calendars simultaneously for a single user in one location?
**Answer:** No, appointments can only be written to one calendar at a time.




**Question:** What happens if I have conflicts between my calendars?

**Answer:** When checking for conflicts, you can select which calendar to read for conflicts. This will help you avoid scheduling issues by ensuring that overlapping events are identified and managed appropriately.




**Question:** Can I connect multiple Outlook accounts to ?

**Answer:** No, only one Outlook account can be connected to a single user in at a time. Ensure that you are connecting the correct Outlook account to avoid any issues.




**Question:** What should I do if I encounter issues with the Outlook calendar integration?
**Answer:** Double-check the connection settings and ensure that you have authorized the integration correctly. If problems persist, consult ’s support team for further assistance.




**Question:** How can I ensure that my calendar events are synchronized properly?
**Answer:** Make sure you have selected the correct calendar for reading and writing events, and regularly check the synchronization status in your calendar settings to ensure everything is up-to-date.




**Question:** Will changes made in be reflected in my Outlook calendar?

**Answer:** Yes, any changes made in to calendar events or appointments will be synced back to your Outlook calendar, provided that the integration is properly configured.  




**Question:** What should I do if appointments are not syncing between my Outlook and calendar?
**Answer:** If appointments are not syncing from Outlook or to Outlook, go to your "Primary Calendar" settings, select "Do not sync events to the primary calendar," then click "Save." Afterward, reselect the same calendar to restore proper syncing. This should resolve any syncing issues.




**Question:** How do I configure my Outlook account to sync with my primary calendar?
**Answer:** Go to **Settings > Profile**, then under "Calendar Configuration," click edit next to "Primary Calendar." Select your Outlook account and the calendar you want to sync for reading and writing events.




**Question:** What should I do if the integration connection fails?
**Answer:** Double-check your account connection and permissions. Ensure you’ve authorized the integration properly. If issues persist, contact support.  




**Question:** Can I sync "Recurring Events" from Outlook?
**Answer:** Yes. If you have a recurring "Weekly Staff Meeting" in Outlook, the system will read those occurrences as conflicts and block those times off every week on your booking page.
