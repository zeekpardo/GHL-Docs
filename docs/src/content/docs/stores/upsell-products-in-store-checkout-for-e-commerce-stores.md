**Upsell products in store checkout for E-commerce stores**

To maximize revenue and increase average order values, store owners can add upsell options within their e-commerce checkout process. This feature allows stores to create upsell funnels that guide customers from the checkout to the Thank You page, offering additional products and optimizing the customer journey. This article will guide you through setting up upsell pages, adding and configuring upsell elements, and leveraging upsell actions to encourage additional purchases.

**_Key Features and Benefits_**

-   **Increased Revenue**: Upselling provides an opportunity to increase the average order value by presenting customers with relevant products after they complete the initial purchase.
-   **Customizable Upsell Funnels**: Store owners can easily create and customize upsell pages, adding as many upsell options as desired to meet customer needs.
-   **Seamless Integration with Checkout Process**: The upsell pages integrate directly with the checkout and Thank You pages, ensuring a smooth flow in the customer journey.

### **How to Set Up and Use the Upsell Feature**

**1\. Create an Upsell Page**

-   Navigate to **Sites > Stores** and create a new page for upsells.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXf5KLBEkA4nmIGg1m64hVzcSZKmrDgdy1iA00RiwZneh3c_QcLkoN7D_kbyxid_3h7474C_io-G2bcxjT1R8OoOLlAnWdgYF-c09UL8p1DB7faT0tGZWxHOIo6_tO6Kbydewjl7sfAUxycwgmZSI7VyGnHw?key=JXGU18P-uJ225M7m_-8gZQ)

-   Add a full-width section to this page and include a row for placing upsell elements.

**2\. Add the Upsell Element**

-   Select **Add Element**, then drag and drop the **Upsell Element** into your row.
-   You can add multiple upsell elements within a row by arranging them in separate columns.

![](https://assets2.modalsupport.net/1770921054516-image11.png)

**3\. Configure the Upsell Element**

-   Select the upsell element and configure its settings.

-   Store owners can enable or disable the shipping charges for each Upsell element by enabling/ disabling the toggle in the right side bar.

![](https://assets2.modalsupport.net/1770921088638-image9.png)

-   To Personalize text access Text Customization under General settings, make edits, and publish your changes.

**Note:** Some text, such as error messages, may not appear in the preview but can still be customized. Existing customization options (e.g., Button Options, Form Options) have been moved to the **Text Customization** section

![](https://assets2.modalsupport.net/1764183540325-image6.jpg)

-   In **Upsell settings**, users can Search and select the product. Then, **Sale Action** needs to be configured which decides the redirection after clicking on the **Buy Now** button. The redirection can be done to another upsell page or the Thank you page.

#### **4\. Automated Shipping Rate Sorting**

The checkout system automatically manages how shipping rates are displayed to the customer to ensure a frictionless experience.

-   **Ascending Order:** All available shipping rates are shown from lowest to highest price.
-   **Default Selection:** The system automatically pre-selects the cheapest available rate for the customer.
-   **Free Shipping Priority:** If a "Free Shipping" option exists, it is prioritized at the very top of the list and selected by default.
-   **Real-Time Updates:** If a customer changes their address or modifies their cart, the system dynamically recalculates the rates and reapplies the sorting and selection logic.
-   **No Setup Required:** This functionality is built-in and active by default across all checkout flows without needing manual configuration.

![](https://assets2.modalsupport.net/1775223151800-image3.jpg)

![](https://assets2.modalsupport.net/1775223174659-image11.jpg)

**5\. Set the Checkout Page Redirection**

-   On your checkout page, set the **Sale Action** to “Go to Page” and select the upsell page. This will ensure customers are directed to the upsell page after their initial purchase.

![](https://assets2.modalsupport.net/1764183572496-image2.jpg)

**PLEASE NOTE:** also offers an autocomplete address field during checkout, which uses Google's Street Address API to autofill customer addresses, making the checkout process smoother and faster. This is compatible with all checkout flows and is designed to increase completion rates by reducing the time and effort needed during checkout. This is already live, and users can enable it by simply toggling it on in their checkout settings. 

![](https://assets2.modalsupport.net/1770921121818-image3.jpg)

Important: This is a paid feature. Please reach out to your agency for pricing details.

**(Important) Add a Skip Button for the Upsell Page**

-   Add a **Skip Button** to the upsell page that links to the Thank You page. This gives customers the option to skip the upsell if they choose.

**Important Note:** Be sure to add a "skip" option for the upsell step. Without this, users will be required to purchase an upsell item to proceed, potentially leaving them stuck on the upsell page with no way to continue if they don’t want to make an additional purchase.

### **Using PayPal Pay Later** 

**PayPal Pay Later** is a payment option for order forms and online stores.

-   **Applicable For:** Both 1-step and 2-step order forms, as well as upsell purchases.
-   **Availability:** Only for one-time purchases, not for recurring products.
-   **Display:** The Pay Later button will appear below the PayPal button during checkout.

**_Notes_**

-   **Separate Order Creation**: Each upsell product will create a separate order when purchased.
-   **Shipping Options for Upsell Products**: Store owners can enable or disable shipping charges for upsell items, allowing customers to choose from available rates if enabled.
-   **Skip Button Requirement**: Adding a skip button is crucial to allow customers to bypass the upsell page if they prefer not to purchase any upsell items.
-   **Best Practices**: For an effective upselling experience, it’s recommended to create a dedicated page specifically featuring the upsell element.

### **Pro Tips**

-   **Curate Relevant Upsells**: Choose products that complement the main purchase for better conversion rates.
-   **Optimize Upsell Descriptions**: Keep the descriptions clear and concise, focusing on how the product adds value to the customer’s main purchase.
-   **Track Upsell Performance**: Regularly review the performance of your upsell pages to adjust products and offers as needed for optimal results.

### FAQs




**Question:** Can I enable or disable shipping for upsell products individually?
**Answer:** Yes, store owners can control shipping options for each upsell product, allowing customers to select their preferred rate from a dropdown menu in the upsell component.




**Question:** What happens if a customer chooses to skip the upsell?
**Answer:** The customer will be redirected to the Thank You page without purchasing an upsell product. Adding a skip button ensures they have this option.




**Question:** Does the upsell feature support multiple upsell pages?
**Answer:** Yes, you can create multiple upsell pages, each with unique upsell products, and chain them by setting the redirection after each upsell purchase.




**Question:** Is a new order created when an upsell product is purchased?
**Answer:** Yes, each upsell purchase is processed as a separate order from the original transaction.




**Question:** How can I set up an upsell funnel with multiple upsell products?
**Answer:** To create an upsell funnel, configure each upsell’s “Sale Action” to redirect to the next upsell page. Set the final upsell page to redirect to the Thank You page.




**Question:** Can I preview all text changes?
**Answer:** Not all changes, like error messages, appear in the preview but will update once published.




**Question:** How does the autocomplete address feature work during checkout?
**Answer:** The autocomplete address feature streamlines the checkout process by using Google's Street Address API to autofill customer addresses. This reduces the amount of typing required from customers, potentially increasing the completion rate of transactions. This feature can be enabled or disabled in the checkout settings.




**Question:** How does the autocomplete address feature work during checkout?
**Answer:** The autocomplete address uses **Google’s Street Address API** to autofill customer addresses, reducing typing and making the checkout process faster. This can be toggled on or off in the checkout settings.


**Question:Can I use PayPal "Pay Later" for subscription-based products or recurring payments?
**Answer:** No. PayPal Pay Later is only available for one-time purchases. It is not compatible with recurring products or subscriptions.
