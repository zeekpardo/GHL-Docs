**Customizing The Dashboard**

Customizable widgets enable opportunity filtering on dashboards, improving analysis and insights into opportunity statuses. This guide covers creating a dashboard with the Lost Reason filter and advanced customization options like titles, text boxes, images, and source filters. By the end, you'll be ready to enhance your dashboard's functionality and drive better decision-making.

**How-to Steps:**

-   **Accessing Widgets Settings:**
-   Navigate to the dashboard settings in your account.

![](https://assets2.modalsupport.net/1773259391780-image11.png)

-   Go to the 'Widgets' section.

 **Adding a Lost Reason Filter:**

-   Click on 'Add Widget' to create a new widget.
-   Select 'Opportunities' as the data source.
-   Choose the 'Lost Reason' filter from the available options.
-   Customize the filter by selecting specific reasons for lost opportunities.

![](https://assets2.modalsupport.net/1773259414322-image20.png)

 **Customizing the Widget Display:**

-   Select the type of visualization for your widget (e.g., pie chart, table).
-   Configure the widget settings to display the filtered data effectively.
-   Save the widget to add it to your dashboard.

 **Using Advanced Settings:**

-   Access advanced settings to further customize the widget.
-   Set additional filters and parameters to refine the displayed data.
-   Apply the settings and save the changes.

![](https://assets2.modalsupport.net/1773259437300-image7.png)

### **Opportunity Widgets: Attribution and UTM Parameters**

Opportunity widgets give users deeper insights and more control over their sales data. With Attribution and UTM parameters, you can filter and analyze opportunities based on the first or last attribution, providing a clearer picture of which channels and campaigns drive your sales efforts.

**Benefits:** These features allow you to create detailed widgets using various UTM properties like UTM Campaign, UTM Source, and more, enabling you to generate customized reports and visualizations. 

**Navigation:** Enter Edit mode on your dashboard, add a widget under the Opportunity category, switch to the condition tab, and configure your attribution parameters.  


![](https://assets2.modalsupport.net/1773259472741-image3.png)

**Note: While adding or editing an appointment widget, you can also navigate to the Conditions tab in the same way. It also has the option to apply the Rescheduled filter totrack rescheduled appointments in both your dashboards**

### **Widget Duplication**

With the widget duplication feature, you can efficiently replicate existing dashboard widgets to save time and streamline customization. This functionality is ideal for quickly creating variations of similar widgets without starting from scratch. Simply duplicate the widget, make adjustments, and save your changes.

**How to Use:**

-   Enter **Edit Mode** for your dashboard.
-   Locate the widget you wish to duplicate.
-   Open the three-dot menu on the widget card.
-   Select **Duplicate** and make any necessary adjustments.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdUVDlsuOzOixcMkNOueCwDyo59y1qRrKweybhmvkoKGP3EB6Yl-eaSKrcFWj6-KdDRSCr5vp9HK_lGR27JxifE3Uftoz63JcoG0cKfXsGYz1pFu-AeoXBJBfxoXXUmi4hR-kBm-w?key=fcYegJwndvpKA2e7YMtcdg)

-    Click **Save** to apply the changes.

**Requirements and Limitations:**

-   Ensure you have editing permissions for the dashboard.
-   The duplication feature only works within the same dashboard.

**FAQ/Use Cases:**




**Question:** Can I apply multiple filters to a single widget?


**Answer:** Yes, you can apply multiple filters to a single widget to narrow down the data and gain specific insights.




**Question:** What types of visualizations are available for widgets?


**Answer:** You can choose from various visualization types, such as pie charts, bar graphs, and tables, to best represent your data.




**Question:** How do I edit an existing widget?
**Answer:** To edit an existing widget, go to the dashboard settings, select the widget you want to modify, and update the settings as needed.  




**Question:** Can I create Custom Metrics using only one data source?
**Answer:** Yes, you can create metrics from a single source if preferred.  




**Question:** Can I modify a custom metric after it's created?
**Answer:** Yes, you can edit the custom metric and update the formula as needed.  




**Question:** Do I need any special permissions to edit widgets?
**Answer:** Yes. You must have **editing permissions** for the dashboard to add, edit, or duplicate widgets.




**Question:** How does widget duplication work?


**Answer:** You can duplicate a widget within the same dashboard to save time. Open the widget’s three-dot menu, select **Duplicate**, make adjustments, and save.




**Question:** Can I add a hyperlink to my dashboard?

**Answer:** Yes. Within a **Textbox** element, you can highlight text and add a hyperlink. This is perfect for linking to external training documents, shared spreadsheets, or your company’s main website.
