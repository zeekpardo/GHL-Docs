**Default dashboard**

Default Dashboards allow administrators and team leaders to set a specific dashboard as the first point of contact for all users within the platform. This ensures that every team member starts their interaction with the most relevant and critical information, tailored to meet the organization's immediate needs.

### **Core Features:**

-   **User-Controlled Default Settings:** Administrators have the flexibility to select and set the most appropriate dashboard as the default, ensuring that it aligns with current business priorities and objectives.
-   **Instant Visibility:** The selected Default Dashboard is the first interface that appears when users log in or navigate to the dashboard section, offering instant access to vital metrics and insights.
-   **Enhanced Data Consistency:** By standardizing the initial view, organizations can ensure consistency in the data and insights accessed by the team, leading to more uniform decision-making processes.

## **How to Set a Default Dashboard**

Setting a Default Dashboard is a straightforward process designed to be user-friendly and quick, allowing for immediate improvements in data interaction and visibility.

### **Step-by-Step Guide:**

Navigate to Your Dashboard: Open the dashboard section within your platform.

Select the Dashboard: Identify and open the dashboard you wish to set as the default.

Adjust Settings: Click on the three-dot menu in the top right corner of the dashboard interface.

Set as Default: Choose the "Set as Default" option from the menu.

## **Practical Applications and Benefits**

### **For Customer Support Centers**

Default Dashboards can be set to display real-time customer service metrics and KPIs, ensuring that support teams are immediately aware of priority issues and performance statistics at the start of their shift.

### **For Sales Teams**

A dashboard showing up-to-date sales figures lead conversion rates, and other critical sales data can be set as the default to keep the team aligned with sales targets and performance benchmarks.

### **For Marketing Departments**

Marketers can benefit from a Default Dashboard that highlights campaign performance, engagement metrics, and ROI data, enabling them to quickly assess and adjust ongoing marketing strategies.

## **Note:**

-   Permissions Requirements: Only users with administrative or full access privileges can set or change the Default Dashboard.
-   Deletion Restrictions: A Default Dashboard cannot be deleted unless another dashboard is designated as the default, ensuring continuity in data presentation.
-   Pinned vs. Default Dashboards: While Default Dashboards serve as the primary interface for all users, pinned dashboards are user-specific favorites for quick access.

## FAQs




**Question:** What is a Default Dashboard?

**Answer:** A Default Dashboard is a dashboard set by administrators or team leaders as the primary interface for all users within the platform. It ensures that every team member starts their interaction with the most relevant and critical information tailored to meet the organization’s immediate needs.




**Question:** How do I set a Default Dashboard?

**Answer:** To set a Default Dashboard, navigate to the dashboard section within your platform, open the desired dashboard, click on the three-dot menu in the top right corner, and select "Set as Default." Confirm the selection in the popup that appears, and the dashboard will be set as the primary landing page for all users.




**Question:** Who can set or change the Default Dashboard?

**Answer:** Only users with administrative or full access privileges can set or change the Default Dashboard. This ensures that the selected dashboard aligns with the organization’s business priorities and objectives.




**Question:** Can I delete a Default Dashboard?

**Answer:** A Default Dashboard cannot be deleted unless another dashboard is designated as the default. This restriction ensures continuity in data presentation and prevents any disruption in access to key metrics and insights.




**Question:** What is the difference between a Default Dashboard and a Pinned Dashboard?

**Answer:** A Default Dashboard serves as the primary interface for all users within the platform, while pinned dashboards are user-specific favorites for quick access. Default Dashboards ensure consistency in data presentation across the organization, whereas pinned dashboards allow individual users to quickly access dashboards that are most relevant to them.




**Question:** What are the benefits of using a Default Dashboard?

**Answer:** Default Dashboards provide instant visibility to critical metrics, enhance data consistency across the organization, and ensure that all team members start their day with the most relevant information. This helps align teams with organizational priorities and improves decision-making processes.




**Question:** How does a Default Dashboard enhance data consistency?

**Answer:** By standardizing the initial view for all users, a Default Dashboard ensures that every team member accesses the same data and insights. This uniformity leads to more consistent decision-making processes and alignment with the organization's goals.




**Question:** Can different teams have their own Default Dashboards?

**Answer:** Typically, there is one Default Dashboard for all users. However, organizations can change the Default Dashboard periodically or as needed to align with different team needs, such as sales, support, or marketing, to ensure that the most relevant information is available at the right time.




**Question:** What are some practical applications of Default Dashboards for different teams?

**Answer:** For customer support centers, Default Dashboards can display real-time metrics and KPIs relevant to service performance. Sales teams can set dashboards with up-to-date sales figures and conversion rates. Marketing departments can use Default Dashboards to monitor campaign performance, engagement metrics, and ROI data.




**Question:** What should I do if I can’t see the option to set a Default Dashboard?

**Answer:** If you do not see the option to set a Default Dashboard, you may not have the required administrative or full access privileges. Contact your system administrator or team leader to assist with setting the Default Dashboard.




**Question:** How does setting a Default Dashboard improve team alignment?

**Answer:** Setting a Default Dashboard ensures that all team members start their interaction with the same critical information, aligning everyone with current business priorities and goals. This shared view helps keep teams focused and working towards the same objectives.




**Question:** Can I set different Default Dashboards for different user roles?

**Answer:** Most platforms allow only one Default Dashboard for all users. However, some systems may support role-based dashboards, letting different user groups see different default views.




**Question:** Can Default Dashboards improve decision-making?

**Answer:** Yes. By showing consistent, up-to-date data to everyone, teams can make informed decisions faster.




**Question:** What happens if I try to delete the dashboard currently set as the Default?

**Answer:** The system includes a **Deletion Safeguard**. You will not be able to delete a dashboard while it is designated as the "Default." You must first set a different dashboard as the new default before the "Delete" option becomes available for the original one.
