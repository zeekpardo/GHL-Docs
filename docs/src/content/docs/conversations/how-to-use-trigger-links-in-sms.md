**How To Use Trigger Links in SMS**

In this article, you will learn how to use trigger links in SMS. 

## Option 1: Through a Conversation

-   Go to the Conversations tab
-   Search for the lead you want to write to
-   Click on the SMS tab to send an SMS
-   Click on the icon to add links.

![](https://assets2.modalsupport.net/1763567690652-image.png)

-   Click on Trigger Links and select the one you need

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfli8Vntns4F_Kqo7capf_e2EkAuCkGDepUZlb7KiE6Y5SrdI2gR2YejWUd1SOD2j5dbaSbaMEvXWg_ASUIVj4T8eNekLfoKIglG4sYl_FJvlWekMxhaI3_JxUP8E9Sl37RtjZjLw?key=s-biSdGAlkD5FBkEx34u8w)

-   Once you send the message, the trigger link custom value will be replaced by a link

**Note:** Trigger links can not be short links

## Option 2: Through a Workflow

-   Edit your workflow and add an SMS action
-   In there, click on the Trigger link tab and select the trigger link you want to use

![](https://assets2.modalsupport.net/1763567761573-image.png)

-   Save once everything is ready

## Option 3: Through an SMS Template

-   Go to Marketing 
-   Click on Templates
-   Click on Add Text Template
-   Click on the Trigger link tab and use the one you need to

![](https://assets2.modalsupport.net/1763567832821-image.png)

-   Save when everything is ready

## FAQs 




**Question:** What are trigger links?  
**Answer:** Trigger links are unique URLs used in SMS campaigns that can be customized to track and respond to specific user actions. They help in automating responses based on user interactions.




**Question:** Can I use trigger links in short links?  
**Answer:** No, trigger links cannot be short links. They must be used in their full form.




**Question:** How do I insert a trigger link through a conversation?  
Answer:**

1.  Go to the Conversations tab.
2.  Search for and select the lead you want to message.
3.  Click on the SMS tab.
4.  Click the icon to view available trigger links.
5.  Select the desired trigger link.
6.  Send the message, and the trigger link will be included as a full URL.




**Question:** How can I add a trigger link to a workflow?  
Answer:**

1.  Edit your workflow and add an SMS action.
2.  Click on the Trigger link tab.
3.  Select the trigger link you wish to use.
4.  Save your workflow changes.




**Question:** How do I use a trigger link in an SMS template?  
Answer:**

1.  Go to Marketing and click on Templates.
2.  Click on Add Text Template.
3.  Click on the Trigger link tab.
4.  Choose the appropriate trigger link.
5.  Save the template when finished.




**Question:** Are there any restrictions on trigger links?
**Answer:** Yes, trigger links cannot be used as short links. They must be in their full URL format.




**Question:** Can I use the same trigger link across multiple conversations or workflows?
**Answer:** Yes. The same trigger link can be reused across different SMS messages, workflows, or templates as needed.  




**Question:** What happens to the trigger link custom value after I send the SMS?
**Answer:** Once the message is sent, the trigger link custom value is automatically replaced by the actual clickable link.




**Question:** Can I put a Trigger Link inside an Image (MMS)?
**Answer:** No. Trigger Links must be part of the text body of the SMS. They function as text-based URLs so that the phone's operating system can recognize them as clickable links.




**Question:** Why does the link look long and random when the SMS is sent?

**Answer:** When you use a trigger link, the system generates a unique tracking ID for that specific contact. This is how the system knows _exactly_ who clicked the link. While it may look longer than a standard URL, this unique string is essential for firing off any associated "Trigger Link Clicked" workflows.




**Question:** Can I see who clicked my link without a workflow?

**Answer:** Yes. You can view individual click activity within the **Contact Timeline**. Additionally, many of the attribution reports in the **Reporting** tab will show "Trigger Link" engagement across your campaigns.
