Adding attachments to your emails or SMS can be of great help to your business, especially by doing it quickly from the Conversation channel and by not having to worry about the file size. In this article, you will learn how to upload large files directly from your devices.

### FAQs




**Question:** How can I attach a file to an email or SMS message?
**Answer:** To attach a file, go to the Conversations channel, select the contact you want to send the file to, and either drag and drop the file into the conversation or use the clip icon to attach it.




**Question:** What is the maximum file size I can send via SMS?
**Answer:** The total size of the message body text and all attachments must be less than 5 MB for SMS. Please note that your phone number must be MMS-enabled to send attachments.




**Question:** Can I send multiple attachments in one SMS?
**Answer:** Yes, you can send up to 10 attachments in one SMS, as long as the total size does not exceed the 5 MB limit.




**Question:** Do Toll-Free numbers support attachments via MMS?
**Answer:** Yes, Toll-Free numbers support MMS, and files up to 5 MB can be sent depending on the carrier.




**Question:** What happens when I upload a video attachment in an email?
**Answer:  If the video is added through the Media Library, a clickable preview thumbnail (GIF) is automatically created and embedded in the email.




**Question:** What should I do if my file exceeds email limits?
**Answer:** Files above 20MB are automatically uploaded to the Media Library and inserted as links in the email, ensuring successful delivery.




**Question:** Can I upload audio files for SMS?
**Answer:** While not universally supported, some carriers allow audio formats like MP3. These will convert to MMS where supported.




**Question:** Is external cloud hosting needed to send large files?
**Answer:  No, larger files are automatically handled via the Media Library and added as links. External hosting is only necessary if sharing outside the platform.




**Question:** Does sending many attachments affect deliverability?
**Answer:** For email, excessive or large attachments may trigger spam filters, so using Media Library links is recommended.




**Question:** Is there a limit to how many files I can store in my Media Library?

**Answer:** Your Media Library is designed to scale with your business. While there isn't a strict "count" limit, we recommend periodically cleaning out old or unused large assets to keep your library organized and easy to search.

For SMS/MMS, exceeding limits will cause message failure.




**Question:** Can I subtract points for negative behaviors?

**Answer:** Yes. The scoring system allows for both addition and subtraction. For example, you can add points for a successful payment but create a rule to **subtract points** if a contact enables "Do Not Disturb" (DND) for all channels.




**Question:** What happens if my file is too large for a standard email attachment?

**Answer:** Any file exceeding the **25 MB** email limit (or **5 MB** for SMS) is automatically uploaded to your Media Library. Instead of a raw attachment, the recipient will receive a secure, clickable link to download the file directly.
