**4\. Facebook Story:** For Facebook Stories, users can add a single image or video with these requirements:

-   Image Format:  JPEG, PNG, or GIF format
-   Image Size: Maximum 4 MB
-   Video Format: Supports all formats
-   Video Size: 1 GB
-   Video Duration: Minimum 3 sec | Maximum 90 sec
-   Video width: Minimum Resolution of 540 x 960 px.

**Facebook Stories do not support thumbnails or OG Meta Tags.**

**Note:**  
Various sizes and types of images work for different things. Two ‘go-to’ sizes are:

-   **Horizontal (landscape): 1,024 pixel X 512 pixel**
-   **Vertical (portrait): 800 pixel x 1,200 pixel**

### FAQs




**Question:** What is the optimal video format and maximum file size for Facebook Pages and Groups?
**Answer:

-   **Format:** MP4, MOV
-   **Maximum File Size:** 1 GB




**Question:** What are the image size recommendations for Instagram Business Profiles?

**Answer:

-   **Format:** PNG, JPG
-   **Maximum File Size:** 8 MB
-   **Recommended Sizes:**
    -   Landscape: 1080 x 566 pixels
    -   Portrait: 1080 x 1350 pixels
    -   Square: 1080 x 1080 pixels




**Question:** What are the requirements for video uploads on TikTok?
**Answer:

-   **Format:** MP4
-   **Maximum File Size:** 50 MB
-   **Video Duration:** 3 sec - 60 sec
-   **Resolution:** 540 pixels




**Question:** What is the content length limit for a Facebook Page post?
**Answer:

-   **Character Limit:** 62,000 characters




**Question:** What are the specifications for video uploads on Instagram Reels?

**Answer:

-   **Format:** MOV, MP4
-   **Maximum File Size:** 1 GB
-   **Video Duration:** Minimum 3 sec - Maximum 15 minutes
-   **Video Width:** 1920 px
-   **Video Thumbnail Format:** All formats
-   **Video Thumbnail Size:** 8 MB




**Question:** Can I post multiple images or videos in a single post (multimedia)?
**Answer:

-   **Instagram Carousels** and **Community posts** support multimedia (images + videos).
-   Most other platforms, such as Facebook, LinkedIn, TikTok, GBP, and Pinterest, **do not support multimedia posts**.




**Question:** What is "Multimedia format," and where can I actually use it?

**Answer:** Multimedia format refers to a single post that contains both images and videos (e.g., 2 photos and 1 video). Currently, this is only supported for Instagram Carousels and Community posts. If you try to post a mix of images and videos to Facebook or LinkedIn, the post will likely fail or only post one media type.




**Question:** What are the video requirements for a LinkedIn post?

**Answer:** LinkedIn supports MP4 and MOV formats with a maximum file size of 200 MB. The duration must be between 3 seconds and 30 minutes, with an aspect ratio between 1:2.4 and 2.4:1.
