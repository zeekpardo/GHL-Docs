**How to Create and Utilize the Group Hashtag Feature**  

This Article goes over how to create and utilize the Group Hashtag feature for ease in Social Media posting. This saves you time, so that with each post you don’t need to add each individual hashtag every single time.

### Step 1: Create a Hashtag

-   Go to the 'Marketing’ tab' >  'Social Planner'
-   Click on the 'Create New Post' button

-   Add hashtags '#'

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcvKKTwXakuy_0BsfgUCp2ro2Pp6eAEEAIv4uM9q_J_R0zmogSBwOuTT6DPawvy7RqBeDoiEXMJ4qwXSuFPjhQJpbMgoeVP-VDdOYw6CIdIOZXmZI9hgSUZ2_3NtIOsK9ONIxij?key=D3S59R5-8LelkqhPhF_rKg)

-   Add a Name for the set of Hashtags and which Hashtags you want to include. (There is an option to add several hashtags when the post is created).

-   Once the hashtags are added, click on 'Save and Apply' to save the hashtag group. 

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeLaQtnhMjPEkQHP6XBnNvYmpMToCqmHEOsegeg3tkwOehBc0VJjgsov6pytyAt4o_aNfJ4DJQckv-2bxzC7kKmuiV6fkYbMvYdqldkeU3sILyls6Om80epAvYE7FbpE83Vh_euJA?key=D3S59R5-8LelkqhPhF_rKg)

Now, when you create a new Social Media post, you can choose the saved hashtag grouping that you would like to apply to this post. 

**NOTE:** The maximum number allowed is 30 hashtags in each post. 

### Importance of the Group Hashtag Feature

**1\. Time Efficiency**Using pre-defined hashtag groups saves you time and eliminates repetitive tasks, allowing you to focus on other aspects of your social media strategy.

**2\. Consistency**Applying consistent sets of hashtags helps build a recognizable brand identity and improves your content’s discoverability across different platforms.

### FAQs




**Question:** What is the maximum number of hashtags I can use in a post?Answer**: You can use a maximum of 30 hashtags in each post.




**Question:** Can I edit an existing hashtag group?Answer**: Yes, you can edit an existing hashtag group by navigating to the Social Planner, selecting the group you want to modify, and making the necessary changes.




**Question:** How do I delete a hashtag group?Answer**: To delete a hashtag group, go to the Social Planner, select the hashtag group you wish to remove, and choose the delete option.




**Question:** Can I use different hashtag groups for different social media platforms?Answer**: Yes, you can create and save different hashtag groups tailored for each social media platform to optimize your post visibility.




**Question:** Is there a limit to the number of hashtag groups I can create?Answer**: There is no specific limit mentioned, but it's advisable to create and manage your hashtag groups based on your needs and the relevance of your content.




**Question:** Can I apply a saved hashtag group to a new post?Answer**: Yes. When creating a new social media post, you can select a saved hashtag group and apply it directly to the post.




**Question:** Do Group Hashtags apply automatically to every post?Answer**: No. Group Hashtags must be selected manually for each post. They will not apply automatically unless chosen during post creation. It **saves time** by avoiding repetitive hashtag entry and ensures **consistency**, which improves brand recognition and content discoverability across platforms.




**Question:** How does the Group Hashtag feature improve my brand identity?
**Answer:** By ensuring **Consistency**. Applying the same sets of pre-defined hashtags across your posts helps build a recognizable brand presence and ensures your content is consistently discoverable in the same niches across different social platforms.  




**Question:** When creating a hashtag group, am I limited to only one hashtag per set?
**Answer:** No. When you add a name for your hashtag set, there is an option to include several hashtags at once within that specific group.
