**​**When testing a campaign email, you may receive a popup notification with an error message that states "An error occurred, please try again." The message will look like this:

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdGnATlacCZPuKQVJccYCLs10yamdz5DxqTtc1emV7i7OAKuUrmf6WZIFJOB-3DrZ7NxzjE8e0ejwZ6JDtJ3iFr9BmkeDEUixMOZH0DD6OfckWxDXHkJGHlHsrhJtfJ-6Rafe0DkMTB1qtcYyPXlBKg4ak?key=gypLeaxtMaIz4SInBc0bLw)

### Check the following to ensure your email test can be sent:

1.  Ensure the custom values being used are valid. Be sure that custom values have proper brackets {{ }} (not just 1 {} or more than 2 {{{}}}). For example, {{contact.name}} or {{contact.city}}.
2.  Check the email address to ensure it follows the proper format such as test@emaildemo.com
