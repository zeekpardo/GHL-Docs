**FAQ**




**Question:** How do I view hidden elements inside my email template?
**Answer:** Click on the 'Manage Element' icon, then use the 'Section' and 'Column' dropdowns to view all elements within your email. Hovering over an element in the list will scroll to and highlight it in the email builder.




**Question:** What should I do if I see a broken image icon in my email?
**Answer:** Use the 'Manage Template' menu to hover over the broken image icon. You can edit or delete the image from the dropdown options. Check visibility settings or correct the image height as needed.




**Question:** How can I fix an image that is not displaying due to height issues?
**Answer:** Edit the image by increasing its height or setting both height and width to auto. If the issue persists, adjust the size using an image editor, delete the image from the email template, and re-add it.




**Question:** What if I cannot edit an element in my email template?
**Answer:** Ensure you are using the 'Manage Template' menu to access and modify hidden or non-editable elements. Use the dropdowns to locate and edit the element as needed.




**Question:** What happens when I hover over elements in the builder?

**Answer:** Hovering over elements will scroll you to their location in the builder, allowing easy identification and editing of hidden or broken items.




**Question:** Why can't I see certain elements in my email template?
**Answer:** Some elements may be hidden due to visibility settings or size constraints (like height being below 10px). Use the 'Manage Template' menu to check these settings and adjust the element's visibility or size accordingly.




**Question:** How can I prevent hidden or broken elements in future templates?
**Answer:** Regularly review templates using the Manage Element panel, avoid extreme size constraints, and test emails across devices before reuse or duplication.




**Question:** What if an image shows a broken icon only in the 'Sent' email but looks fine in the builder?

**Answer:** This usually means the image URL is not public or has been deleted from the **Media Library**. Even if the builder "cached" the view, the recipient's email provider cannot fetch it. Use the Manage Element tool to select the image and re-upload it to ensure a fresh, public link.




**Question:** Can I adjust the size of the Play Button on my video preview?

**Answer:** Yes. Within the video element settings, you have the ability to make changes to both the size and the opacity of the Play Button to ensure it fits your design preferences.
