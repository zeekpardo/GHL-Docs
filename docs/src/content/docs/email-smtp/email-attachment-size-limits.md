In this article, you will learn the file size limit of the attachments you can add to an Email. 

All email server providers limit the maximum email size that an email account can receive. 

If you send an email that exceeds the maximum email size limit of the recipient’s email provider, your email will bounce off, and you will receive an email error back in your Conversations tab. Usually, you will receive one of the following error messages:

-   “Attachment size exceeds the allowable limit”;
-    “552: Message size exceeds maximum permitted”;
-    “System Undeliverable, message size exceeds outgoing message size limit”;
-    “The size of the message you are trying to send exceeds the global size limit of the server. The message was not sent; reduce the message size and try again”.
-   Some email providers enforce commercial limitations, offering a low email size limit on free accounts, but users can increase the limit by upgrading to a paid account.
-   For security reasons, email server providers limit the maximum size of attachments to prevent large files from overwhelming their systems.
-   Cloud storage services like Dropbox, Google Drive, or OneDrive can be used to upload large files, allowing you to include a download link in your email.
-   Mailgun supports a maximum message size of 25MB.

#### **FAQs**




**Question:** What is the maximum file size I can attach to an email?
**Answer:** The maximum file size for attachments depends on the email provider. For instance, Mailgun supports a maximum message size of 25MB.




**Question:** What happens if my email exceeds the file size limit?
**Answer:** If your email exceeds the size limit, it will bounce back with an error message, and you will receive a notification in your Conversations tab.




**Question:** How can I send larger files if they exceed the email attachment limit?
**Answer:** Upload the files to cloud storage services like Dropbox, Google Drive, or OneDrive, and include the download link in your email.




**Question:** What improvements have been made to the file upload process?
**Answer:** Users can now upload larger files directly from their device or the media library, and dynamic video thumbnails are generated for improved email engagement.




**Question:** How does the new upload feature benefit me?
**Answer:** The new feature simplifies the process of attaching large files and enhances email content with automatic video thumbnails, improving both efficiency and engagement.




**Question:** Can I track if my recipient has accessed the file uploaded through a cloud link?
**Answer:** While traditional email systems don’t track link clicks, some cloud services (like Dropbox or Google Drive Business) provide **link analytics** to monitor views and downloads.  



**Question:** What happens if my email attachment exceeds the maximum file size limit for my email provider?
**Answer:** If your email exceeds the maximum file size limit, it will bounce back, and you will receive an error message like "Attachment size exceeds the allowable limit" or "Message size exceeds maximum permitted." You'll also get a notification in your Conversations tab.




**Question:** What happens if I try to attach a file larger than 20MB directly to an email?  
**Answer:** Within , any email attachment that exceeds 20MB is automatically uploaded to your media library. Instead of a bulky attachment that might cause the email to bounce, the system seamlessly inserts a secure download link into your email body for the recipient.




**Question:** Is there a limit to how many individual files I can attach?
**Answer:** There is no hard limit on the _number_ of files, but the **combined size** of all attachments must remain under the limit. If you have ten 3MB files, the total (30MB) will exceed the limit and trigger the automatic link enhancement.




**Question:** What happens when I upload a video to the media library to be used in an email?

**Answer:** When you upload a video via the media library for an email, the system automatically generates a GIF thumbnail for that video. This enhancement is designed to make your emails more visually appealing and significantly boost user engagement.
