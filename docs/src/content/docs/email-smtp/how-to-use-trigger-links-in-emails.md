Trigger links are a great way to pivot off a "click" within an Email and simultaneously direct the contact to a specific website. When a trigger link is clicked, the system will attribute the click to the contacts' activity timeline and trigger a predetermined action within a workflow created for this purpose.

## There are four primary locations where a trigger link can be used for emails:

-   Contacts
-   Conversations
-   Workflows
-   Email Campaigns/Templates

### Part 1: Using Trigger Links from Contacts/Conversations

-   Navigate to Contacts and click on a specific contact.

-   Select the email tab.

-   Alternatively, navigate to “**Conversations**” to view a specific conversation, and select the email tab. Click on the icon at the bottom of the screen to add a Trigger Link.

![](<https://assets2.modalsupport.net/1765279814685-unnamed - 2025-12-09T193005.287.jpg>)

-   A window will open. Select “Trigger Link.” Choose the trigger link from the dropdown menu.

![](<https://assets2.modalsupport.net/1765279832144-unnamed - 2025-12-09T193024.497.png>)

The **Web Address** will auto-populate when you select the trigger link.

The **Link Text** will give a hyperlink to the word or phrase for the trigger link.

![](<https://assets2.modalsupport.net/1765279920007-unnamed - 2025-12-09T193150.845.jpg>)

**Note:** If you need to create a trigger link, navigate to Marketing > Trigger Links > Links and add a link.

Alternatively, you can also select trigger links by clicking on the flash icon (shown in the screenshot below)

**Please note:** This method does not allow hyperlinking a trigger link at this time.

![](<https://assets2.modalsupport.net/1765279942418-unnamed - 2025-12-09T193213.394.jpg>)

### Part 2: Using Trigger Links from Workflows

-   Navigate to Automation > Workflows and hit the “+ Create Workflow” button or edit an existing workflow

![](<https://assets2.modalsupport.net/1765279962306-unnamed - 2025-12-09T193233.551.png>)

-   Add or edit the Send Email Action

-   You will see a Trigger Links Dropdown that you could use the same way as before

![](<https://assets2.modalsupport.net/1765279989306-unnamed - 2025-12-09T193255.329.jpg>)

-   Ideally, type a word, highlight it, and select the add link icon to hyperlink it.

![](<https://assets2.modalsupport.net/1765280137724-unnamed - 2025-12-09T193527.621.png>)

-   Select the trigger link from the dropdown, and it will auto-populate the URL value. Edit as preferred

![](<https://assets2.modalsupport.net/1765280158904-unnamed - 2025-12-09T193550.117.png>)

-   Save the link, action, and workflow

### Part 3: Using Trigger Links from the Email Builder

The email builder is an editor for email campaigns and email templates

-   Navigate to **Marketing > Emails > Campaigns or Templates** and edit or create one

![](<https://assets2.modalsupport.net/1765280184985-unnamed - 2025-12-09T193617.194.png>)

-   Add or edit an existing Text Element by clicking on it

-   Similar to the use in Automation you can hyperlink or use the Trigger Link directly. Or hyperlink the link by typing a word, highlighting it, and selecting the hyperlink icon.

![](<https://assets2.modalsupport.net/1765280210924-unnamed - 2025-12-09T193642.568.png>)

![](<https://assets2.modalsupport.net/1765280222393-unnamed - 2025-12-09T193652.504.png>)

-   Remember to save any changes.

**Pro Tips**

1.  **Organize Trigger Links:** Keep a well-organized list of trigger links with clear, descriptive names. This helps in quickly identifying and selecting the right link when setting up emails, workflows, or campaigns.
2.  **Test Before Deployment:** Always test trigger links to ensure they correctly direct to the intended web address and that the associated workflow actions are properly triggered. This helps avoid errors and ensures a smooth user experience.
3.  **Use Descriptive Link Text:** When creating trigger links, use descriptive link text to make it clear what action the link will trigger. This improves transparency and can help reduce confusion for recipients.
4.  **Monitor Link Performance:** Regularly review click-through rates and performance metrics for trigger links to assess their effectiveness and make adjustments to your strategy as needed.

**FAQs**




**Question:** What is a trigger link?
**Answer:** A trigger link is a hyperlink used in emails that, when clicked, attributes the click to a contact’s activity timeline and triggers a predetermined action within a workflow.




**Question:** How do I use trigger links in contacts or conversations?
**Answer:** Navigate to the Contacts or Conversations section, select a contact or conversation, and use the email tab to add a trigger link. Choose the desired trigger link from the dropdown menu, and the web address will auto-populate.




**Question:** How do I add trigger links to a workflow?
**Answer:** Go to Automation > Workflows, create or edit a workflow, and add or edit the Send Email action. Use the Trigger Links dropdown to select and auto-populate the URL value. Save the changes to the link, action, and workflow.




**Question:** How can I add trigger links in the email builder?
**Answer:** In the email builder, navigate to Marketing > Emails > Campaigns or Templates. Edit or create a text element, and hyperlink it with the trigger link by selecting the hyperlink icon and choosing the desired trigger link from the dropdown.




**Question:** Where do I go if I haven't created any trigger links yet?

**Answer:** If you need to set up a new link from scratch, navigate to **Marketing > Trigger Links > Links**. Once you add a link there, it will become available in the dropdown menus across Contacts, Workflows, and Email Campaigns.




**Question:** Why can't I hyperlink my trigger link when using the "Flash Icon" in Conversations?

**Answer:** The "Flash Icon" (Quick Actions) is designed for speed. Currently, this specific method inserts the trigger link directly and does not support hyperlinking it to a custom word or phrase at this time.




**Question:** Why does the URL look long and scrambled when I hover over it in my inbox?

**Answer:** This is normal. To track exactly _which_ contact clicked the link, the system "wraps" your original URL with unique tracking parameters. Once the contact clicks, the system records the activity and instantly redirects them to your intended destination.
