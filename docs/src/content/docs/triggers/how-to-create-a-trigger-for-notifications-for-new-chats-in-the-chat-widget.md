**Please note that triggers is a deprecated feature.**

**Follow the steps in our articles "How to Add a Chat Widget to Funnels and Websites" and "How to Add a Chat Widget to a Wordpress Site" to learn how to customize a chat widget.**

After the widget is set up, you'll need to create a trigger to send a notification that a conversation's been initiated in the widget.

**Step 1:** Navigate to Triggers > + Add Trigger. Set up the trigger as shown below:![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcn6AyNoYmXnscSSqUNQbpJYg9c53KxWpMjC9P9EuhlmQa5M7H8uzlpkuZ2ncMQ28TQ4A8POFOoWlIWKCo6-IXB4dyCHo6ndQWrZM8dCd3Btn8A8wfms3n8YdMspEjncyrrTl_mqK9GSxQPM17bdUxEiEs?key=jOts9XqC-q8vR8f4Tv4lNw)



**Step 2:** The trigger event should be "Customer Replied" and the filter for "Reply Channel" = "Chat Widget" should be enabled.

**Step 3:** For the trigger actions, use the "Message Body" custom value in a notification SMS or email. (The screenshot included uses a SMS for this example.) This custom value will display the chat widget message in the notification.

**Step 4:** Save and activate your trigger.
