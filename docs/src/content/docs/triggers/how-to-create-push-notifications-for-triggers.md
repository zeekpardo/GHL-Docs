Push notifications on Triggers allow you to send an alert to yourself or your Lead when someone is added to a Campaign. You can set a custom message that will be sent along with this notification.

### **Step 1: Setting Up Trigger**

**![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXd9gOzM_5Sxsq8ggZ-sO7R4JP9cRNpAlfo794icS2Dq6dFEuNoOhkdwzoousDganVglQmyKB4NY-2b42S5ZK6NpEBwaUi2icE5RS6fHe_FzzVys_u2sbCtei_q4z83FGWITgcEIj2Uu7J7BKpWxUZQSLcU?key=5bD9lmliDkspoS3MCLIIWQ)**

-   Navigate to the Automation tab.
-   Navigate to the Triggers section.
-   You can select an existing Trigger, or click to Add New Trigger.
-   Fill out _Part 1 "What should trigger this rule?"_
    -   Choose "Added to Campaign". You can further filter to a specific campaign if you want the notification to only apply to a specific campaign. If you don't further filter it, the action will apply for all campaigns.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXczE7fw3QL3lx-P41YpATLW8tl27gMGA9CL8H35NTTW5BKn49EAkW5zsqbadE0QlFiXAjmJ50MtqvgHmytowu1HPjuBpxI6z8o-m9bZ9bBHlQEP3JMLqmqLENSZTiqvU98hGkKGFdV57rxKH-T44vOeYL_C?key=5bD9lmliDkspoS3MCLIIWQ)

-   Fill out _Part 2 "What actions should we perform?"_
-   Choose Send Notification. Fill out the form appropriately. Type in the message you wish to use.
-   Save.
-   Activate the trigger.

**_NOTE_:** you can use various actions for Part 1 to completely customize this trigger, such as "customer replied". The example above showing how to use notifications when a contact is added to a campaign is one of the most common ways this feature is used.



### **FAQ**

**1\. What are push notifications in triggers used for?**

Push notifications in triggers allow you to send alerts to yourself or your leads when specific actions occur, such as when someone is added to a campaign. These notifications can include a custom message to keep you informed or engage the lead instantly.

**2\. Can I customize the trigger to send notifications based on different actions, not just campaign additions?**Yes, you can customize the trigger using various actions in Part 1. For example, you can trigger notifications based on "customer replied," "appointment booked," or other events that are relevant to your workflow.

**3\. Who can receive push notifications set up through triggers?**Push notifications can be configured to be sent to yourself, your team members, or your leads, depending on how you set up the trigger.

**4\. What should I do if my notifications are not working as expected?**If notifications are not being sent as expected, double-check that the trigger is activated and correctly configured. Ensure the correct conditions are set in Part 1 and that the action in Part 2 is correctly configured to send the notification to the desired recipient.

**5\. Can I use custom messages in push notifications?**Yes, you can type a custom message in the notification setup. This message can include personalized details or instructions relevant to the recipient.

**6\. What other actions can be combined with sending notifications in a trigger?**You can combine multiple actions with sending notifications. For example, you can send an internal notification, update a contact’s status, and initiate an automated sequence all within the same trigger, providing comprehensive workflow automation.

**7\. What happens when I activate the trigger?**Once activated, the trigger will automatically monitor and execute the defined actions whenever the conditions are met. In this case, it will send the configured notification whenever a contact is added to a specified campaign.
