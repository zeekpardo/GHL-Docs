Quizzes are a versatile tool that can enhance engagement, gather valuable insights, and even help segment audiences based on their responses. Whether you’re an educator, fitness trainer, or marketer, quizzes can serve multiple purposes, from assessing knowledge to offering personalized recommendations. In this guide, we’ll explore how to use the Quiz Builder effectively to create and manage quizzes that meet your goals.

### Key Features & Benefits

-   **Drag-and-Drop Interface**: Easily add and organize different question types by simply dragging and dropping elements into place.
-   **Multiple Question Formats**: Customize your quiz with formats like multiple choice, true/false, short answer, and more to suit your quiz objectives.
-   **Customizable Scoring**: Tailor scoring for each question or category to gain insights or offer personalized results.
-   **Branching Logic**: Use branching to create dynamic quizzes where subsequent questions adapt based on a participant's answer.
-   **Result Customization**: Fully personalize the result page, showing participants a tailored outcome based on their responses.
-   **Easy Sharing**: Share quizzes via direct links or embed them on your site or funnel for a seamless experience.
-   **Clone Any Form Element**: Duplicate custom fields, text blocks, and image elements without rebuilding them.

**Overview of the Quiz Builder**

## How to Use Quiz Builder

### Step 1: Accessing Quiz Builder

1.  **Go to Sites > Quizzes** in the dashboard.
2.  Click **Add Quiz** and choose to create from scratch or from pre-built Templates.

![](https://assets2.modalsupport.net/1763657002873-image7.png)

### Step 2: Designing Your Quiz

**1\. Add Questions**

-   Use the sidebar to drag question types onto your quiz slides.
-   Choose from single-choice, multiple-choice, short-answer, and other formats based on your quiz needs.

![](https://assets2.modalsupport.net/1763657044833-image10.jpg)

**2\. Set Up Scoring**

-   For each question, assign scores based on answers, which can contribute to an overall score or category-specific score.

![](https://assets2.modalsupport.net/1763657101853-image8.jpg)

**3\. Customize Question Flow with Branching**

-   Use branching to skip or direct users to specific questions based on their previous answers, making the quiz more personalized.

![](https://assets2.modalsupport.net/1763657149877-image16.jpg)

**4\. Element-Level Label Alignment for Quizzes**

Element-Level Label Alignment allows users to set label positioning in quizzes, providing more precise layout control. To apply, select a field within a Single Column layout, choose the desired label alignment—Left, Right, Top, or Form Default and save the changes.

​​

![](https://assets2.modalsupport.net/1763657194092-image9.jpg)

**5\. Add a New Page and Clone, Delete, or Edit the Page  Setting**

You can add additional pages within the quiz builder. You can then modify it settings: the name of the page and its positioning. You can also clone the page or delete the additional page. The side-tray settings icon allows you to adjust page settings smoothly without losing context, especially on smaller screens.

**Zero Vertical Padding Support for Quizzes**

Quizzes support true zero padding on the top and bottom, allowing complete control over vertical spacing. This helps create cleaner, more precise designs without unnecessary whitespace, ideal for achieving pixel-perfect alignment and modern layouts.

**How to apply zero padding:**

1- Open the quiz builder.  
2- Go to **Styles** → **Layout** → **Padding**.

## FAQ




**Question:** Can I set different scores for each answer option?
**Answer:** Yes, you can assign unique scores to each answer option, which will be used to calculate the participant’s total score.




**Question:** How do I customize the result page based on quiz scores?
**Answer:** Use the **Edit Template** option on the result page to set up score-based tiers (e.g., low, medium, high). Each tier can display different colors, messages, and calls to action.




**Question:** Can I use branching to skip irrelevant questions?
**Answer:** Yes, the branching feature lets you direct participants to different questions based on their previous answers, allowing you to customize their quiz experience.




**Question:** How do I share my quiz?
**Answer:** You can share your quiz via a direct link or by embedding it on your website or funnel page using the provided embed code.




**Question:** What types of files can I ask participants to upload?
**Answer:** The Quiz Builder supports file uploads, allowing participants to submit files like PDFs or images, which can be useful for assignments or documentation requests.  




**Question:** Can I integrate Quiz Builder with my website?
**Answer:** Yes, you can embed quizzes on your website using the provided embed code or direct links.




**Question:** Can I customize the scoring system?
**Answer:** Yes, Quiz Builder allows you to assign different points to each question and set up category-based scoring.




**Question:** Does Quiz Builder support branching logic?  
**Answer:** Yes, you can create quizzes that adapt dynamically based on user responses.




**Question:** How can I analyze quiz results?
**Answer:** You can view detailed analytics and insights within Quiz Builder to track performance and engagement.




**Question:** Can I automate follow-ups based on quiz scores?  
**Answer:** Yes, using the Quiz Submitted Trigger in workflows, you can set up automated follow-ups like emails and SMS messages.




**Question:** Is Quiz Builder suitable for certification exams?
**Answer:** Yes, you can create quizzes for certifications, assign pass/fail criteria, and generate certificates accordingly.




**Question:** Is there a progress bar feature for quizzes?
**Answer:** Yes, you can enable a progress bar in quizzes to help users track their progress on mobile devices. To do this, navigate to the **Styles** section, then select **Footer**, and toggle the **Enable Progress Bar** option. You can also customize the colors and step label to match your brand.




**Question:** How do I adjust element-level label alignment in quizzes?
**Answer:** Select a field in the Single Column layout, choose label alignment (Left, Right, Top, or Form Default), and save the changes.




**Question:** How do I apply zero vertical padding in quizzes?
**Answer:** Go to Styles → Layout → Padding in the quiz builder and set top and bottom padding to zero for a clean design.  



**Question:** How do I add dynamic elements to the result page?  
**Answer:** Yes, you can include dynamic elements on the result page, such as calls-to-action for scheduling consultations, enrolling in programs, or sharing results. These elements help increase user engagement.




**Question:** Is it possible to edit the text directly on the quiz slide?

**Answer:** Yes, the builder supports **Inline Text Editing**. You can click directly on any text block or heading to change the wording, font size, or color without using a separate side menu.




**Question:** Can I show the "Correct Answer" immediately after a user picks an option?

**Answer:** Yes. In the element settings for choice-based questions, you can enable "Show Feedback." This allows you to display a custom message (and even a different color) immediately after they answer, which is excellent for educational or training quizzes.
