The Lesson Completed trigger activates a Workflow when a contact completes a lesson within a category in a membership product. It allows you to reward users, provide additional resources, or move them to the next stage in their learning path. In this article, we will go over this trigger so you can track when contacts complete a course.

### How to use:

-   To proceed with this trigger, click on Create a new workflow or edit an existing one within the automation tab.

![](<https://assets2.modalsupport.net/1762435856896-unnamed \(81\).jpg>)

-   Add the trigger for “Lesson Completed” from the list of available triggers. You can add filters to further customize the workflow, such as firing off a specific lesson. 

![](https://assets2.modalsupport.net/1775250738624-image4.png)

**_Please Note:_** _Custom values are available for email customization. If/Else fields are also available for filtering when using multiple lessons. (See Screenshots Below)_

-   Found in **custom values** > **Course** > **Lesson Completed**.

![](<https://assets2.modalsupport.net/1762435970834-unnamed - 2025-11-06T213246.634.png>)

-   Select **Courses** > **Product/ Category/ Lesson**

![](<https://assets2.modalsupport.net/1762435988889-unnamed - 2025-11-06T213303.915.png>)

-   **Test Your Workflow**: Before finalizing, run tests to ensure all triggers and conditions work as expected. This will help avoid issues in live scenarios.

![](<https://assets2.modalsupport.net/1762436011474-unnamed \(83\).jpg>)

### FAQs




**Question:** What does the Lesson Completed trigger do?

**Answer:** The Lesson Completed trigger activates a workflow when a contact finishes a lesson within a membership product's category. This helps in managing user progress, rewarding achievements, and facilitating further actions.




**Question:** Can I customize actions based on specific lessons completed?
**Answer:** Yes, you can add filters to customize the workflow based on specific lessons completed. This allows you to tailor actions and responses for different lessons within the course.




**Question:** How can I use custom values with the Lesson Completed trigger?

**Answer:** Custom values for email customization can be used to personalize communications. You can find these values under custom values > Course > Lesson Completed.




**Question:** What are If/Else fields, and how do they help with multiple lessons?

**Answer:** If/Else fields allow you to set conditions within your workflow based on the completion of different lessons. This enables more detailed and specific responses depending on which lesson was completed.




**Question:** Where do I find the Lesson Completed trigger settings?** You can find the 
**Answer:** trigger settings by navigating to the course section and selecting Product/Category/Lesson. From there, you can configure the trigger and customize actions based on lesson completion.




**Question:** How can I personalize messages based on lesson completion?
**Answer:** Use the custom values and If/Else logic to include lesson-specific details in emails or notifications, such as the lesson title, completion date, or next recommended lesson.




**Question:** If a student retakes a lesson and completes it again, will the workflow fire a second time?

**Answer:** This depends on your **Workflow Settings**. If you have "Allow Re-entry" turned ON, the workflow will trigger every time they mark the lesson as complete. If you only want the reward or action to happen once, ensure "Allow Re-entry" is turned OFF.
