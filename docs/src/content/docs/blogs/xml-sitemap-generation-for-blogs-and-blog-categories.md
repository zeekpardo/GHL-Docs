This guide will walk you through using XML Sitemap Generation, specifically designed for blogs and SEO. The update allows business owners to index URLs for their blogs in funnel or website pages, making it easier for search engines to discover and crawl content. Let's go through the steps:

## Part 1: XML Sitemap Settings

### To enable XML Sitemap for blogs, follow these steps:

-   Navigate to "Settings" and then select "Domains." Once there, select the “Manage” button of the relevant domain

![](<https://assets2.modalsupport.net/1765280914544-unnamed - 2025-12-09T194823.231.jpg>)

-   Next, look for the option titled "XML Site map" next to your domain and click on it. If needed, add the domain

![](<https://assets2.modalsupport.net/1765280936869-unnamed - 2025-12-09T194847.866.jpg>)

-   A window will display available funnels and blogs with checkboxes.

![](<https://assets2.modalsupport.net/1765280960522-unnamed - 2025-12-09T194908.403.png>)

-   Choose to add the entire blog or specific categories.

**Please Note:** To accomplish this correctly it requires you to make use of the “Blog Post List” Element in the site builder.

## Part 2: Generating the Sitemap

### Once you've selected the blogs and categories, proceed with these steps:

-   Click on the "Proceed" button.
-   The subsequent menu allows you to add, edit, and delete custom paths for the ending URL directory (e.g., /custom).

![](<https://assets2.modalsupport.net/1765280976219-unnamed - 2025-12-09T194929.784.png>)

-   Click on the "Generate & save" button.

## Part 3: Confirmation and Automatic Mapping

### After generating the sitemap, follow these steps:

-   A confirmation window will appear, stating that you have created an XML Sitemap for search engine consumption.

![](<https://assets2.modalsupport.net/1765281004594-unnamed - 2025-12-09T194951.916.png>)

-   Click on the "Okay" button to confirm.

-   Going forward, any additional blog categories and blogs you add will be automatically mapped to the sitemap as long as the Blog Post List Element is present on the site and the categories have been added

![](https://assets2.modalsupport.net/1773243117572-image2.png)

## Part 4: Indexing Blog URLs

### To index blog URLs, follow these steps:

-   Navigate to "Domain Settings" and select "XML Site map."
-   Choose the funnel/website page(s) with the blogs/categories you want to index.

-   Additional content will be indexed automatically for the site with the Blog Post List element present upon saving it.

## Custom Paths (Optional)

### If needed, provide a custom path and then follow these steps:

Step 1: Once you select the funnel/website page in the XML menu, click on "Continue."

Step 2: Click "Generate and save" to obtain the sitemap URL.

## Automated Crawling

The feature supports the automated discovery of blogs and categories while saving the funnel page in the funnel builder. This eliminates the need to go to the domains page to modify the funnel page sitemap. 

**Please note** that automatic crawling works only if the funnel/website page is selected for crawling. You can view the sitemap on the confirmation step to confirm this setting on your site.

By following these steps, you can harness the power of XML Sitemap Generation to enhance your blog's SEO performance and search engine presence.

## Use Case:

Suppose you regularly update your blog with new posts and want to ensure that each new post is promptly indexed by search engines for maximum visibility. By enabling XML Sitemap Generation, you automate the process of updating your sitemap whenever you add new content. This saves time and ensures that your latest posts are always discoverable by search engines, helping to drive more organic traffic to your blog.

## FAQ’s




**Question:** What is XML Sitemap Generation, and why is it important for my blog's SEO?
**Answer:** XML Sitemap Generation allows you to index URLs for your blogs on funnel or website pages, making it easier for search engines to discover and crawl your content. This enhances your blog's visibility on search engines, improving your overall SEO performance.




**Question:** How do I enable XML Sitemap for my blogs?
**Answer:** To enable XML Sitemap for your blogs, go to "Settings" and select "Domains." Click on the "XML Site map" link next to your domain, and add the domain if necessary. In the displayed window, select the blogs or specific categories you want to include in the sitemap. Ensure that you use the “Blog Post List” Element in the site builder for proper indexing.




**Question:** Can I customize the paths in my XML Sitemap?
**Answer:** Yes, you can customize the paths in your XML Sitemap. After selecting the blogs and categories, you can add, edit, or delete custom paths for the ending URL directory in the XML menu. This allows you to refine how your content is indexed by search engines.




**Question:** Will my new blog posts and categories be automatically indexed?
**Answer:** Once you generate the sitemap, any new blogs or categories added will be automatically mapped to the sitemap, provided that the Blog Post List Element is present on the site and the categories are added. This feature ensures that your content is continuously updated for search engine indexing.




**Question:** How does the automated crawling feature work?
**Answer:** The automated crawling feature supports the discovery of blogs and categories when saving the funnel page in the funnel builder. It removes the need to manually modify the funnel page sitemap on the domains page. Automated crawling only works if the funnel or website page is selected for crawling, and you can verify this setting during the confirmation step.




**Question:** What is a practical use case for this feature?

**Answer:** If you frequently publish blog posts, enabling XML Sitemap Generation ensures each new post is promptly indexed by search engines, saving time and improving visibility and organic traffic.  




**Question:** Is there a specific element I need to use in the builder for the sitemap to work correctly?

**Answer:** Yes. To ensure your blogs and categories are indexed and mapped correctly, you must use the **“Blog Post List” Element** within the site builder on your funnel or website pages.
