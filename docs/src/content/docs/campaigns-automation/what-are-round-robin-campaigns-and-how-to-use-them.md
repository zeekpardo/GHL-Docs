**Important Note**: “**Campaigns**” in **Automation** is a deprecated feature and has been replaced by **Workflows**. If you want to learn more about how to assign multiple users in a round-robin manner in Workflows, please check out our article “**Workflow Action: Assign To User**”  

Round Robin Campaigns are useful for businesses with multiple users on their staff. These campaigns assign leads to users on a 1:1 basis, allowing all users (often salespeople) to be assigned leads in a fair and unbiased manner. 

### **Step 1: Create multiple users.** 

 You need to set up multiple users within for them to do a round-robin assignment.

Follow the steps below to create multiple users.

-   Click on **Settings** at the bottom left and navigate to the “**My Staff**” section.
-   Click +Add Employee.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcOQyeAAHQ6OUGtQy1U2AENZmYDRaMIQiUkDdDHh4tVcZau-5O4kkEf-YTUOPP1gRrPV99yEsxqG0ZE-qj3hb2fxy8ris0516zds6c3ikzSMkGMWAp3y7l6jIrHyJv5T8WxIQ9BYFpug6mjlE7V5AzkJqHI?key=tSXBe8WccTIexulemL82ZQ)

-   Fill in the user information.
-   Save.
-   Repeat the steps for all users.
    -   Note: It is recommended to use the user's unique email and phone number as these are going to be the methods of contact for the leads they are assigned to.

### **Step 2:  Assign multiple users to a campaign.**

  Users must be assigned to a campaign in order to turn it into a Round Robin campaign.

-   Navigate to Campaigns.
-   Create a new campaign or open an existing campaign.
-   Click the Campaign Configuration dropdown.
-   Click the Users dropdown and select the appropriate users to assign to the campaign.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfEpqLCXYMkqCj0AD52D05Q5KT8NVk8MurTdnvMWXlpaLGI-hD8elKxSXYjBBDwatTxIjEF9hGg-sGmQqpW5APy53o3SC73x0rF894XDZYrz561Bar8BL8pIf0YGUT1g7NPMWePvmE0J8tFYzijsMREyd-W?key=tSXBe8WccTIexulemL82ZQ)

-   Save.
    -   **_NOTE_:** we recommend leaving the 'from email' and the 'from name' (subfields of From address) within campaign configuration blank as will default back to the name and emails you set up for the user (see step 1.). This ensures all emails within the campaign come from the assigned user.
-   You can use merge fields/custom values to customize emails and SMS, further making them usable for multiple users.

### **FAQ**

**1\. How do I ensure leads are distributed evenly among users?**By setting up a Round Robin Campaign and assigning multiple users to it, leads are distributed one by one to each user, ensuring fairness.

**2\. Can I assign users to multiple campaigns?**Yes, users can be assigned to multiple campaigns, and their leads will be distributed according to the configurations of each campaign.

**3\. What happens if a user is removed from the campaign?**If a user is removed from the campaign, they will no longer receive new leads from that campaign. The system will automatically reassign leads to other users in the campaign.

**4\. Can I use different email addresses for each user?**Yes, each user can have a unique email address. This ensures that leads receive communications from the correct user.
