The Manual SMS Event within a campaign creates a manual task for the user to send a custom SMS to the associated lead. This allows the user to respond to the lead at the time that best works for them. 

### **Part 1: Navigate to the Existing Campaign or Create a new one**

**Step 1:** Click "create an event" and choose "Manual SMS"

**Step 2:** Save and Publish the event.

### **FAQ**

**1\. What is the purpose of the Manual SMS Event in a campaign?**

-   The Manual SMS Event creates a task for the user to send a custom SMS to a lead, allowing for timely and personalized communication based on the user's schedule.

**2\. How do I assign a specific user to receive the Manual SMS task?**

-   You can assign a user either through a trigger or directly at the campaign level by navigating to campaign configuration, selecting "Assign User," and saving the changes.

**3\. Can I test the Manual SMS Event before using it in a live campaign?**

-   Yes, you can test the event by adding a sample/test contact to the campaign and verifying the Manual SMS task under Manual Actions.

**4\. What should I do if "Manual SMS" tasks are not appearing in Manual Actions?**

-   Ensure that the campaign is correctly configured and that the test contact has been added to the campaign. If issues persist, review the setup and check for any missed steps.
