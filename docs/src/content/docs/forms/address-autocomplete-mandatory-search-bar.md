**Address Autocomplete & Mandatory Search Bar**

Enhancing user experience and ensuring accurate data entry, the Address Autocomplete and Mandatory Search Bar features streamline address input in forms and surveys. These features help prevent errors and improve efficiency by auto-filling address fields and enforcing valid selections.

## **Key Features**

### **1\. Auto-Complete Address Behavior**

-   When enabled, the selected address auto-populates relevant fields such as **Street, City, Zip Code, State, and Country**.
-   Users can complete forms faster with fewer manual inputs.

### **2\. Mandatory Search Bar**

-   **Enabled**: Users must select an address from autocomplete suggestions before proceeding.
-   **Disabled**: Users can manually enter an address without selecting from autocomplete.
-   Ensures consistency and accuracy in address input.

### **3\. Improvements**

-   Faster and more accurate address entry.
-   Minimizes errors by enforcing valid address selection.
-   Provides customization options to enable or disable these features as needed.

## FAQs




**Question:** Can I enable Address Autocomplete without making the search bar mandatory?  
**Answer:** Yes, you can enable address autocomplete while allowing users to enter their address manually.




**Question:** What happens if a user types an address without selecting a suggestion?  
**Answer:** If the Mandatory Search Bar option is enabled, they won’t be able to proceed without selecting a suggested address.




**Question:** Will this feature work for international addresses?  
**Answer:** Yes, the autocomplete function supports global addresses based on the address database.




**Question:** Can I hide specific address subfields?  
**Answer:** Yes, you can customize the display settings to show or hide specific address fields.




**Question:** Do these changes apply automatically to all forms and surveys?  
**Answer:** Once enabled, these settings apply to all forms and surveys using the address field.  
By leveraging these enhancements, businesses can ensure a seamless, efficient, and error-free data collection process in their forms and surveys.




**Question:** What are the benefits of using Address Autocomplete and Mandatory Search Bar?
**Answer:** These features improve efficiency by reducing manual input, enhance accuracy by ensuring valid addresses, and offer customization options for better form management.  


**Question:Can I show all the individual address subfields while still using the autocomplete feature?
**Answer:** Yes, you can choose to display all subfields (such as Street, City, and Zip Code) while keeping the autocomplete functionality active, or you can choose to hide them and rely solely on the search bar.




**Question:** What should a user do if their specific address doesn't appear in the search bar?

**Answer:** If the **Mandatory Search Bar** is enabled and an address is brand new (like a new construction home), the user might get stuck. In these cases, it is best to leave the Mandatory toggle **OFF** so they can manually type their info if the database hasn't updated yet.




**Question:** Can I use this feature to make my forms look shorter and cleaner?

**Answer:** Yes. You have the option to **hide subfields** (like City and State) and rely solely on the search bar. This reduces the visual clutter of the form while still capturing the full address data in the background.
