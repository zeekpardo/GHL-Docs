**Understanding Automatic Country Detection in Forms**

Automatic country code detection is a great feature, it works effortlessly when you use a Phone field, allowing forms to recognize the location of the person accessing the page. Here's how you can make the most of this feature:

### Step 1: Using the Phone Field

-   Navigate to **Sites** on the left > **Forms** at the top > From the dropdown select **Builder**.

![](https://assets2.modalsupport.net/1765219196032-image4.png)

-   Create a new form or edit an existing form.

![](https://assets2.modalsupport.net/1765219242131-image3.png)

-   Drag and drop the Phone field into the form using the Standard Fields on the left-hand side.

![](https://assets2.modalsupport.net/1765219309690-image5.png)

-   Click on the phone element to make the settings appear on the right-hand side. From there check the box for “**Enable Country Picker**”

![](https://assets2.modalsupport.net/1765219405009-image7.jpg)

-   Save the form.

![](<https://assets2.modalsupport.net/1762531011614-unnamed \(1\).jpg>)

-   Now, you can click on “**Preview”** next to “Integrate” to view the form yourself or choose to view it within your own website or funnel. The flag will show. This flag represents the country you're physically located in, as well as the person accessing the form.

![](https://assets2.modalsupport.net/1762531041571-unnamed.png)

### FAQs




**Question:** How does the automatic country code detection feature work in forms?
**Answer:** The automatic country code detection feature uses the location of the person accessing the form to display the relevant country code in the Phone field automatically. This is done by detecting the user's IP address and matching it with the appropriate country.




**Question:** Can I manually select a different country code if the detected one is incorrect?
**Answer:** Yes, the Phone field with the “Enable Country Picker” option allows users to manually select a different country code from the dropdown if the automatically detected one is incorrect.




**Question:** Does the automatic country code detection work on all devices?
**Answer:** Yes, the feature works across all devices, including desktops, tablets, and smartphones, ensuring a consistent user experience regardless of the device used to access the form.




**Question:** Will the automatic country code detection still work if the user is using a VPN or proxy?
**Answer:** If a user is using a VPN or proxy, the detected country code may reflect the location of the VPN or proxy server rather than the user’s actual location, which could result in an incorrect country code being displayed. Users can manually adjust the country code if needed.

With our Form Builder's automatic country code detection, your forms become smarter, offering a personalized experience to users based on their location. No manual input or complex configurations are needed; the system handles everything seamlessly.




**Question:** How do I enable the flag icon to appear in the phone field?

**Answer:** To display the flag, you must select the phone element in the Form Builder and check the box labeled “Enable Country Picker.” Once this is done and the form is saved, the flag representing the user's location will be visible.




**Question:** Can I set a "Default Country" if I only serve customers in one specific region?
**Answer:** While the system defaults to the user's IP location, you can often set a "Preferred Country" in the field settings. This ensures that even if a user's IP is masked, your primary market is shown first.  




**Question:** Where do I go to add the Phone field to my form so I can use the country detection feature?
**Answer:** You can find the Phone field by navigating to the "Standard Fields" section on the left-hand side of the Form Builder. From there, you simply drag and drop it into your form.
