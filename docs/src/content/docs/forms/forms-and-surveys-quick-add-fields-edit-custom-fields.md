**Forms and Surveys: Quick Add Fields & Edit Custom Fields**

The Quick Add Fields & Edit Custom Fields feature is designed to make your forms/surveys more efficient and user-friendly. This feature eliminates the need for multiple steps and simplifies customization, providing a seamless experience for users.

## **How to Use**

### **Quick Add Custom Fields**

**1\. Access Form/Survey Builder**:

-   Navigate to sites on the left, the form or survey builder section on the top right in your platform. Click on the builder.

![](https://assets2.modalsupport.net/1765571027817-image4.png)

**2\. Drag and Drop Custom Fields**:

-   Select the desired custom field type (e.g., Radio Select, Single Line, Multi Line, Checkbox, etc.).
-   Drag and drop the field into the form or survey.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdb-LEWKTB7VgC-_G9DxciCklZJyTvkgGZNukxbnQP_8sJUo79Hc27zBWYyiS6ezNVkGu2bz_k1RIQ2qY5OHNw7-wXA4d_LbDWmHLAGkaRU7XI-jtKyfGD6asUVAr3msj8tmqhr3tbssSOyieAxg9d4EyMw?key=lvPFFQjde0QhCxmfrab2Zg)

**3\. Customize the Field**:

-   Customize the options for dropdowns, radio buttons, checkboxes, and file input options for file upload fields.
-   Set the custom field name and unique key.

**4\. Save the Form/Survey**:

-   Once you’ve added and customized your fields, save the form or survey to make the custom fields available across all areas.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfXdM7CP2zTmdfnAS9fnkDwsKKenUptVoarHZ2059XKMiMUun5I4KO2XTuRh6j8WwdY8DaoBQW15OBa4kTQVjuzS9G99DcvGxiXvDvwByVwlCk__YnTgTjCPWx2zfVEXOGa12jtJTxzGawG6nJuPrWd-0v7?key=lvPFFQjde0QhCxmfrab2Zg)

**5\. Conditional Logic with Address Fields**

-   You can use address fields (e.g., street address, city, state, country, postal code) to configure conditions within forms. This allows for tailored actions based on user inputs, such as redirecting to a URL, showing or hiding fields, or displaying custom messages.
-   Set up address-based conditions in the form editor by selecting address fields from the dropdown menu, defining criteria (e.g., city or postal code), assigning actions like redirects or field visibility, and saving your changes.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdp6u168n8cXwriclZBMoUGszkN-BY5Y2SDBY2nvh1D8AF_wWS4lmddFUf6pWJhE0srpplnq_Cpjy80avJr0l7GwK_jmJSFQSkmkXRHg1UWtzybrdIsGhfigcl9dh2H-ys8I5A_Vg?key=lvPFFQjde0QhCxmfrab2Zg)

**6\. Edit Custom Fields**:

-   You can edit properties of newly created or existing custom fields from the custom fields section.
-   Adjust options, reorder, or delete as needed.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXcUWbu1_TNJmUHrtBr91aUYpdBukMok3UgCeQ7Z8DhzQ4v5lj2tX9PvcW2rIIpOPXxDP2xJ6yjlQhwy7qxGHNBqtb12jLjPRfv43f2DqhAD1ydLWcZbTLiDMGYxx6viXhh2Yb9YGzhF4oig0Y4T3xSywIA?key=lvPFFQjde0QhCxmfrab2Zg)

**7\. Math Calculation in Forms/Surveys**:

The Math Calculations feature in forms and surveys allows users to add custom fields and set up a calculator based on those fields. This feature expands the flexibility of scoring, making it ideal for more complex calculations such as quizzes, assessments, or budgeting tools. Users can define calculation logic for these fields, with real-time score updates available during the preview.

**Note:** Ensure that calculations are configured correctly, as these updates are applied in real-time.

**Supported Field Types:**

-   Numeric Fields
-   Monetary Fields
-   Radio Select
-   Checkbox
-   Single Dropdown
-   Multi Dropdown

**How To Use:** To use this feature, simply navigate to the form or survey builder, add a "Score" field, define your calculation logic, and preview the real-time results.

## **Notes**

-   **Field Locking**: After saving the form or survey, the custom field name and unique key are locked and can only be edited from the custom fields section.
-   **Simplified Workflow**: This feature significantly reduces the number of steps, making adding and managing custom fields quicker and easier.

## **Pro Tips**

-   **Organize Your Fields**: Use descriptive names for custom fields to make them easily identifiable later.
-   **Preview Changes**: Always preview your form or survey to ensure all fields are correctly configured before finalizing.
-   **Leverage Field Types**: Use various field types, such as Radio Select, Multi-Line, and Date Picker, to gather more specific information from respondents.

**FAQ**




**Question:** Can I edit a custom field after saving the form or survey?
**Answer:** Yes, you can edit custom fields, but once the form or survey is saved, the custom field name and unique key are locked. You can only edit them from the custom fields section.




**Question:** What types of custom fields can I add to my form or survey?
**Answer:** You can add various custom fields such as Radio Select, Single Line, Multi-Line, Checkbox, Dropdowns, File Upload, and more.




**Question:** How does the Math Calculation feature work in forms/surveys?
**Answer:** The Math Calculation feature allows you to set up a calculator based on field types like Radio Select, Checkbox, Single Dropdown, and Multi Dropdown options in the form or survey. Simply add a "Score" field, define the logic, and preview real-time results.




**Question:** Can I preview the form or survey before finalizing it?  
**Answer:** Yes, it is recommended to preview your form or survey to ensure all fields are correctly configured before finalizing.




**Question:** Can I create a field in one form and use it in another?  
**Answer:** Absolutely. Once you "Quick Add" a custom field in the builder, it is saved to your account's global custom field library. You can then drag that same field into any other form or survey without having to recreate it.




**Question:** Can I see the results of a math calculation before I publish the form?

**Answer:** Yes. The Math Calculation feature provides **real-time score updates** during the preview mode, allowing you to verify that your logic is working correctly before going live.




**Question:** Can I use address-related fields to trigger specific actions, like showing certain questions or redirecting users to a different URL?
**Answer:** Yes. You can use **Conditional Logic with address fields** (such as city, state, or postal code). This allows you to set up tailored actions based on user input, including redirecting to a URL, showing or hiding specific fields, or displaying custom messages.
