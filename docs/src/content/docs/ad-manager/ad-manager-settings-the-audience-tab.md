Ad Manager, focusing on ease of use, expanded functionality, and enhanced user control over audiences, templates, and integrations. From creating and managing audiences to using templates and connecting multiple Meta Ad accounts, this guide covers everything you need to maximize the potential of your Ad Manager tool.

## **Audiences tab for Audience Management**

The **Audiences** tab under Ad Manager settings now allows you to create and manage audiences independently from campaign creation. Previously, custom and lookalike audiences could only be set up while building a campaign, but you now have the flexibility to set up your target groups in advance, ready for use anytime.

### **Important**

-   **Audiences Tab**: A dedicated section under Ad Manager settings to create and manage audiences.
-   **Create Audiences in Advance**: You can create custom or lookalike audiences before starting a campaign, streamlining your campaign-building process.
-   **Audience Summary:** Provides a detailed overview of all audiences, helping you manage and optimize ad targeting with greater ease. 
-   **Edit and Delete Options**: You can rename or edit descriptions for audiences, and delete them if they’re not tied to any active campaigns.

**Note: Customer List Custom Audience with Dynamic Smartlist Option**

-   You can also create custom audiences from smart lists, with the option to choose between static or dynamic audience updates.
-   **Static** audiences include only contacts currently in the smart list at the time of creation.
-   **Dynamic** audiences automatically update to include contacts added or removed from the smart list after creation, ensuring your campaigns always target the most up-to-date contacts.

### **FAQs**




**Question:** Can I create audiences before starting a campaign?
**Answer:** Yes, you can now create custom and lookalike audiences in advance under the **Audiences** tab, allowing for a more streamlined campaign-building process.




**Question:** Can I edit or rename an audience after creating it?
**Answer:** Yes, you can rename or update the description of an audience by clicking the ellipsis (three dots) next to it and selecting the **Edit** option.




**Question:** What happens if I try to delete an audience linked to an active campaign?
**Answer:** If an audience is tied to an active campaign, a notification will appear stating that it cannot be deleted until it is no longer in use.




**Question:** Where can I find a list of all audiences connected to my Meta Ad account?
**Answer:** You can view and manage all audiences under the **Audiences** tab within Ad Manager settings.




**Question:** Can I view the audience summary for all audiences?
**Answer:** Yes, the audience summary is available for all lookalike and custom audiences, providing key insights for better decision-making.




**Question:** Can I create custom audiences from smart lists?
**Answer:** Yes, you can create custom audiences from smart lists with the option to choose between static or dynamic updates, ensuring your campaigns always target the most up-to-date contacts.  




**Question:** What is the difference between static and dynamic custom audiences created from smart lists?
**Answer:** Static audiences include only contacts that are present in the smart list at the time of creation, while dynamic audiences automatically update to include contacts that are added or removed from the smart list after creation. This ensures that your campaigns always target the most up-to-date contacts.




**Question:** Can I use these audiences for both Facebook and Instagram?

**Answer:** Yes. Since the Ad Manager connects to your Meta account, any audience you create can be utilized for placements across both Facebook and Instagram, depending on your campaign settings.




**Question:** Can I delete an audience that was used in an old campaign?

**Answer:** Yes, as long as the campaign is **not active**. If you try to delete an audience currently linked to an active campaign, the system will trigger a notification explaining that it cannot be deleted until it is no longer in use.
