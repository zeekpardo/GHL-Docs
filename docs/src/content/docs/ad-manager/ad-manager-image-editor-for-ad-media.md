**Ad Manager: Image Editor for Ad Media**

The Image Editor functionality within Ad Manager allows you to edit your ad media directly while setting up or modifying your campaigns. This tool enables you to customize images seamlessly, offering greater flexibility and efficiency without the need for external design software.

### **How to Use the Image Editor**

**1\. Access the Image Editor**:  
From the Ad Manager homepage, select the campaign you want to edit or create a new campaign. After uploading the image for your ad, click the pencil icon located on the image thumbnail to edit.

### **Note**

The Image Editor supports both PNG and JPEG formats, and real-time previews allow users to see the final result before saving. The tool is fully integrated with Ad Manager, so all adjustments are immediately reflected in your ad previews, saving you time and reducing the need for additional tools.

### **Pro Tips:**

-   **Test Different Filters**: Experiment with different filters to see which one resonates best with your target audience. The right filter can change the entire tone of your ad.
-   **Utilize Stickers and Emojis**: Don't be afraid to use stickers and emojis to add personality to your ads. They can make your visuals more engaging and fun.
-   **Always Preview**: Before finalizing, always use the real-time preview to check how your image looks across different devices to ensure consistency.

### FAQ




**Question:** Can I adjust the crop size for different platforms?
**Answer:** Yes, you can crop the image to fit various aspect ratios, including those for Facebook, Instagram, and custom sizes.




**Question:** Does the image editor support all file formats?
**Answer:** The editor supports PNG and JPEG formats. Ensure your files are in one of these formats for optimal results.




**Question:** Can I add multiple text layers to an image?
**Answer:** Yes, you can add multiple text elements with customizable font options, sizes, and colors to create a layered effect in your ad visuals.




**Question:** How can I see how my image will look on different devices?
**Answer:** Use the real-time preview feature to check how your image will appear across various devices before saving. This ensures your ad looks great everywhere.




**Question:** Will the Image Editor be available for video files?
**Answer:** Currently, the Image Editor supports only still images (PNG/JPEG). Video editing features are planned for future updates.




**Question:** Is there a limit to the number of edits I can make on an image?
**Answer:** No, you can make unlimited edits on an image before saving, including text, filters, stickers, and brush adjustments.




**Question:** Can I save my edited image to use on my website later?

**Answer:** Once you have saved your edited ad image, it will be stored in your account assets. You can typically find it in your Media Library or download the final version if you want to use the same design for your website or email newsletters.




**Question:** Will editing my image inside Ad Manager reduce the image quality?

**Answer:** No. The editor is designed to maintain the integrity of your original file. However, we recommend starting with a high-resolution image (at least **1080x1080px**) to ensure that when you crop or add text, the final ad remains crisp and clear across all devices.




**Question:** Does the editor allow for free-hand drawing or manual marking on the ad?

**Answer:** Yes, there is a **Brush Tool** available. You can customize the brush size and color to paint or draw directly onto the image for precise, custom edits.
