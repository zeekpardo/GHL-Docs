**Workflow Action: Update Affiliate**

In this article, we will walk through how to use the Update Affiliate workflow action. This is useful to automate adding Affiliates to the Affiliate Manager.

### Key Features and Benefits

-   **Automated Status Management:** Effortlessly switch affiliate profiles between "Active" and "Inactive" based on real-time behavior, such as low performance.
-   **Operational Efficiency:** Eliminate the need for manual record-keeping by allowing the system to handle administrative updates to your affiliate database.
-   **Enhanced Database Accuracy:** Keep your affiliate list clean and relevant, ensuring that only qualified partners have access to your marketing campaigns and portals.
-   **Real-Time Syncing:** Immediately reflect changes in an affiliate's status across the entire platform, which can trigger or halt subsequent automated communications.
-   **Segmented Partner Control:** Use this action in conjunction with conditional logic to manage different tiers of affiliates, ensuring each partner is categorized correctly as they grow.

### Step 1: Start a new workflow or edit an existing one

-   Navigate to Automation on the left > Workflows on the top and create a new workflow or edit an existing one

![](https://assets2.modalsupport.net/1766072109590-image4.png)

### Step 2: Set up applicable Workflow Trigger(s). 

We have multiple articles available explaining the setup of each of the Workflow Triggers. Feel free to review our articles for more information.

**_Pro Tip:_** You can use any Workflow Triggers to trigger the Update Affiliate action or you could leave this workflow without any triggers.

### Step 3: Setting up the Update Affiliate Action

-   Click on the “+” symbol below the trigger to add an action.

![](https://assets2.modalsupport.net/1763567847224-2.png)

-   Under Actions, select Update Affiliate Manager in the Affiliate  section of the Actions Menu or use the search feature

![](https://assets2.modalsupport.net/1763567883226-3.png)

-   Select whether or not to mark as **active** or **inactive**
-   Press **Save** in the bottom right corner of the Action setup menu

![](https://assets2.modalsupport.net/1763567950311-4.png)

### Step 4: Press Save in the top right corner of your browser to save the workflow

-   If your Workflow is ready to be enabled, make sure the toggle switch under Save is set to Publish
-   Be sure to test your Workflow to ensure it is set up correctly using the Test Workflow button next to the Publish toggle switch.
    -   For an in-depth overview of Workflow Testing, see our article, “Using the Testing Features in Workflows.”

![](https://assets2.modalsupport.net/1763568153477-5.png)

Now your workflow is set up, and you are ready to go with the Action Add To Affiliate Campaign!

### Pro Tips

-   **Triggers Flexibility**: While you can use any Workflow Trigger to activate the Update Affiliate action, consider using triggers that align closely with your affiliate management goals to optimize efficiency.
-   **Testing Is Key**: Always test your workflow after setup to ensure that the Update Affiliate action functions as expected. Utilize the Test Workflow button to simulate the process and verify that affiliates are correctly updated.
-   **Action Status**: Regularly review and update the active/inactive status of affiliates to ensure your affiliate data remains current and relevant.

### FAQs




**Question:** Can I use multiple triggers for the Update Affiliate action?
**Answer:** Yes, you can configure multiple triggers for the same workflow. This allows you to automate updates based on various conditions or events.




**Question:** What should I do if the Update Affiliate action is not working as expected?
**Answer:** Check your workflow triggers and settings to ensure they are correctly configured. Also, verify that the affiliate information is accurate and up-to-date.




**Question:** How do I ensure my workflow is correctly published?
**Answer:** Make sure to toggle the switch to "Publish" after saving your workflow. This activates the workflow and allows it to run according to your setup.




**Question:** Can I set up the Update Affiliate action without a trigger?
**Answer:** Yes, you can set up the Update Affiliate action without any triggers. However, without a trigger, the action will not automatically execute, and you may need to run it manually.




**Question:** How often should I test my workflows?
**Answer:** It’s a good practice to test your workflows regularly, especially after making changes or updates. This helps ensure that all actions, including Update Affiliate, perform as intended.




**Question:** What specific status changes can I apply to an affiliate using this action?

**Answer:** When setting up the Update Affiliate action, you can specifically choose whether to mark an affiliate as "active" or "inactive" to ensure your affiliate data stays current.




**Question:** What happens to an affiliate's pending commissions if I use a workflow to mark them as "Inactive"?

**Answer:** Marking an affiliate as "Inactive" prevents them from generating _new_ tracked leads or commissions. However, any commissions already recorded and "Pending" in the system typically remain in their ledger for your manual review and payout, unless you manually void them.
