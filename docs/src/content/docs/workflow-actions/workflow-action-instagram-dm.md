**Workflow Action: Instagram DM**

In the article, you will learn how to set up the Workflow Action: Instagram DM

**_Please Note:_** _You will need to have integrated Facebook and Instagram for this to work._

### Step 1: Start a new workflow or edit an existing one

-   Navigate to Automation > Workflows and create a new workflow or edit an existing one.

![](<https://assets2.modalsupport.net/1766057120865-unnamed - 2025-12-18T192512.496.png>)

### Step 2: Set up applicable Workflow Trigger(s). We have multiple articles available to explain the setup of each of the Workflow Triggers. Feel free to review our articles on these for more information on setting these up.

### Step 3: Setting up the Instagram DM action

-   Click on the “**+**” symbol below the trigger to add an action.

![](<https://assets2.modalsupport.net/1766057138170-unnamed - 2025-12-18T192533.085.png>)

-   Under Actions, select **Instagram DM** in the **External Communications** section of the Actions Menu or use the search feature. Upon selection, the Action setup menu will appear

![](<https://assets2.modalsupport.net/1766057159884-unnamed - 2025-12-18T192550.949.png>)

-   (_Optional_) You can change the workflow builder display name of this action using the textbox below **Action Name**. Note that this will only affect the display name within the workflow builder and has no other impact on the content or functionality of this Workflow Action.

![](<https://assets2.modalsupport.net/1766057181244-unnamed - 2025-12-18T192613.666.png>)

-   You also have the option to use a previously created template for this action

-   In the message drop you can select any of the custom values that apply to your specific needs (see below)

![](<https://assets2.modalsupport.net/1766057204445-unnamed - 2025-12-18T192636.935.png>)

-   **For Example,** I selected the custom field **Contact>First Name** & **Contact>Last Name**; this will automatically populate the contact's First and Last name in the Messenger

-   Press the **Save Action** in the bottom right corner

![](<https://assets2.modalsupport.net/1766057221623-unnamed - 2025-12-18T192657.191.png>)

### Step 4: Adding Attachments

You can attach files directly to your Instagram DM action using two convenient methods. When you click Add Attachment, a modal will appear with the following options:

-   Upload a File: Select a new file directly from your local device.

![](https://assets2.modalsupport.net/1775581190033-image7.jpg)

-   Choose from Media Storage: Select an existing asset already saved within your Media Library. This allows you to reuse approved assets (like PDF guides or promotional images) across multiple workflows without needing to re-upload them each time.  


![](https://assets2.modalsupport.net/1775581224247-image2.jpg)

### Step 5: Press Save in the top right corner of your browser to save the workflow

-   If your Workflow is ready to be enabled, make sure the toggle switch under **Save** is set to **Publish**

![](<https://assets2.modalsupport.net/1766057244643-unnamed - 2025-12-18T192717.686.png>)

-   Be sure to test your Workflow to ensure it is set up correctly using the **Test Workflow** button next to the **Publish** toggle switch. 
-   For an in-depth overview of Workflow Testing see our article, “Using the Testing Features in Workflows.”

**Please Note:**This action should only be used to send DMs to contacts who have messaged your Instagram account within the last 24 hours.

-   If the **"Instagram Comment(s) on Post"** workflow trigger is used, the DM will be sent as a **Reply to the comment** if there is a recent comment on your Instagram post.
-   If there is no recent comment, a **Direct DM** will be sent, provided the contact has messaged your business within the last 24 hours.

**Recommendation:**For sending Instagram DMs to contacts, it’s recommended to use the **"Instagram Interactive Messenger"** action.

-   When replying to a comment, set the **Reply Type** to **"Reply to Comment via DM"**.
-   When sending a direct DM, set the **Reply Type** to **"Reply to DM"**.

**Example:**

**Include Attachment:** Add a relevant attachment such as a product catalog, brochure, or promotional image that aligns with the message. This helps to provide further information to the lead, offering them a visual representation of your offerings and encouraging engagement.

**FAQs  

Question: Why am I unable to send Instagram DMs to people who are commenting on my Instagram posts?**
**Answer:** For a business to send a DM to a contact, the contact has to send at-least one DM within the last 24 hours. To reply to comments, use "Instagram Interactive Messenger" action from workflows and set the reply type as "Reply to comment via DM".




**Question:** Why do I need to integrate Facebook and Instagram for this action to work?
**Answer:** The Instagram DM action relies on the integration between Facebook and Instagram to function properly. This integration allows your workflow to send direct messages via Instagram, leveraging Facebook’s platform for managing and automating communication.




**Question:** Do I have to re-upload my files every time I want to use them in a different workflow?
**Answer:** No. You can select **Choose from Media Storage** to pick any file already uploaded to your account. This makes it much faster to reuse brand assets across different automation steps.




**Question:** How do I start a new workflow or edit an existing one?
**Answer:** To start a new workflow, navigate to your Workflow builder and select the option to create a new workflow. To edit an existing one, find the workflow you wish to modify and select the edit option. This will open the workflow for adjustments.




**Question:** Where can I find information on setting up Workflow Triggers?
**Answer:** You can find detailed information on setting up Workflow Triggers in our dedicated articles on the subject. These resources provide step-by-step instructions and explanations for each type of trigger available.




**Question:** How do I add the Instagram DM action to my workflow?
**Answer:** Click the “+” symbol below the trigger in your Workflow builder to add a new action. From the Actions Menu, either select Instagram DM from the External Communications section or use the search feature to find it.




**Question:** Can I rename the Instagram DM action in the workflow builder?
**Answer:** Yes, you can rename the Instagram DM action by using the textbox below the Action Name in the Action setup menu. This name change is only for display purposes within the Workflow builder and does not affect the action’s functionality or content.




**Question:** How do I use a previously created template for the Instagram DM action?
**Answer:** When setting up the Instagram DM action, you can choose to use a previously created template. This option is available in the Action setup menu and allows you to apply a pre-designed message format for your direct messages.




**Question:** What custom values can I use in the Instagram DM message?
**Answer:** In the message dropdown, you can select custom values relevant to your contact, such as Contact>First Name, Contact>Last Name, etc. These values will be automatically populated in the message based on the contact’s information.




**Question:** Can I send images or PDFs through the Instagram DM workflow action?

**Answer:** Yes. You can use the **Include Attachment** feature to add product catalogs, brochures, or promotional images. Adding a visual element is highly recommended as it increases lead engagement and provides a professional representation of your brand.




**Question:** How can I verify that my Instagram DM workflow is actually working before going live?

**Answer:** Once you have saved your workflow and set the toggle to **Publish**, use the **Test Workflow** button located next to the toggle. This allows you to run the logic through a test contact to ensure the integration is functioning correctly.
