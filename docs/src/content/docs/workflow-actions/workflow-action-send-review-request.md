**Workflow Action: Send Review Request**

In this article, you will learn how to properly set up the Workflow Action: Send Review Request and where to navigate if you want to customize the layout of the SMS/Email.

### **Step 1:** Start a new workflow or edit an existing one

![](https://assets2.modalsupport.net/1768443070921-image1.png)

### **Step 2:** Creating and Editing the Action

-   Click the “**+**” button below to Add New Workflow Trigger

Note: You can add any trigger based on which you want to send the review to the leads.

![](<https://assets2.modalsupport.net/1763070081269-Captura de pantalla 2025-11-13 164111.png>)

-   Select **Send Review Request** in the Action List by clicking on the + icon. You can either search ‘Send Review Request’ or look for it in the list.

![](<https://assets2.modalsupport.net/1763067651817-Captura de pantalla 2025-11-13 160043.png>)

-   (_Optional_) You can change the workflow builder display name of this action using the textbox below **Action Name**. Note that this will only affect the display name within the workflow builder and has no other impact on the content or functionality of this Workflow Action.

-   Select the type of review you want to be sent out (**E-mail** or **SMS**)

![](https://assets2.modalsupport.net/1765985490491-image.png)

-   Pro tip: if you want to configure the review message just click on the “**Reputation Settings**” shortcut

![](https://assets2.modalsupport.net/1765985518289-image.png)

-   This will automatically take you to the **Reputation Management** tab and allow you to insert/generate a review link, customize the SMS and Email layout as well as preview it via mobile, tablet, or laptop.

-   Press **Save Action** in the bottom right corner of the Trigger setup menu

![](https://assets2.modalsupport.net/1765985534973-image.png)

### **Step 3:** Press **Save** in the top right corner of your browser to save the workflow

-   If your Workflow is ready to be enabled, make sure the toggle switch under **Save** is set to **Publish**
-   Be sure to test your Workflow to ensure it is set up correctly using the **Test Workflow** button next to the **Publish** toggle switch. 

-   For an in-depth overview of Workflow Testing see our article, “Using the Testing Features in Workflows”

![](https://assets2.modalsupport.net/1765985555631-image.png)

Your workflow is set up and ready to go with the Send Review Request Action.

### FAQs




**Question:** What does the Send Review Request action do?
**Answer:** This action sends a review request via email or SMS to your contacts, encouraging them to leave feedback or a review.  




**Question:** Can I customize the content of the review request?
**Answer:** Yes, you can customize the email and SMS content by using the Reputation Settings shortcut. This allows you to insert review links and adjust the layout of your messages.  




**Question:** How do I access the Reputation Settings to customize my review request?
**Answer:** Click on the “Reputation Settings” shortcut within the action setup menu. This will take you to the Reputation Management tab, where you can make your customizations.  




**Question:** What should I do if the review request does not appear as expected in the preview?
**Answer:** Double-check your customizations in the Reputation Management tab. Ensure that all layout and content adjustments are saved properly. You can also preview the request on different devices to confirm its appearance.  




**Question:** Can I use this action without customizing the message?
**Answer:** Yes, you can use the Send Review Request action with the default message settings if customization is not required.  




**Question:** How can I test if the Send Review Request action is working properly?
**Answer:** Use the Test Workflow button to simulate the workflow and ensure that the review request is sent correctly. Verify that the email or SMS is received as expected.  




**Question:** Are there any limitations to the types of reviews I can request?
**Answer:** The action supports requesting general reviews. For specific types of reviews or integrations, ensure that your review platform or settings support these features.




**Question:** Is there a limit to how many review requests I can send?
**Answer:** While there is no system limit, you should be mindful of carrier and email provider "spam" triggers. Sending too many requests at once can hurt your deliverability. Using a "Drip Mode" or spacing out requests is best for large batches.  




**Question:** Does the SMS Review Request use my standard SMS credits?

**Answer:** Yes. The review request is sent through your integrated telephony provider (like Twilio) and will consume credits at the standard rate for your account.
