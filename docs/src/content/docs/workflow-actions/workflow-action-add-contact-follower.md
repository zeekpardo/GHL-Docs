**Workflow action: Add Contact Follower**

The "**Add Contact Follower**" action integrates seamlessly into workflows, enabling users to automate adding followers to contact profiles. Whether adding or removing followers, this feature eliminates the manual effort previously required, streamlining the follower management process.

## Why it Matters

Automating follower management holds significant importance as it enhances efficiency and productivity. Users can focus their efforts on more strategic tasks by automating the addition or removal of followers, leading to improved workflow performance and streamlined processes.

## How to Use

**1\. Access Workflow Editor**: Navigate to Automation on the left > Workflows on the top and create a new workflow or edit an existing one

![](<https://assets2.modalsupport.net/1766060391777-unnamed - 2025-12-18T201942.359.png>)

**2\. Add Action**: Select the option to add a new action within the workflow editor.  

![](<https://assets2.modalsupport.net/1766060410723-unnamed - 2025-12-18T202004.972.png>)

3\. Select "Add Contact Follower": Search for the "Add Contact Follower" action among the available options and add it to your workflow.  

![](<https://assets2.modalsupport.net/1766060487413-unnamed - 2025-12-18T202120.686.png>)

**4\. Configure Action Parameters**: Configure the action parameters to specify the contact and user(s) you want to add as followers. This may involve selecting the contact profile and choosing the users to be added as followers.

**5\. Execute Workflow**: Once the action is configured, execute the workflow to initiate the process of adding followers to the specified contact profile.  

**6\. Monitor and Review:** Monitor the execution of the workflow and review the results to ensure that followers have been successfully added to the contact profile.

**7\. Save:** Click on save to save the action

![](<https://assets2.modalsupport.net/1766060510876-unnamed - 2025-12-18T202145.404.png>)

**Example**

**Scenario:** Sales Team Collaboration

**Objective:** The user wants to ensure that whenever a contact submits a form, a follower is assigned to the contact.

**Workflow Steps:**

**Trigger:**

-   **Form Submitted** – The workflow begins when a high-value lead is created in the system.

**Action:**

-   **Add Contact Follower** – The action "Add Contact Follower" is included in the workflow, and the follower is chosen from the dropdown menu.

**Outcome:**

-   When a high-value lead is created, both the sales representative and the account manager are automatically assigned as followers.

**FAQs**




**Question:** Can I remove followers using this action?


**Answer:** No, the "Add Contact Follower" action is specifically for adding followers. Removing followers requires a different action.




**Question:** How do I know if the followers were added successfully?


**Answer:** Monitor the workflow execution and review the results to verify the successful addition of followers.




**Question:** Is there a limit to how many followers I can add at once?


**Answer:** Refer to your platform’s documentation for specific limits on follower additions.




**Question:** Can I automate the removal of followers as well?


**Answer:** Yes, you will be able to do so by using the **Remove Contact Followers Workflow Action** within the platform.

![](https://assets2.modalsupport.net/1768842880246-image.png)




**Question:** If a contact is already being followed by a user, what happens if the workflow runs again?


**Answer:** The system is smart enough to recognize existing followers. It will not create duplicate entries for the same user; it will simply ensure that the user remains on the follower list.
