In this article, we will walk through how to use the Add To Affiliate Manager workflow action. This is useful to automate adding Affiliates to the Affiliate Manager.

## **Navigation**

### **Step 1:** Access Workflows

-   Navigate to Automation on the left > Workflows on the top and create a new workflow or edit an existing one

![](https://assets2.modalsupport.net/1762559701946-image3.png)

### **Step 2: Set up applicable Workflow Trigger(s).** 

We have multiple articles available to explain the setup of each of the Workflow Triggers. Feel free to review our articles on these for more information on setting these up.  

![](https://assets2.modalsupport.net/1773760849758-image1.png)

**_Pro Tip:_** You can use any Workflow Triggers to trigger the Add To Affiliate Campaign action or you could leave this workflow without any triggers.

-   Click on the “+” symbol below the trigger to add an action.

### **Example: Convert Contacts to Affiliates When a Tag is Added**

**Scenario:** A business wants to recruit customers who have responded to a post or shown interest in joining the affiliate network. By adding a specific tag to these contacts, they can easily convert them into active affiliates.

**Action Setup:**

-   **Action:** Add to Affiliate Manager
-   **Name:** Convert Contacts to Affiliates

**Workflow Trigger:**

-   **Trigger:** Contact Changed  
    The contact will enter the workflow when the specified filter is added or changed.
-   **Filters:** Tag added: "Affiliate"

**Workflow Actions:**

-   **Action:** Add to Affiliate Manager

**Outcome:**Whenever the "Affiliate" tag is added to a contact, they will automatically be converted into an affiliate.

## FAQ’s




**Question:** Is it necessary to add a trigger to use the Add To Affiliate Manager action?
**Answer:** No, it is not necessary to add a trigger to use the Add To Affiliate Manager action. You can leave the workflow without any triggers, allowing you to manually run the workflow when needed.




**Question:** What types of Workflow Triggers can be used with the Add To Affiliate Manager action?
**Answer:** You can use any workflow triggers that fit your specific use case, such as form submissions, link clicks, or custom triggers. This flexibility allows you to automate the addition of affiliates based on various actions.




**Question:** How can I ensure that my workflow is published and live?
**Answer:** Ensure your workflow is published and live by setting the toggle switch under Save to Publish. This action confirms that the workflow is active and ready to execute the Add To Affiliate Manager action.




**Question:** What should I name my workflow when adding the Add To Affiliate Manager action?
**Answer:** Naming your workflow should be descriptive and relevant to its function, such as “Add New Affiliates” or “Affiliate Campaign Enrollment.” This helps keep your workflows organized and easy to identify.




**Question:** Where can I find more information on setting up Workflow Triggers?
**Answer:** You can find more information on setting up Workflow Triggers by reviewing the related articles available, which provide detailed explanations of each trigger type and how to configure them.




**Question:** What should I do after setting up the Add To Affiliate Manager action?
**Answer:** After setting up the Add To Affiliate Manager action, save your changes, test the workflow to ensure it works as intended, and publish it if everything is set up correctly.




**Question:** Why is it important to test the workflow after setting it up?
**Answer:** Testing the workflow after setting it up ensures that all triggers and actions function as expected. It helps identify and fix any potential issues before the workflow goes live, providing a smoother experience for both you and your affiliates.




**Question:** Can this action automatically convert contacts into affiliates based on tags?
**Answer:** Yes. For example, you can create a workflow where contacts tagged with “Affiliate” are automatically added to your Affiliate Manager when the tag is applied.




**Question:** If a contact is already an affiliate, will this action create a duplicate?

**Answer:** No. is designed to recognize existing affiliate profiles. If a contact is already registered in your affiliate manager, the system will simply skip the creation step or update their existing status, ensuring your records remain clean.




**Question:** Can I add a contact to a specific Affiliate Campaign using this action?
**Answer:** Yes. When configuring the "Add to Affiliate Manager" action, you will be prompted to select which specific campaign(s) the contact should be enrolled in. This ensures they are immediately assigned the correct commission rates and links.
