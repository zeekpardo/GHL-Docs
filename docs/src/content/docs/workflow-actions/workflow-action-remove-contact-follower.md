**Workflow Action: Remove Contact Follower**

The "Remove Contact Follower" action seamlessly integrates into workflows, facilitating the automated removal of followers from contact profiles. By eliminating the need for manual intervention, this feature streamlines follower management processes and enhances overall workflow efficiency.

## How to Use

**1\. Access Workflow Editor:** Navigate to Automation on the left > Workflows on the top and create a new workflow or edit an existing one

![](<https://assets2.modalsupport.net/1766072794191-Captura de pantalla 2025-12-18 104625.png>)

**2\. Add Action**: Select the option to add a new action within the workflow editor interface.  
![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeD42lj1cQVPjSsvuYS-d16iiosF2iCbOPg1CuDG2EdY9GiF4173b2PVd5T5fVmafCpKJK5r0B29YYNgn2GUpXxYicuvyRnK_YZg3nnH9zkwfFeYolwvCoMK6QgTsB3HBjRxEh3?key=f2Xzym0Ab4ybygti-RQ1xQ)

**3\. Select "Remove Contact Follower":** Locate the "Remove Contact Follower" action among the available options and add it to your workflow.  
![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXczbOctid992XA9afz4CxJEcLUwpmLXmSH2VFv-w7xjzu1d7epHFHRPILWcyRfsMlvOv8TgBk20cg5AitPrcFdpx7jglfoM5whjvjFqZDd_11Rh0OAM455Qo-9abMjjJGowgAlpGCsadDKTQvoUtb0bXyqK?key=f2Xzym0Ab4ybygti-RQ1xQ)

**4\. Set Up Action Parameters**: Specify which contact's followers you want to remove by selecting their profile.

**5\. Monitor and Review**: Keep an eye on the workflow execution and check the results to ensure followers have been successfully removed.

### **Example**

**Configuration Example:**

In a workflow where contacts are reassigned based on their engagement stage:

**Action Name:** "Remove Contact Follower"

**Workflow Steps:**

-   **Trigger:** Opportunity Status Changed – This trigger activates when an opportunity linked to the contact is marked as "Closed."
-   **Action:** Remove Contact Follower – This action removes the follower assigned to the contact.

### FAQs




**Question:** How do I know if the followers have been removed successfully?
**Answer:** Monitor the workflow execution results and review the contact profiles to verify that the followers have been successfully removed.




**Question:** Can I customize the criteria for removing followers?
**Answer:** Yes, you can use a combination of Workflow triggers and other Workflow actions before you remove a follower to ensure they are only removed when you intend them to.




**Question:** Is it possible to undo the removal of followers once the workflow has been executed?
**Answer:** Typically, the removal action is permanent. Ensure that you review the contact profiles before executing the workflow to avoid unintended changes.




**Question:** Where can I find the "Remove Contact Follower" action in the menu?

**Answer:** You can find this action by clicking the **"+"** icon in your workflow builder. It is typically located under the **Contact** category of actions. You can also use the search bar at the top of the actions menu to find it quickly.




**Question:** Can I use this action to remove followers when a sales deal is finished?

**Answer:** Yes. For example, you can set the workflow trigger to "Opportunity Status Changed" so that when an opportunity is marked as "Closed," the follower assigned to that contact is automatically removed.




**Question:** Does this action only allow for the addition of followers, or can it handle removal as well?

**Answer:** While the specific action is titled "Add Contact Follower," this feature is designed to streamline the follower management process by allowing users to automate both adding and removing followers.
