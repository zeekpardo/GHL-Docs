The Manual SMS Action within a workflow allows users to create a task to send a personalized SMS message to a lead at a time that suits them. This feature is especially useful when you want to ensure that the message is tailored and sent at the optimal time for engagement.

### Overview of the Manual SMS Action

This feature works by creating a manual task in the system for the user assigned to that lead. Once the task is created, the user can decide when to send the message and customize it further if needed.

### Step-by-Step Guide

#### Part 1: Creating or Editing a Workflow with a Manual SMS Action

**1- Access Your Workflow:**

-   Navigate to an existing workflow or create a new one.
-   Click the "+" button to add a new action.

![](<https://assets2.modalsupport.net/1767692910007-unnamed \(66\).png>)

**2- Choose "Manual SMS":**

-   From the list of available actions found in Communication, select "Manual Action to SMS", also known as Manual SMS.

![](<https://assets2.modalsupport.net/1767692928529-unnamed \(67\).png>)

-   Customize the message template or content as needed.

![](<https://assets2.modalsupport.net/1767692951350-unnamed \(68\).png>)

#### Part 2: Assigning the Task to the Correct User

To ensure the right user receives the task to send the SMS, you’ll need to assign a user within the workflow.

**1- Add the "Assign to User" Action:**

-   Place this action at the beginning of the workflow.

![](<https://assets2.modalsupport.net/1767692978548-unnamed \(69\).png>)

-   Select the user or team who should receive the task.

![](<https://assets2.modalsupport.net/1767692999793-unnamed \(70\).png>)

**2- Save and Activate Your Workflow:**

-   Ensure all settings are correct and save the workflow.

#### Part 3: Testing the Workflow and Manual SMS Action

Before launching the workflow, it’s a good practice to test it using a sample contact.

**1- Add a Test Contact:**

-   Use a sample/test contact. If you need help creating one, check the relevant tutorials in the help library.

![](<https://assets2.modalsupport.net/1767693020369-unnamed \(71\).png>)

**2- Test the Workflow:**

-   Add the test contact to the workflow and monitor the process.

**3- Check the Manual SMS Task:**

-   Go to **Conversations > Manual Actions** to confirm that the Manual SMS task was created.

![](<https://assets2.modalsupport.net/1767693055856-unnamed \(72\).png>)

-   Click "Let’s Start," customize the message if needed, and hit send.

![](<https://assets2.modalsupport.net/1767693072908-unnamed \(73\).png>)

### Tips for Using the Manual SMS Action

-   **Double-check User Assignments:** Make sure the "Assign to User" action is properly set up to avoid any missed tasks.
-   **Customize Messages:** While the workflow can provide a basic template, manually customizing the message can improve engagement.
-   **Test Before Launching:** Always run a test contact through the workflow to ensure everything works as expected.

### FAQs




**Question:** What is the Manual SMS Action in a workflow?
**Answer:** The Manual SMS Action allows you to create a task for a user to manually send a personalized SMS to a contact. This task is assigned within the workflow and is accessible in the Conversations > Manual Actions section.




**Question:** How do I assign a user to send the Manual SMS?
**Answer:** To assign a user, add the "Assign to User" action in the workflow before the Manual SMS Action. Select the user or team who should receive the task, ensuring the correct person is responsible for sending the SMS.




**Question:** Can I customize the message before sending the Manual SMS?
**Answer:** Yes, when the task is created, the user can further customize the message template before sending it, allowing for more personalized and engaging communication.




**Question:** How do I test the workflow before launching it?
**Answer:** To test the workflow, create a sample/test contact and add it to the workflow. Check the Conversations > Manual Actions section to confirm the Manual SMS task is created and working as expected.




**Question:** What should I do if the Manual SMS task is not assigned to the correct user?
**Answer:** Double-check the "Assign to User" action in your workflow. Ensure it is placed correctly and assigned to the appropriate user or team to avoid any missed tasks.




**Question:** How does a Manual SMS Action fit into complex workflows?
**Answer:** In workflows that involve multiple systems or teams, Manual SMS Actions can act as checkpoints. These provide an opportunity for human intervention to ensure everything functions correctly or handle exceptional cases that may require individual attention.




**Question:** What is the benefit of using a Manual SMS task over an automated SMS?

**Answer:** Manual SMS tasks allow for **optimal timing and high-level personalization**. Unlike automated messages that send immediately based on triggers, manual actions let a human agent decide the exact moment to hit send and tweak the content to fit the specific lead's needs.




**Question:** What is the most important step to take before fully launching a workflow that includes manual actions?

**Answer:** You should always **run a test contact** through the workflow. This allows you to verify that the task is appearing correctly in the Manual Actions tab and that the user assignment is functioning as intended.
