**Number Intelligence (Number Validation): Block Spam Calls and Identify Unknown Callers**

Efficient communication is essential for the success of any business. Number Intelligence, previously known as the Number Validation, is a feature designed to save time, increase productivity, and minimize disruptions. With improved spam call detection, caller identification, and phone number validation, Number Intelligence ensures that every call you receive is meaningful and worthwhile.

**Important Note:**

The Number Intelligence feature, including Spam Detection and Name Lookup, will be available and applicable starting **November 25, 2024**. Accounts already using Number Validation will see the capabilities automatically activated on this date. Ensure your settings are configured as needed to make the most of these enhancements moving forward.

### Key Features and Benefits

-   **Block Spam Calls:** Automatically flags spam calls from unknown U.S. numbers, reducing disruptions.
-   **Identify Unknown Callers:** Retrieves caller names for unknown numbers, enabling more personalized and informed interactions.
-   **Validate Numbers Before SMS:** Verifies the SMS capability of numbers before sending messages, preventing wasted efforts on invalid numbers.
-   **Seamless Integration:** Works automatically for accounts already using Number Validation, with easy opt-in or opt-out options.

### How to Use

**1\. Enable Number Intelligence (If not already active):**

-   Navigate to Account Settings (located on the bottom left corner of your screen) > Phone System > Additional Settings > Number Intelligence.
-   Check the feature to enable spam detection, caller identification, and number validation.

**2\. Configure Preferences: (Opting out)**

-   **Opt out now**: Disable Number Validation under Account Settings > Phone Numbers > Messaging > Messaging Compliance to avoid any related charges now and after the release.

![](https://assets2.modalsupport.net/1767017529936-image1.png)

-   **Opt out later**: Disable Number Intelligence under Account Settings > Phone Numbers > Messaging to avoid any related charges after the release on Monday, Nov 25, 2024

![](https://assets2.modalsupport.net/1767017557761-image2.png)

**3\. Web and Mobile App Compatibility:**

-   On the web app, access all three features: Spam Detection, Name Lookup, and Number Validation.
-   On the mobile app, utilize Name Lookup and Number Validation

**4\. Start Filtering and Identifying Calls:**

-   Automatically block spam and identify unknown callers.
-   Receive notifications for invalid SMS-capable numbers during communication.

**5\. Call Handling with Spam Detection**

Incoming calls that are flagged by Spam Detection will be labeled as "Spam Likely" on the web dialer when received and will also be tagged in the Contacts tab for better transparency.

**FAQs**




**Question:** How does Spam Detection work?
**Answer:** Spam Detection uses an advanced algorithm to identify and block suspected spam calls from unknown U.S. numbers.




**Question:** Can I opt out of Number Intelligence?
**Answer:** Yes, you can disable Number Intelligence or individual features in the Account Settings under Advanced Settings.




**Question:** Does this feature work for international numbers?
**Answer:** Currently, Spam Detection applies to U.S. numbers, but Name Lookup and Number Validation support international numbers based on availability.




**Question:** Will Spam Detection affect legitimate calls?
**Answer:** The system is designed to minimize false positives. You can review flagged calls to ensure legitimate calls are not blocked.




**Question:** Are there additional charges for using Number Intelligence?
**Answer:** Yes, usage charges apply. Check your account settings for detailed pricing information.




**Question:** How can Number Intelligence help my team?
**Answer:** It reduces wasted time on spam calls, identifies unknown callers for more personalized conversations, and validates SMS numbers before campaigns helping teams, especially sales, focus on qualified leads and high-value tasks.




**Question:** Will I see the caller's name even if they aren't in my contact list?

**Answer:** Yes. The **Name Lookup** feature retrieves caller names for unknown numbers. This allows you to greet callers personally and have more informed interactions before you even save them as a contact.




**Question:** How will I know if a call is being flagged as spam while I'm using the web dialer?

**Answer:** Incoming calls flagged by the system will be clearly labeled as **"Spam Likely"** on the web dialer. Additionally, these calls will be tagged in your **Contacts tab** for better transparency and record-keeping.
