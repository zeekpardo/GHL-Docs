**Bulk Actions - Add or Remove Tags**

Using the Bulk Actions features in your Contacts/Smart Lists section, you can perform multiple one-off actions to selected or all contacts. Here’s how to add or remove tags using the bulk action feature:

### **Step 1: Choose Your Contacts** 

-   Navigate to your contacts.
-   Check the box to select contact records.

![](https://assets2.modalsupport.net/1763588487784-image9.png)

-   You can also check the box at the top to select all on that page, and you can click the hyperlink to select all contacts in .

![](https://assets2.modalsupport.net/1763588532824-image7.png)

### **Step 2: Choose Your Action**

-   Choose the icon for “Add Tag” or “Remove Tag”.

![](https://assets2.modalsupport.net/1763588607623-image8.png)

-   In the pop-up window that appears, type in a keyword and select one or more tags to apply or remove. Then click “add tags” to add them to contacts, or “remove tags” to remove them.

## FAQs:




**Question:** How do I select all contacts in my list for a bulk action?
**Answer:** Navigate to your Contacts or Smart Lists section and check the box at the top of the list to select all contacts on that page. To select all contacts in your company, click the hyperlink that appears after selecting the page contacts.




**Question:** Can I add multiple tags to contacts at once using the bulk action feature?
**Answer:** Yes, you can add multiple tags to selected contacts at once. In the pop-up window, type in keywords and select one or more tags to apply. Click "Add Tags" to apply them to the contacts.




**Question:** How do I remove tags from multiple contacts at once?
**Answer:** After selecting the contacts, choose the "Remove Tag" icon. In the pop-up window, type in the tag(s) you want to remove, then click "Remove Tags" to remove them from the selected contacts.




**Question:** What happens if I try to add a tag that already exists on a contact?
**Answer:** If the tag already exists on the contact, it will not be duplicated. The system will simply retain the existing tag without adding another copy.




**Question:** Can I undo a bulk action if I make a mistake?
**Answer:** Unfortunately, there isn't an "undo" feature for bulk actions. You would need to manually remove or add tags as needed to correct the mistake.




**Question:** Do bulk actions apply to tags in Smart Lists as well?
**Answer:** Yes. You can perform bulk add or remove tag actions on contacts in any Smart List, just like in the main Contacts section.




**Question:** Can I schedule bulk actions for a later time?
**Answer:** No, bulk actions are executed immediately after confirmation and cannot be scheduled for a future time.




**Question:** Do I have to scroll through every page of contacts to select my entire database?

**Answer:** No. After you check the box at the top to select all contacts on the current page, a hyperlink will appear that allows you to select every contact in your entire account with one click.




**Question:** How do I find the specific tags I want to apply or remove within the bulk action pop-up window?
**Answer:** In the pop-up window that appears after selecting your contacts, you can type in a keyword to search for your existing tags. You can then select one or more tags from the results to apply or remove.
