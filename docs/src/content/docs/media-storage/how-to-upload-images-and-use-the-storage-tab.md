Your Media Storage allows you to store image files for use across . You can upload images from your own devices, add images from Pixabay and Unsplash (free image providers), and organize/categorize them into folders. Saving images in your Media Storage allows you to regularly access and utilize the same images or logos repeatedly without having to re-upload them.

### Step-by-Step Guide to Using Media Storage

#### Step 1: Accessing Media Storage

Navigate to Media Storage:

-   Click on Media Storage from the left sidebar.

#### Step 2: Navigating Media Storage

Within Media Storage, you can sort through files and folders. You can upload media and create folders using the buttons in the top right corner. You can search at the top for media you’ve uploaded and add images from Unsplash or Pixabay (free image providers).

### Sync Google Drive with Your Media Storage

Users can sync files directly between Google Drive and their media storage in . This integration simplifies file management, allowing users to access and organize all their essential files from within , eliminating the need to switch between platforms. 

**How to Use**: Connect your Google Drive to Media Storage, then drag, drop, and organize files as needed. Note that folder creation and uploads are two-way synced, while rename and delete functions remain disabled for Google Drive to protect file integrity. ### FAQ's




**Question:** What feature is available in Media Storage?
**Answer:** You can upload compressed .zip files in Media Storage, with a size limit of 100 MB.  




**Question:** How do I upload a zip file to Media Storage?
**Answer:** Navigate to Media Storage, click the New Upload button, select "Upload File," and choose the zip file from your device.  




**Question:** What types of files can I upload to Media Storage?
**Answer:** Supported file types include images (PNG, JPEG, GIF, etc.), documents (PDF, Word, Excel, etc.), videos (MP4, AVI, MOV, etc.), and audio files (MP3, WAV, etc.).  




**Question:** Are there any size limitations for different file types?
**Answer:** Yes, video files can be up to 4 GB, documents and audio files up to 100 MB, and images up to 100 MB.  




**Question:** What improvements have been made to Media Storage?
**Answer:** Enhancements include a complete UI overhaul, faster performance, better thumbnail quality, advanced sorting, and improved drag-and-drop functionality.  




**Question:** Can I use images from Media Storage in different parts of the system?
**Answer:** Yes, images added to Media Storage can be used across various sections of the system, including funnels, websites, email builders, and social planners.  




**Question:** What options are available for organizing media?
**Answer:** You can sort media by file size, name, size, and modified date, and choose between grid and list views. Folders also allow for easy organization and management.  




**Question:** How can I quickly access and manage files in Media Storage?
**Answer:** Use right-click menus for quick actions like renaming, moving, or sharing files, and hover over files to view details such as file name and size.  




**Question:** How does syncing Google Drive work with Media Storage?
**Answer:** You can sync files directly between your Google Drive and Media Storage. This integration allows for easy file management by letting you access and organize files from within Media Storage. Folder creation and uploads are two-way synced, but rename and delete functions remain disabled for Google Drive to protect file integrity.  




**Question:** What happens to images I select from third-party providers like Unsplash or Pixabay if the original source is deleted?
**Answer:** Images selected from Unsplash or Pixabay are automatically copied to your own servers. This ensures that the images remain visible and accessible in your Media Storage even if the original provider removes them.
