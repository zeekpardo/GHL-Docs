The Media Storage tab is your central hub for managing and organizing all your media files. Here’s an overview of its features:

-   **Access and Navigation**: The Media Storage tab is accessible from the left sidebar. It centralizes all your media assets in one location.
-   **View Options:** Choose between grid and list views to organize and view your media files in the way that best suits you.
-   **Enhanced Drag and Drop:** Upload media files quickly and easily using improved drag-and-drop functionality.
-   **Advanced Sorting:** To find what you need more efficiently, sort your media by various criteria, including file size, name, and modification date.
-   **Quick Access Menus:** Utilize right-click dropdown menus and hover-click options for fast file actions, such as moving, deleting, or renaming files.
-   **Private File Upload Support**Private files can also be uploaded securely alongside public files in the media storage tab.

### **FAQ**


**Question:  How do I access the Media Storage tab?
**Answer:** Navigate to Media Storage by clicking on its icon from the left sidebar.




**Question:** What file types can I upload to Media Storage?
**Answer:** Media Storage supports a wide range of file types, including images, videos, and documents, up to 4GB. There are no specific storage limits.




**Question:** How can I sort and organize my media files?
**Answer:** You can sort your media by file size, name, and modification date. Use the sorting options and choose between grid and list views for optimal organization.




**Question:** What are the quick actions available in Media Storage?
**Answer:** Right-click on any media file to access options such as moving, deleting, or renaming. Hover over a file to view details or click on an image to see a detailed preview and download it or copy its link.




**Question:** Can I upload private files?
**Answer:** Yes, you can upload files as private. They are marked with a lock icon, and private links are valid for 10 minutes.




**Question:** What are the different Media Storage modes?
**Answer:** There are three modes:

-   **All**: Access both public and private files.
-   **Public**: View public files only.
-   **Private**: Manage private files with download or link-sharing options.




**Question:** How should I name and tag files for better organization?
**Answer:** Use clear and descriptive **file names** and **keywords** to improve searchability and help team members quickly locate assets.




**Question:** What happens to deleted files?
**Answer:** Deleted files go to **Trash** for 30 days. You can either **restore** them or **delete them permanently** to free up space.




**Question:** Are there different file size limits depending on where files are used?
**Answer:** Yes. File size limits vary by feature (Media Storage, Funnels, Social Planner, Memberships). Refer to the Media Storage Specs & Formats section for exact limits.  




**Question:** What happens to my Google Drive data when using the "Login As" (proxy access) feature?
**Answer:** To ensure user privacy and security, the "Connect Drive" button is disabled and all Google Drive data is hidden when a user is accessed via the "Login As" feature.
