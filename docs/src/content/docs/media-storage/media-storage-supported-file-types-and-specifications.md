### FAQ's




**Question:** What types of files can I upload to the Media Storage?
**Answer:** You can upload a variety of file types, including images (PNG, JPEG, GIF, SVG, etc.), videos (MP4, AVI, MOV, WebM, etc.), audio files (MP3, WAV, MIDI, etc.), documents (PDF, Word, Excel, PowerPoint, etc.), fonts (TTF, OTF, WOFF), and more. Refer to the file type list for specific details on supported formats.  




**Question:** What is the maximum file size I can upload to Media Storage?
**Answer:** The maximum file size varies depending on the file type and the platform it is being uploaded to. For example, image files like PNG and JPEG can be up to 100MB, while videos can range from 50MB to 4GB. Please check the corresponding file types and their limits in the article above.  




**Question:** How do I organize my media in the Media Storage?
**Answer:** You can create folders to organize your content and easily search through your media, including files from Pixabay, Unsplash, or system backgrounds.  




**Question:** Can I upload videos larger than 100MB?
**Answer:** Yes, videos such as MP4, AVI, MOV, and other video file formats can be uploaded with file sizes ranging from 50MB to 4GB, depending on the platform you're using.  




**Question:** Can I upload Google Docs or Google Sheets to the Media Storage?
**Answer:** Yes, you can upload Google Docs, Google Sheets, and other Google file types (Google Slides, Google Forms) to the Media Storage, with a size limit of 100MB for these documents.




**Question:** Can I upload files from external sources like Google Drive or Dropbox?
**Answer:** Currently, Media Storage does not support direct uploads from external sources like Google Drive or Dropbox. However, you can download the files to your local system and then upload them manually to the Media Storage.  




**Question:** Are all file types supported across all tools (Sites, Social Planner, Courses, etc.)?
**Answer:** Not always. While most file types are supported across platforms, some tools have lower size limits or may not support certain file types. Refer to the file-type table for each tool’s compatibility and limits.  




**Question:** Can I share a direct link to a file in my Media Storage with someone else?

**Answer:** Yes. Each file has a **"Get Link"** option. This generates a public URL that you can send to a client or team member so they can view or download the file without logging into the platform.




**Question:** What is the maximum file size for a video if I am uploading it specifically for a Course?

**Answer:** For the Courses section, the maximum supported size for video files (such as MP4, MOV, or AVI) is 5GB. This is higher than the 4GB limit typically allowed for Sites, Funnels, and the general Media Library.
