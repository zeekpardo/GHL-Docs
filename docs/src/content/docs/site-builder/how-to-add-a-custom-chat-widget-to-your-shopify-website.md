Welcome! If you’re looking to connect with your website visitors in real time and boost conversions, adding a custom chat widget is a great place to start. This feature lets you engage visitors directly, answer their questions, and provide support instantly. In this article, we’ll walk you through the simple steps to add a chat widget to your Shopify website.

### Why Add a Chat Widget?

A chat widget is a powerful tool to:

-   **Improve Customer Engagement**: Reach visitors instantly and build stronger relationships.
-   **Increase Conversions**: Provide immediate responses to potential customers, guiding them through their buying journey.
-   **Provide Instant Support**: Answer questions in real time and resolve issues promptly.

### How to Add a Chat Widget to Your Shopify Website

Here’s how to set up a chat widget on your Shopify website. Don’t worry—this guide will make it easy!

#### Step 1: Obtain the Embed Code

To get started, you’ll first need the chat widget embed code. Follow these steps:

**1- Navigate to the Chat Widget Section**: Look for an option called **Sites on the left menu bar > Chat Widget on the top right**.

Click on any existing chat widget or create a new one.

![](<https://assets2.modalsupport.net/1766486784240-unnamed \(13\).png>)

**2- Get the Code**: Select **Get Code** to copy the embed code to your clipboard.

![](<https://assets2.modalsupport.net/1766486800678-unnamed \(14\).png>)

#### Step 2: Edit Your Shopify Website’s Code

Now, let’s add the embed code to your website.

**1- Log in to Your Shopify Website Dashboard**: Open your website’s dashboard.

**2- Access Your Theme**: Navigate to the “Online Store” section and select your current theme.

![](<https://assets2.modalsupport.net/1766486839100-unnamed \(15\).png>)

**3- Edit the Code**: Click on the “Actions” button next to your theme and select **Edit Code**.

![](<https://assets2.modalsupport.net/1766486857139-unnamed \(16\).png>)

#### Step 3: Add the Embed Code to Your Shopify Website

**1- Open the theme.liquid File**: In the theme code editor, look for a file named **theme.liquid** located under the **Layout** folder.

![](<https://assets2.modalsupport.net/1766486878200-unnamed \(17\).png>)

**2- Locate the Closing </body> Tag**: Scroll to the bottom of the file and find the **</body>** tag.

-   This is where we’ll add the embed code.

**3- Paste the Embed Code**: Above the closing **</body>** tag, paste the code you copied earlier.

![](<https://assets2.modalsupport.net/1766486903079-unnamed \(18\).png>)

#### Step 4: Save and Preview

**1- Save Changes**: After pasting the embed code, make sure to click **Save** to apply the changes.

![](<https://assets2.modalsupport.net/1766486923741-unnamed \(19\).png>)

**2-Preview Your Site**: Refresh your website to see the chat widget in action. You should see the widget appear in the bottom corner of the page.

### FAQ’s




**Question:** Will the chat widget work on all browsers?
**Answer:** Yes! The widget is designed to be compatible with all major browsers, including Chrome, Firefox, Safari, and Edge.




**Question:** How do I customize the appearance of the chat widget?
**Answer:** Customization options vary depending on the platform you're using. Typically, you can adjust colors, position, and other design elements through your platform’s settings.




**Question:** Can I integrate the chat widget with my email or ?
**Answer:** Yes! Most platforms allow you to integrate the chat widget with your email or to track customer interactions and follow up with leads.




**Question:** Is the chat widget mobile-friendly?
**Answer:** Absolutely! The widget is responsive and will work smoothly on both desktop and mobile devices.




**Question:** How can I track the performance of my chat widget?
**Answer:** Check your platform’s analytics or dashboard for insights on how many people are using the chat widget, response times, and customer satisfaction.




**Question:** Is the chat widget GDPR/Privacy compliant?
**Answer:** Most platforms provide privacy features, such as consent messages. You should ensure compliance with GDPR/CCPA by configuring consent notices in your chat widget settings.




**Question:** Do I need coding skills to add the chat widget?

**Answer:** No advanced coding skills are required. While you do need to access the "Edit Code" section of Shopify, the process is a simple "copy and paste" into a specific file (theme.liquid).  




**Question:** Where is the best place to paste the code within thetheme.liquidfile?

**Answer:** You should scroll to the bottom of the file and paste the embed code **directly above the closing</body>tag**. This ensures the widget loads correctly without interfering with other site elements.




**Question:** I pasted the code but I don't see the widget. What should I do?

**Answer:** First, ensure you clicked Save in the Shopify code editor. Second, check if you have any "popup blockers" active in your browser. Finally, verify that the "Enable Chat Widget" toggle is turned ON within your platform's Chat Widget settings.
