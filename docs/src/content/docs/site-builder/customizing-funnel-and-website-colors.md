**Customizing Funnel and Website Colors**  

You can customize each step within the funnel or website builder, including colors, fonts, and more. Follow these steps to learn how to customize the colors of your funnel or website using several different methods.

You can create overall Funnel/Website Settings by utilizing the Typography Settings to ensure consistency throughout your page. You can also customize individual funnels or website pages for the builder. If you are familiar with code, you can also use custom CSS.

**_Please Note:_** _For guidance on customizing fonts, refer to the article “Custom Fonts in the Site Builder.”_

### **Part 1: Updating the Overall Font Colors Using Typography Settings**

Using the Typography tab allows you to ensure that all fonts are consistent throughout the funnel/website pages. When customizing the fonts, you can choose the specific color for text, links, and more.

-   Navigate to the “Settings” button at the top of the builder.
-   Click on Typography.

![](https://assets2.modalsupport.net/1764180337113-image.png)

-   Choose the native Headline and Content fonts used throughout the funnel.

![](https://assets2.modalsupport.net/1764180396972-image.png)

### **Part 2: Making Individual Page Edits**

Each funnel/website page comprises the following pieces: Sections, Rows, Columns, and Elements. They need to be created in that order. Sections can house multiple rows, rows can house multiple columns, and columns can house various elements.

![](https://assets2.modalsupport.net/1764180493311-image.png)

Using the drag-and-drop builder, you can add sections, rows, columns, and elements to the builder. Then, use the edit menu on the left-hand side for each piece you’ve added to customize it to your liking.

Here are some examples of specific elements that colors can be customized within:

-   Background and text color of a button

![](https://assets2.modalsupport.net/1764180523412-image.png)

-   The background color of a section
-   Text colors for a headline or other text elements

![](https://assets2.modalsupport.net/1764180651174-image.png)

-   Navigation Menu Popup background Colours (such as the mobile hamburger menu and search bar.)

![](https://assets2.modalsupport.net/1766668799461-image16.jpg)

### **Part 3: Custom Colors**

You can implement custom colors directly from the Site Builder. If you were to set up Custom Values with Hex code, you could create a color template that you can change within Settings > Custom Values.

**_Please Note:_** _The saved colors will only appear within the funnel page to which they were added. Setting up custom colors for every page would be necessary._

-   You can add a custom color by clicking on the text element and selecting “**\+ Add**” option in the text element settings.

![](https://assets2.modalsupport.net/1764180759414-image.png)

-   Add the name of the color directly and enter the hex code.

-   Or, select the color from the visual color editor.

![](https://assets2.modalsupport.net/1764180803330-image.png)

-   Add a custom value and put the hex code into the value.

![](https://assets2.modalsupport.net/1764180840288-image.png)

-   Place the custom value into the custom color option.

![](https://assets2.modalsupport.net/1764180857414-image.png)

Change the hex values in the custom values area to change the colors automatically for the pages this is set up within!

### **Part 4: Using Custom CSS**

If you have the expertise, custom CSS allows you to customize the funnel/website beyond standard options completely. CSS, or Cascading Style Sheets, is the technology used by most websites to create visually engaging web pages, including layout, colors, and fonts.

**_Please Note:_** _If you lack the expertise, you may need to contact a developer for assistance._

-   Navigate to the button at the top of the builder.
-   Click on Custom CSS.

![](https://assets2.modalsupport.net/1764180882914-image.png)

-   Insert your custom CSS.

![](https://assets2.modalsupport.net/1764180900762-image.png)

-   Save to confirm your changes.

With any changes you make, using any option, you can preview the page to view it before you save and exit the builder.

Make sure to save any changes before you leave the builder to confirm the updates you've made.

### FAQs




**Question:** How do I apply consistent font colors across all pages in my funnel or website?
**Answer:** Use the Typography settings under the “Settings” button at the builder's top. This allows you to set uniform font colors for headlines, content, links, and more across all pages.




**Question:** Can I customize colors for individual sections or elements on specific pages?
**Answer:** You can edit colors for individual sections, rows, columns, and elements directly within the drag-and-drop builder using the left-hand edit menu.




**Question:** How can I set up custom colors using hex codes?
**Answer:** You can add custom colors by selecting a text element and using the “Didn’t find your color? Add a custom one!” option. Enter the hex code and name the color; it will be available on that page.




**Question:** Is it possible to apply CSS for advanced customization?
**Answer:** Yes, if you have coding expertise, you can use Custom CSS by navigating to “Settings” > “Custom CSS.” Insert your CSS code to customize your design fully.




**Question:** Can I create a unified color palette for a consistent page design?
**Answer:** You can access the color palette from the right sidebar or inline toolbar while editing text. This allows you to manage default and custom colors, ensuring consistency across all elements.




**Question:** Will custom colors be saved across all pages in my funnel or website?
**Answer:** No, custom colors are saved only within the specific page they are set up on. You would need to replicate the setup for each page.




**Question:** How do I preview my changes before saving?
**Answer:** You can preview your page using the “Preview” option in the builder before saving and exiting to ensure your changes look as expected.




**Question:** Do I need coding knowledge to customize colors?
**Answer:** Basic color customization can only be done with coding. However, Custom CSS can be used for advanced design, and you may need assistance from a developer.




**Question:** How can I apply gradient or blur backgrounds to my funnel or website?
**Answer:** Use the Page Builder to apply gradient and blur effects to sections, rows, and columns. The background settings allow easy adjustments between colors, images, or videos.




**Question:** How can I change colors across multiple pages automatically without editing each one?

**Answer:** You can create a **Custom Value** (found in Settings > Custom Values) and input a Hex code. If you use that Custom Value in the color settings of your pages, updating the Hex code in your main settings will automatically update the colors across all pages using that value.




**Question:** Will "Custom CSS" affect the loading speed of my website?

**Answer:** Minimal CSS has a negligible impact on speed. However, if you are adding thousands of lines of code or calling external stylesheets, it could slow down the initial render. For color and font tweaks, using the built-in Custom CSS window is the most efficient and safe method.
