Global sections are a great way to set up your website efficiently, with sections that will appear on every page. This saves you time, as sections won't need to be copied over from page to page. A header or footer, for example, are common global section that appears on the top of most websites. In this article, you will learn how to create and save global sections. 

### **Part 1: How To Create And Save Global Sections.** 

-   Navigate to the builder 
-   Click on the + icon on the top left, and add a section

![](<https://assets2.modalsupport.net/1763667128625-Captura de pantalla 2025-11-20 143200.png>)

-   Edit the section as needed by adding your columns, rows, and elements. 
-   Navigate back to Section by clicking on the layers icon, then on the section, then on the 3 dots, then on Save Section

![](<https://assets2.modalsupport.net/1763667172664-Captura de pantalla 2025-11-20 143239.png>)

![](<https://assets2.modalsupport.net/1763667193286-Captura de pantalla 2025-11-20 143307.png>)

-   Name the section and save it as a global section

![](<https://assets2.modalsupport.net/1763667219828-Captura de pantalla 2025-11-20 143330.png>)

### **Part 2: How To Add Your Previously Saved Global Section**

-   Click on the + icon to add the Global section you just saved

![](<https://assets2.modalsupport.net/1763667247784-Captura de pantalla 2025-11-20 143401.png>)

-   Click, drag, and drop the global section where you need it.

![](https://assets2.modalsupport.net/1773335900377-image6.png)

-   Save

## **FAQ’s**




**Question:** Will updates to a global section automatically apply to all pages where it is used?
**Answer:** Yes, any updates made to a global section will automatically apply to all pages where that section has been added. This ensures consistency across your site and saves time on manual updates.




**Question:** Do I need to add the global section to each page individually?
**Answer:** Yes, you need to add the global section to each page where you want it to appear. Once added, any changes made to the global section will reflect on all pages where it has been placed.




**Question:** How do I edit a global section after it has been added to a page?
**Answer:** To edit a global section, navigate to the section within the builder, make your desired changes, and save. The updates will then be applied automatically to all pages where the global section is used.




**Question:** What types of content are best suited for global sections?
**Answer:** Common uses for global sections include headers, footers, navigation bars, and other elements that are consistent across multiple pages, such as contact information or social media links.




**Question:** Can I save multiple global sections, and how do I manage them?
**Answer:** Yes, you can save multiple global sections. Each section can be named and managed within the builder, allowing you to easily add, update, or remove sections as needed.




**Question:** What happens if I delete a global section from one page?
**Answer:** Deleting a global section from one page will only remove it from that specific page. The section will remain available for use on other pages and can be re-added if needed.




**Question:** Can I convert a regular section into a global section?
**Answer:** Yes, you can convert a regular section into a global section by navigating to the section, clicking on the layers icon, then on the section, then on the 3 dots, and selecting Save Section as a global section.




**Question:** How do I add a previously saved global section to a page?  
**Answer:** Click the + icon in the builder, select the global section you saved, then drag and drop it onto the desired page location and save.




**Question:** What happens to my pages if I delete a Global Section from the "Global" library?
**Answer:** Deleting a section from the Global Library will prevent you from adding it to _new_ pages in the future. However, it will typically remain on the pages where it was already placed. To remove it from your site entirely, you must delete it from the individual pages or replace it.
