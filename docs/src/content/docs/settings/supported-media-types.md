You can host your brand’s precious media/resources in the Media Storage! By doing so, you’ll be able to manage everything in one area, forgoing a cluttered desktop and accessing your assets wherever you are. Here are the media formats supported in and within each feature.

The Media Storage is located in the left sidebar. Many of the features work with the same file types that are allowed in the library. Make sure to look out for file size limits. 

![](<https://assets2.modalsupport.net/1762360518909-unnamed \(1\).png>)

## **Funnels:**

When accessing the media library from Funnels, various types of media can be uploaded with specific size limits depending on the elements being used.

**Uploading an Image in a Funnel:** To upload media for an image element in a funnel, open the media library by clicking on the appropriate option for adding images. The media uploaded will adhere to the size constraints set for images within the funnel.

### **FAQ's**





**Question:** What file types are allowed for image uploads in the media library?Answer**: The allowed file types for images include PNG (.png), JPEG (.jpg, .jpeg, .jfif, .pjpeg, .pjp), TIFF (.tif, .tiff), WEBP (.webp), SVG (.svg), ICON (.ico, .cur), and GIF (.gif).




**Question:** What is the maximum file size allowed for video uploads in Funnels?Answer**: The maximum file size allowed for video uploads in Funnels is 4GB.




**Question:** Can I upload documents in Forms and Surveys?Answer**: Yes, all file types are allowed for uploads in Forms and Surveys, with a maximum file size of 50MB per subaccount.




**Question:** What file types are supported for videos in the Social Planner?Answer**: Supported video file types for Social Planner include AVI (.avi), QuickTime (.qt, .mov), MP4 (.mp4), Ogg (.ogv), Windows Media Video (.wmv, .asf), and WebM (.webm) with a maximum file size of 1GB.




**Question:** Is there a file size limit for video uploads in Social Planner?  
Answer**: Yes, the maximum file size allowed for video uploads in Social Planner is 1GB.




**Question:** Can I share files stored in Media Storage with other team members?  
Answer**: Yes, files can be shared via the platform depending on user permissions set in your account.




**Question:** Can I upload SVG files for my website logo?  
Answer**: Yes. SVG files are supported and are highly recommended for logos and icons because they stay sharp at any size and usually have a much smaller file size than PNGs.




**Question:** What is the maximum file size for image uploads in the Social Planner compared to the general Media Library?

**Answer:** In the **Social Planner**, the maximum file size for images is **10 MB**. This is significantly lower than the general **Media Library or Funnels**, which allow for image uploads up to **100 MB**.
