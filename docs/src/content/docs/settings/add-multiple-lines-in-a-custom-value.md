Custom values are a great way to autofill information in various places within the system. You can use the HTML <br> tag to split the text into multiple lines also known as the line break element. Here is how you can accomplish this within the system:

### Step 1: Adding a Custom Value

Please navigate to Settings > Custom values and click on + Add Custom Value

![](https://assets2.modalsupport.net/1775760117188-image2.png)

### Step 2: Enter the Name and Value

Name the value as preferred and within the value separate lines by including <br> to separate the text into multiple lines

![](https://assets2.modalsupport.net/1775760127811-image1.png)

### Step 3: Test the value

This example shows how to enter a custom value into an email with the email menu, you can also copy and paste this into many places within the system depending on what you are trying to accomplish. For testing that, you will need to navigate to Contacts and then open a Contact.

![](https://assets2.modalsupport.net/1775760137181-image3.png)

### Use Cases

**1-Personalized Emails:**

-   **Scenario:** You want to send personalized thank-you emails to customers.
-   **How to Use:** Create a custom value with the customer’s name and a thank-you message, including line breaks to format the text neatly.

**2-Automated SMS Responses:**

-   **Scenario:** Send automated SMS replies with multiple lines of information.
-   **How to Use:** Define a custom value for the SMS content using <br> tags to separate different pieces of information.

### FAQs




**Question:** What are custom values?
**Answer:** Custom values are predefined text or data that you can use to automatically populate fields across your system. This helps save time and maintain consistency.




**Question:** Can I use HTML tags in custom values?
**Answer:** Yes, you can use HTML tags like <br> to format text with line breaks, bold text, or other formatting options.




**Question:** Where can I use custom values?
**Answer:** Custom values can be used in emails, SMS messages, forms, Funnels & Websites and other system templates where text or data input is required.




**Question:** Can custom values be updated or edited?
**Answer:** Yes, you can edit or delete custom values from the Custom values section in the settings.




**Question:** How do I ensure that custom values display correctly?
**Answer:** Always test custom values in their intended context (e.g., email templates) to ensure they appear as expected.




**Question:** How can custom values help with personalized emails?
**Answer:** You can create a custom value containing a customer’s name or personalized message. By inserting it into an email template with line breaks, you can send neatly formatted, personalized emails automatically.  




**Question:** How do I create a new Custom Value in the settings menu?
**Answer:** To add a new value, navigate to **Settings**, select **Custom Values**, and then click on the **\+ Add Custom Value** button.




**Question:** Can I use other HTML tags besides<br>in a Custom Value?

**Answer:** **Yes.** You can use basic HTML tags like <b> for **bolding**, <i> for _italics_, or <a> for creating clickable links. However, be cautious with complex CSS or script tags, as some communication channels (like SMS) will display the raw code instead of rendering the formatting.
