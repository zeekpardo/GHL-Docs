Custom Fields allow you to create contact-specific fields that contain specific information about that contact. These allow you to add to the Standard Fields and gather whatever type of information you require. 

### Step 1: Create Custom Fields

-   Navigate to Settings on the bottom left >Select Custom Fields from the settings menu on the left>Click on All Fields on the top right

![](<https://assets2.modalsupport.net/1764675804134-unnamed - 2025-12-02T194314.273.png>)

-   Click “Add Field” 

-   In the Pop-up, you will have options for what kind of custom field you want to create
-   Choose from the Types of Custom Fields.

![](<https://assets2.modalsupport.net/1764675824998-unnamed - 2025-12-02T194338.067.png>)

**_NOTE_**: Once you create a custom field, you cannot edit the field type. To change it, you will need to delete the field and create a new one. 

Types of Fields:

-   Text Input   - single-line Textbox Field
-   Text Input - Multi-line Textbox Field
-   Text Input- Text-box list
-   Numerical - Number Field
-   Phone - Phone Number Field
-   Monetary - Monetary Field
-   Checkbox - Multi-option Checkbox Field (multiple options can be chosen)
-   Single Options - Dropdown Option Field (only one option can be chosen)
-   Multiple Option - Dropdown Option Field (multiple options can be chosen)
-   Radio - Radio (circular) Button Option Field (only one option can be chosen)
-   Date - Date Field
-   Textbox List - Textbox List Field
-   File Upload - Allows Files to be uploaded (or required) with submission - Files allowed include PDF, DOC/DOCX, JPG/JPEG, PNG, GIF
-   Signature - Signature Field

-   Enter the Name of the Field.
-   The Name of the Field is what you will see in the contact profile. This is also what will show on the form/survey if "Show Label" is enabled. 
-   Enter the Placeholder. Placeholder is what will show in the box (specifically for text or other manual entry fields) in a form/survey.

![](<https://assets2.modalsupport.net/1764675856129-unnamed - 2025-12-02T194407.389.png>)

You can add this to a specific folder group. When entering the information for this field, there will be a dropdown for “Group”. This is where you can select which folder you want this field to be organized into. 

![](<https://assets2.modalsupport.net/1764675873552-unnamed - 2025-12-02T194423.823.png>)

-   Enter any other applicable information for the field

### Step 2: Utilize Custom Fields

-   Drag and Drop From the Custom Fields Section of Form or Survey Builder to Utilize the Custom Fields. You can also use them in any other feature in the system such as workflows. Custom fields are centralized and they can be used throughout the entire system.

Using Custom Fields in Forms

![](<https://assets2.modalsupport.net/1764675899733-unnamed - 2025-12-02T194450.129.png>)

Using Custom Fields in Workflows

![](<https://assets2.modalsupport.net/1764675916696-unnamed - 2025-12-02T194509.481.png>)

**_NOTE_:** Custom Fields can also be created within the Form or Survey Builder. They cannot be edited (permanently) or deleted within the Form or Survey Builder. 

### Step 3: Edit & Delete Custom Fields

-   To either Edit or Delete A Custom Field, navigate back to Settings > Custom Fields
-   Click on any custom Value and then click on bulk action
-   Click on the edit or delete

![](<https://assets2.modalsupport.net/1764675962393-unnamed - 2025-12-02T194553.729.png>)

You can also create Custom Field Folders and map fields within the folders. 

-   Custom fields can be moved to different folders helping keep track of your custom fields.
-   These Folders will also show up in the contact Card to keep your data collection organized.
-   You are also able to seamlessly restructure contact fields

![](<https://assets2.modalsupport.net/1764675982512-unnamed - 2025-12-02T194615.002.png>)

**_NOTE_:** You can also drag and drop the fields to reorder how they are seen in a contact profile under "Additional Info". Just click on the three horizontal lines on the far left side of each Custom Field and drop it in the order you want. 

### Pro Tips

-   **Plan Custom Fields in Advance:** Before creating custom fields, plan out the types of information you need to collect and how it will be used. This helps in selecting the appropriate field types and avoids the need for frequent changes.
-   **Use Descriptive Names and Placeholders:** Choose clear and descriptive names for custom fields to make it easy for users to understand what information is being requested. Use placeholders to guide users on what should be entered in the fields.
-   **Organize Fields with Folders:** Utilize custom field folders to categorize and organize your fields effectively. This makes it easier to manage and locate fields, especially as the number of custom fields grows.
-   **Leverage Custom Fields Across Features:** Remember that custom fields are centralized and can be used across various features like forms, surveys, and workflows. This helps maintain consistency and improves data integration.
-   **Regularly Review and Clean Up Fields:** Periodically review your custom fields to ensure they are still relevant and remove any that are no longer needed. This helps in keeping your data collection organized and efficient.

**Use Cases for Custom Fields:**

1.  **Lead Qualification**: Custom fields can be used to gather additional lead information, such as industry, budget, or project timeline, helping businesses segment and prioritize leads based on their specific needs.
2.  **Customer Feedback**: Use custom fields to capture customer feedback, satisfaction ratings, or service preferences in forms or surveys, allowing businesses to tailor their services based on customer responses.
3.  **E-commerce Customer Details**: In an e-commerce environment, custom fields can store specific details such as product preferences, shipping instructions, or loyalty program status, helping to personalize the customer experience.
4.  **Event Registration**: Custom fields can be added to event registration forms to collect attendee-specific details, like dietary restrictions or session preferences, ensuring a smoother event experience for both organizers and participants.
5.  **Subscription Management**: Use custom fields to track subscription plans, renewal dates, or payment status for subscribers, allowing businesses to efficiently manage recurring payments and renewals.

### FAQ's




**Question:** Can I change the type of a custom field after creating it?
**Answer:** No, once a custom field is created, you cannot change its type. To modify the field type, you will need to delete the existing field and create a new one.




**Question:** How do I add custom fields to forms or surveys?
**Answer:** To add custom fields to forms or surveys, drag and drop the field from the Custom Fields section of the Form or Survey Builder. Custom fields can also be used in workflows and other system features.




**Question:** How can I organize custom fields effectively?
**Answer:** Use custom field folders to categorize fields and keep them organized. You can also reorder fields in a contact profile by dragging and dropping them under "Additional Info."




**Question:** What if I need to edit or delete a custom field?
**Answer:** To edit or delete a custom field, go to Settings > Custom Fields, select the field, and use the bulk action options to edit or delete it.




**Question:** Can I create custom fields directly within the Form or Survey Builder?
**Answer:** Yes, custom fields can be created within the Form or Survey Builder, but they cannot be edited or deleted from within the builder. For those actions, navigate to Settings > Custom Fields.




**Question:** What are common use cases for custom fields?
**Answer:** They can be used for lead qualification, customer feedback, e-commerce customer details, event registration, and subscription management, helping to streamline processes and personalize interactions.




**Question:** What file formats are supported for the "File Upload" custom field?

**Answer:** The File Upload field allows users to submit several types of documents and images, specifically PDF, DOC/DOCX, JPG/JPEG, PNG, and GIF.




**Question:** Is there a limit to how many Custom Fields I can create?

**Answer:** While there is no strict technical limit, having hundreds of fields can make your contact cards look cluttered. We recommend using **Folders** (as mentioned in Step 3) to group related fields together so they stay organized and easy to read.




**Question:** What is the difference between "Single Options" and "Radio" fields?

**Answer:** Both allow the user to choose only one option, but "Single Options" typically uses a dropdown menu format, while "Radio" uses circular buttons.
