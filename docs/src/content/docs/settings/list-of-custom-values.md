### FAQs




**Question:** What are custom values and where can I use them?
**Answer:** Custom values are placeholders that automatically pull in relevant data, such as contact details, user information, appointment specifics, etc., in your workflows, campaigns, and communication templates. They make your messaging more dynamic and personalized without needing manual input each time.




**Question:** How do I insert a contact’s full name in my messages or campaigns?
**Answer:** To include a contact’s full name, use the custom value {{contact.name}} wherever you want the name to appear in your message, campaign, or workflow.




**Question:** Can I customize appointment reminders using custom values?
**Answer:** Yes, you can use custom values like {{appointment.start\_time}} for appointment start time or {{appointment.cancellation\_link}} for providing cancellation links in your automated appointment reminders.




**Question:** How do I display the account information in a workflow or message?
**Answer:** You can display account information using custom values like {{location.name}} for the account name or {{location.phone}} for the account’s phone number.




**Question:** What custom value should I use to display the user’s email signature?
**Answer:** To include a user’s email signature, you can use the custom value {{user.email\_signature}} in your communication templates or workflows.




**Question:** Is there a way to include dynamic date and time values in my messages?
**Answer:** Yes, you can use custom values like {{right\_now.day\_of\_week}}, {{right\_now.hour\_ampm}}, or {{right\_now.year}} to dynamically insert the current date and time information.


**Question:  How can I include attribution data in my campaigns?
**Answer:** Attribution data like UTM parameters can be included using custom values such as {{contact.attributionSource.utmSource}} for UTM Source or {{contact.lastAttributionSource.gclid}} for the latest Google Click ID.




**Question:** What’s the difference between {{contact.phone}} and {{contact.phone\_raw}}?
**Answer:** The custom value {{contact.phone}} typically formats the phone number according to international standards, while {{contact.phone\_raw}} provides the unformatted phone number, which is recommended for use in links or UTM parameters.


**Question:Can I personalize calendar invites using custom values?
**Answer:** Yes, you can personalize calendar invites using custom values such as {{appointment.add\_to\_google\_calendar}} or {{appointment.add\_to\_ical\_outlook}} to provide quick links for adding appointments to calendars.


**Question:How can I add the assigned user’s contact information in notifications?
**Answer:** To add the assigned user’s contact details, use values like {{appointment.user.phone}} for the phone number or {{appointment.user.email}} for the email.




**Question:** How can I customize appointment rescheduling and cancellation links?
**Answer:** Yes, you can use the custom values {{appointment.reschedule\_link}} and {{appointment.cancellation\_link}} to provide direct links for rescheduling or canceling appointments in your notifications and workflows.




**Question:** How can I use custom values to include contact information in my workflows or campaigns?

**Answer:** You can use custom values like {{contact.name}}, {{contact.email}}, {{contact.phone}}, or {{contact.address1}} to include contact details such as full name, email, phone number, and address in your workflows or campaigns. These values automatically pull in the contact's data when the workflow or campaign is triggered.  




**Question:** Which custom value should be used when adding a phone number to trigger links or UTM parameters?
**Answer:** For trigger links or UTM parameters, it is recommended to use the custom value Tel:{{contact.phone\_raw}} to ensure the link functions correctly.




**Question:** Are there custom values for recurring appointments?

**Answer:** Yes. You can use {{appointment.recurring.repeats}} to show how often the meeting occurs and {{appointment.recurring.times\_to\_repeat}} to show the total number of scheduled sessions.




**Question:** Can I display the specific time zone of either the contact or the appointment in my messages?
**Answer:** Yes. You can use {{contact.timezone}} to reference the time zone set on the contact’s profile, or {{appointment.timezone}} to display the time zone specifically associated with a scheduled meeting. This is helpful for ensuring both parties are clear on the timing of a call or event.
