The Universal Search feature allows you to quickly search for contacts, payments, and opportunities within . You can easily click into any found entry, or create a new entry if the query yields no results. Check out these simple steps for how to use the Universal Search:

### Step 1: Navigate to the dashboard menu.

-   On the top left of your screen, you’ll see a search bar.

![](<https://assets2.modalsupport.net/1768475172024-unnamed - 2026-01-15T190606.729.png>)

-   Click on it and a popup will appear.

![](<https://assets2.modalsupport.net/1768475193101-unnamed - 2026-01-15T190623.636.png>)

-   In the popup window, you can type the query you wish to search for.

-   In this example, we’ll search for “test”.
-   On the left, you can sort the search by All Categories, Contacts, Opportunities, or Payments.

![](<https://assets2.modalsupport.net/1768475213644-unnamed \(85\).jpg>)

-   Click on any found Contacts, Opportunities, or Payments to quickly navigate to the page with the appropriate information.
-   Return to the search bar and repeat this as many times as you need to.

### Step 2: Add a Contact or Opportunity from the Search tab

-   In both the Contacts and Opportunities tabs, if your search doesn’t yield a result, you can create a new contact or opportunity right from here.
-   Click the “add” hyperlink that appears within the tab, and fill out the appropriate information for the contact or opportunity you wish to create.

![](<https://assets2.modalsupport.net/1768475233984-unnamed \(86\).jpg>)

### FAQs 




**Question:** What is the Universal Search feature?
**Answer:** The Universal Search feature allows you to quickly search for contacts, payments, and opportunities within . You can also create new entries if your search yields no results.




**Question:** Can I add a new contact or opportunity directly from the search results?
**Answer:** Yes! If your search yields no results in the Contacts or Opportunities tabs, you’ll see an “add” hyperlink. Click it to create a new contact or opportunity and fill in the required information.




**Question:** Can I repeat the search process multiple times?
**Answer:** Yes, simply return to the search bar and perform as many searches as needed.




**Question:** Will clicking a search result take me directly to the item?
**Answer:** Yes, clicking on any contact, opportunity, or payment from the search results will navigate you straight to the corresponding page with all its details.  




**Question:** Can I filter my search results to only show specific types of information?
**Answer:** Yes. Within the search popup window, you can use the left-hand sidebar to sort and filter your results by **All Categories**, **Contacts**, **Opportunities**, or **Payments**.




**Question:** What happens if I have two contacts with the same name?

**Answer:** Universal Search will display all matching results. You can use the **Email Address** displayed under the name in the search results to distinguish between the two and ensure you select the correct record.




**Question:** Can I search for a generic term to see all related data across different sections?

**Answer:** Yes. For example, if you search for a term like “test,” the system will pull up all matching records. You can then use the left-hand menu to see how many "test" entries exist specifically under Contacts, Opportunities, or Payments.
