Settings: Domains

In this article, we will walk you through the process of adding, configuring, and managing your domains.

## Adding a New Domain

To get started, follow these steps

-   Go to Settings, then Domains and URL Redirects tab.
-   Click **“+ Connect a domain”** to add a new domain.

### Using Domain Connect to set up your DNS records 

If you have your domain's DNS on GoDaddy/CloudFlare, you can use the Domain Connect feature. Domain Connect provides an automated and standardized process for connecting domains to services, eliminating the need for manual configuration.

Here is how it works:

-   Log into your account and go to Settings > Domains tab.
-   Click the "Add Domain" option.
-   Input your domain or subdomain
-   Click on the option “Add record manually”

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdvvjd9ZF4TA6eNZQeCsEv-Upx1Dd9M4Wq5pC3yRRHFL9ChWK9CXzY8bRkHL5SDIcDldjzNTSmiUIh_jvoyq57YVgdsV48rD2RPDUKpfUvhYfzHY5mdVfw91e8VTeeP0uark1T_ItyBU7L5PuqIUyAbets5?key=AEEQwWbkQK4T2jAB8vmTPQ)

-   You will see a pop-up with the DNS records, click on Verify records.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdMfR3O141tg3BIOixIK8DWzk6Fbu-4N5KLgVWMakoDjt19eWkwgb0bPSNfNb_yITHLy74bitHe85ZdSXgqQB2fVa7JUlsT5bAgmJkJx9b9WVqIKNnKAREfryuPIrq46TQUQxglWIM5Jd3P5k2Z_iabqNnC?key=AEEQwWbkQK4T2jAB8vmTPQ)

-   Once the DNS records are verified, you will be able to use your domain.
-   Run a test to verify the domain connection. Changes may take at least 30 seconds to reflect.

**Please note:** Manual DNS record addition remains an option if your domain provider is not supported.

## FAQ’s




**Question:** How do I add a new domain?
**Answer:** To add a new domain, go to Settings > Domains, click “+ Connect a domain,” and then follow the steps to add your domain and DNS records.




**Question:** How long does it take for my DNS records to propagate?
**Answer:** DNS records can take up to 24 hours to propagate. If you face any issues, we recommend waiting and trying again before seeking further assistance.




**Question:** Can I use Domain Connect to set up DNS records?
**Answer:** Yes, if your domain provider supports Domain Connect (such as GoDaddy or CloudFlare), you can automate the process of connecting your domain without needing manual configuration.




**Question:** What should I do if my domain provider is not supported by Domain Connect?
**Answer:** If your domain provider is not supported by Domain Connect, you can manually add the DNS records as instructed in the setup process.




**Question:** What kind of domains can I manage through this system?

**Answer:** You can manage both internal domains (purchased via ) and external domains (connected from third-party providers).




**Question:** What information is available for each domain?

**Answer:** You can view the domain name, subdomains, connected products, status, expiry/renewal date, and any required actions for both internal and external domains.




**Question:** How can I set a default page for my domain?
**Answer:** In the domain settings, click the edit option and select a default homepage. This is the page visitors will see when they navigate to your domain.  




**Question:** What is a 404 error page, and how do I set it?
**Answer:** A 404 error page appears when a user navigates to a broken or non-existent link. You can create a custom 404 error page and set it in the domain settings.  




**Question:** How do I remove a domain from my account?
**Answer:** To remove a domain, click the delete button next to the domain in the Domains tab, which will disconnect it from your account.




**Question:** Can I turn auto-renew on or off at any time?
**Answer:** Yes, as long as it’s before the domain’s renewal date.




**Question:** What happens if my auto-renewal fails?
**Answer:** The system will automatically retry up to two more times. If all attempts fail, you’ll receive alerts and can renew manually.




**Question:** Can I test if my domain is connected correctly?

**Answer:** Yes. After adding DNS records, run a **verification test**. Changes may take at least **30 seconds** to reflect.




**Question:** How can I troubleshoot domain setup issues?

**Answer:** If you encounter domain setup issues, a "Troubleshoot This Issue" link will appear when setting up your domain manually. This link provides direct access to helpful documentation for resolving common DNS issues such as multiple A records found, AAAA record conflicts, CAA block issuance, or CNAME/A record mismatches.




**Question:** How can I test if my domain is connected correctly?

**Answer:** After adding the DNS records, run a verification test. Changes may take at least 30 seconds to reflect.  




**Question:** Will my domain stop working if I turn off "Auto-Renew"?
**Answer:** It will continue to work until its **Expiry Date**. Once that date passes, if Auto-Renew is OFF and you haven't renewed manually, the domain will enter a "Grace Period" and eventually be released back to the public for anyone to buy. We recommend keeping Auto-Renew **ON** for your primary business domains.
