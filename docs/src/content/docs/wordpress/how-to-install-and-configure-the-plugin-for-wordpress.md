**How to Install and Configure the Plugin for WordPress**

In this guide, we'll walk you through the process of installing and configuring the plugin that integrates your WordPress website with the app management system. This allows you to seamlessly connect your WordPress site with your app to enhance functionality, whether it’s embedding funnels, managing chat widgets, or configuring other settings.

#### **Step 1: Install the Plugin**

**1- Log into your WordPress Website&Navigate to Plugins**:  
Begin by accessing the backend of your WordPress site. On the left sidebar, go to the **Plugins** section and click on **Add New**.

**2- Search for the Plugin**:  
In the search bar at the top right, type “Lead Connector.” This will bring up the official plugin.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfdplR1cnELQVA7FIngxtY7lwhpGMdclx0zl4lcFt4AGz9Xe-1ky6FB9xmyQsW2GiabenjPdQd88bNYSBYPTLTQ9q-3f71F2OwZuQ5Gnu-Iz8YIozqz1-lDCFL2xFH5uNdcBmUtsg?key=t3xKK1BPwiu3zYDTZ8HInw)

**3- Install the Plugin**:  
Click **Install Now** and allow the plugin to be installed.

Once the installation is complete, click **Activate** to enable the plugin on your website.

#### **Step 2: Configure the Plugin**

**1- Go to Lead Connector Settings**:  
In the left sidebar, you’ll see the **Lead Connector** option. Click on it to start the configuration.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXe3jJ4B20dckmFYv_dGGnDUQI81tSLJ8wCulMQcfxQlGuXlk6j-bnucOHf-M6CV9P3vEmRaF7JK12hfdILVeHVdGCl6d6V8bu3YVIWk31LnPWjIl0hIio1RjxiwpwwzddfLGlOLKA?key=t3xKK1BPwiu3zYDTZ8HInw)

**2- Obtain Your API Key**:  
Log into your account, go to **Settings**, and select **Business Profile**. Scroll to the bottom to find your API key.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeqDB08gnaDiCwp3yA9ItmOmBRqf-axjnwu8Nqm6HYAz69xOrxq3LQ1PsmkYAoU_khU5CdbUVnDmVL91ZHr29bIZfhgo682qNChzsPo03ipxZUE7_VXvyhFMxh-Qm1fVHDw8WtYcw?key=t3xKK1BPwiu3zYDTZ8HInw)

**3- Copy the API Key**:  
Click the **copy** icon next to the API key and go back to the WordPress plugin settings.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfffjErLUSxDUkGrR7C4LThBxyC4P5xe-A5c2eZwazU1hSsJz5q-023u1loVLM9QkC8h5tiNaa8ciy7G1qpS8TKu08a1D1CrEjBzyBv9dJ9k-lqrquluTyL6kmmgZ8lz11armZy?key=t3xKK1BPwiu3zYDTZ8HInw)

Paste the API key into the **Lead Connector** settings and click **Save** to establish the connection.

#### **Step 3: Enable the Chat Widget**

**1- Enable Chat**:  
In the Lead Connector settings, locate the **Chat Widget** option and enable it.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdEfAfmEqGOMlAYd4zTQzVetH8mVpPOM-Vf3SbfFfTBLMPIzGVE-NO6idtDpKtXkHxXZIG0bh7U4TFdAU609nrh1FJ_N9sKNlAZu1LEwkMQNP0RCj1jfAZnWQX3clQh9kIzzPVTGw?key=t3xKK1BPwiu3zYDTZ8HInw)

**2- Customize the Chat Widget**:  
Go back to the **Chat Widget** settings in your app, configure the design, update the language, and modify the widget appearance (like changing colors or pictures).

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXfftQsipvrd9gaoYyzCaCbwUurh0pA_qHxdAHCXBjlR-jCIoqOATCJaprqeWPOREU9IISfaA6rLse6Q6L2LIRkcBzDvI-NJvjEO2utPdy3E10GOGMX9n6OplLD4-uK_AhTDW8bE?key=t3xKK1BPwiu3zYDTZ8HInw)

**3- Save Changes**:  
Once you're happy with the settings, click **Save**. Your chat widget is now ready to appear on your website.  

#### **Step 4: Add a Funnel Page**

**1- Go to the Funnels Section**:  
In your app management system, navigate to the **Funnels** section to select the funnel you want to embed.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXeLL-RscHA84UFSZLZWfXA-M0EjVaOg3kaao5XyjUacKzNUEO8CoSMiHDLFUWEWJ3UfRltkqLOu9JIkep1428PNpcB2uUkkf-vysnCehCgTXOD-Db3RXFrm61Gpiee0s1OCgE-CSg?key=t3xKK1BPwiu3zYDTZ8HInw)

**2- Select the Funnel**:  
Choose the funnel you wish to add to your WordPress site and select the page (e.g., Lead Magnet) to embed.  

**3- Embed the Funnel**:  
In the WordPress plugin, go to the **Lead Connector** settings, click on **Funnels**, and then **Add New**. Choose the funnel page, adjust the settings, and hit **Save**.

![](https://lh7-rt.googleusercontent.com/docsz/AD_4nXdRuI0yoqRyWnJZy2eXIiAuxER6lHihNG6Txy7BpysWnfJXfBeY6ZQThXq3WT7f6m6Z_UpYdX_8y75tfLodUJaK8OpeHcDeOQDK3xQJUxLs6timbQWziLbrdbk7QsvcpckDWPjiwg?key=t3xKK1BPwiu3zYDTZ8HInw)

**4- Preview the Funnel**:  
After saving, you can preview your funnel page on your WordPress site to ensure it's properly embedded.  

#### **Step 5: Finalizing the Setup**

1.  **Edit the Funnel If Needed**:  
    If you need to make changes to the funnel, click **Edit** to be redirected back to the app management system. You can now adjust the funnel content as required.
2.  **Check the Live Chat**:  
    Finally, visit your website’s homepage, and you’ll see the chat widget pop up at the bottom right. Click on it to start interacting with the chat.

### **Conclusion**

By following these steps, you can successfully install and configure the plugin for seamless integration between your WordPress site and app. This enables smooth funnel embedding, chat widget management, and enhanced user experience.

### FAQs




**Question:** How do I install the plugin on WordPress?
**Answer:** Navigate to the **Plugins** section, click **Add New**, search for "Lead Connector," install it, and click **Activate**.




**Question:** What is the API key used for in the plugin?
**Answer:** The API key connects your WordPress site to the app management system, allowing seamless data sharing and integration.




**Question:** How can I customize the chat widget?
**Answer:** You can customize the chat widget by changing colors, text, and images from the **Chat Widget** settings in the app.




**Question:** How do I add a funnel page to my website?
**Answer:** Go to the **Funnels** section, select the funnel page, and use the **Lead Connector** plugin in WordPress to embed it.




**Question:** Can I edit the funnel once it's added to the site?
**Answer:** Yes, you can go back to the app management system and click **Edit** to make changes to the funnel.




**Question:** How do I ensure that the chat widget is properly displayed on my website?
**Answer:** After enabling and customizing the chat widget in the Lead Connector settings, visit your website’s homepage. You should see the chat widget appear at the bottom right of the page. Click on it to confirm it’s working as expected.




**Question:** Will the plugin automatically update funnel changes made in the app?
**Answer:** Yes. Any edits made to a funnel in the app management system will reflect on your WordPress site after saving and refreshing the plugin settings.




**Question:** Where can I find my API key to connect the plugin?

**Answer:** You can find your API key by logging into your account, navigating to **Settings**, and selecting **Business Profile**. The key is located at the bottom of that page.




**Question:** Why is my funnel page showing a 404 error on WordPress?

**Answer:** This usually happens if the "Permalink" or "URL Slug" in the plugin settings conflicts with an existing WordPress page.




**Question:** What should I do if I need to modify the content or layout of a funnel after I have already embedded it into my WordPress site?
**Answer:** You can easily make changes by clicking the "Edit" button within the plugin settings, which will redirect you back to the app management system. Once you adjust the funnel content there and save it, the changes will reflect on your WordPress site after refreshing the plugin settings.
