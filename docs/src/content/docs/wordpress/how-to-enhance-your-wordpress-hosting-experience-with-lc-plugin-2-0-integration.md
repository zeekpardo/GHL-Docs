LC Plugin 2.0 offers a refined approach to integrating essential digital marketing tools into WordPress sites. This version simplifies the user experience by streamlining the integration of communication tools and enhancing security protocols, making it a valuable asset for users seeking to manage marketing and communications more efficiently from their WordPress admin dashboard.

## Key Features & Benefits

LC Plugin 2.0 includes several key features that enhance website functionality and user experience:

-   **Effortless Integration of Communication Tools**: Easily embed phone numbers and forms into your website without the hassle of script tags or HTML, using a simple shortcode provided for any post or page.
-   **One-Click Email SMTP Setup**: Automatic SMTP configuration boosts email deliverability, ensuring emails reach the inbox and not the spam folder, which enhances user engagement and communication reliability.
-   **Enhanced Security**: The switch from API key authentication to OAuth offers a more secure and robust connection, protecting user data and interactions.

## Easy Installation and Setup

To start using LC Plugin 2.0, follow these steps:

1- Log into your WordPress admin dashboard.

![](<https://assets2.modalsupport.net/1764155657309-unnamed - 2025-11-26T191406.266.png>)

2- Navigate to 'Plugins' → 'Add New'.

3- Search for "LeadConnector", then click 'Install' & 'Activate'.

4- Open LeadConnector from the left sidebar in your dashboard.

5- Click 'Sign in with LeadConnector', select your\_\_CompanyName account, and you’re ready to go!

## FAQ




**Question:** What should I do if my email deliverability does not improve?**A1: Check that your domain is correctly connected and DNS settings are configured properly. Refer to support documentation for detailed troubleshooting tips.




**Question:** Can I still use my existing API key with LC Plugin 2.0?**A2: Existing API key connections will continue to function as usual, though transitioning to OAuth is recommended for better security.




**Question:** How can I add custom fields to the forms created with LC Plugin 2.0?
**Answer:** You can add custom fields directly through the plugin’s form builder interface, allowing you to customize data collection to meet specific needs.




**Question:** What happens if I change themes on my WordPress site?
**Answer:** LC Plugin 2.0 is designed to function independently of your theme, so theme changes should not impact the plugin’s functionality.




**Question:** Is there a limit to the number of forms or phone numbers I can embed using LC Plugin 2.0?
**Answer:** There are no limits within the plugin itself, though performance may vary based on your specific hosting environment and WordPress setup.




**Question:** Can I update plugins and themes to a version of my choice?
**Answer:** No, updates are only available for the latest versions provided by the plugin or theme developers.




**Question:** Can I downgrade plugins or themes?
**Answer:** No, once updated, you cannot revert to an older version through standard WordPress management tools.




**Question:** Can I manage plugins and themes directly from the WordPress dashboard as well as through external tools?
**Answer:** Yes, the WordPress dashboard remains a primary tool for managing plugins and themes, offering direct control over these components.




**Question:** How do I ensure the security of my LC Plugin 2.0 integration?
**Answer:** The security of your LC Plugin 2.0 integration is enhanced through the switch from API key authentication to OAuth, which offers a more secure and robust connection. Make sure to regularly update the plugin to benefit from the latest security patches.




**Question:** Does LeadConnector 2.0 require manual coding to add phone numbers or forms to my pages?
**Answer:** No. The plugin allows you to embed phone numbers and forms using a simple shortcode, removing the need for script tags or HTML




**Question:** Does the SMTP setup in the plugin conflict with other email plugins like WP Mail SMTP?

**Answer:** If you are already using an SMTP plugin, we recommend choosing one primary method to avoid conflicts. The LC Plugin’s one-click SMTP is designed to simplify life for users who don't have an existing setup, ensuring high deliverability specifically for your marketing automation emails.




**Question:** Can I use the plugin to manage my WordPress Themes as well as Plugins?

**Answer:** Yes. Within your dashboard, you can navigate to the 'Themes' section to view, activate, deactivate, or delete installed items to keep your site clean and efficient.
