**Creating and Managing Custom Objects - Overview 101**

Custom objects provide a flexible and powerful way to structure and manage specialized data for your business. Whether you need to track properties, pets, legal cases, or events, custom objects enable you to organize unique information in a manner that fits your specific needs.

## Step-by-Step Guide to Creating a Custom Object

### Step 1: Create a Custom Object

1.  **Navigate to Settings**: Go to your account and click on the **Settings** menu.
2.  **Select 'Objects'**: Under Settings, select **Objects**.
3.  **Add Custom Object**: Click on the **"Add Custom Object"** button.

![](https://assets2.modalsupport.net/1764172807052-image.png)

![](https://assets2.modalsupport.net/1764172899513-image.png)

You have the ability to rename Contacts, Opportunities, and Companies to your preferred terms (e.g., Clients, Deals, Organizations).

### Step 2: Add Details about the Custom Object

1.  **Define the Object Name**:  

    -   Add a **singular** name for the custom object (e.g., Pet).
    -   Add a **plural** name for the custom object (e.g., Pets).
2.  **Internal Name**:  

    -   The system will automatically generate an **internal name** for the object and its **primary field**. This can be edited by clicking the **"</>"** icon.
3.  **Primary Display Field**:  

    -   Add a **display field name** to identify the object clearly (e.g., Pet Name, Pet Identification Number).
4.  **Add an Icon**:  

    -   Choose an **icon** to help easily identify the custom object.
5.  **Object Description**:  

    -   Add a brief **description** explaining what the object represents (e.g., "This object tracks details about pets in the veterinary clinic").
6.  **Create the Custom Object**: Click on **"Create Custom Object"** to finalize.

![](https://assets2.modalsupport.net/1764172920313-image.png)

### Step 3: Editing a Custom Object

-   Once created, the custom object will appear on the same page.
-   Click on the object to open it for editing.
-   **Important Note**: The **primary display field** and **internal names** cannot be edited once created.
-   Objects are instantly updated across the entire platform, including navigation menus, filters, reports, and dashboards.

![](https://assets2.modalsupport.net/1764172957672-image.png)

### Deleting a Custom Object

1.  **Access the Object**: Click on the 3 dots next to the custom object name.
2.  **Select 'Delete'**: Choose **"Delete"** from the dropdown menu.
3.  **Confirm Deletion**: Type **"DELETE"** in the confirmation box and click **Confirm**.

![](https://assets2.modalsupport.net/1764172973242-image.png)

**Important Warning**: Deleting a custom object is irreversible. All records, associations, workflows, and custom fields related to the custom object will be permanently removed and cannot be recovered.

## FAQ's

**Question:** Who can create and edit custom objects?
**Answer:** Only admins can create, edit, and delete custom objects. Users have read-only access.

**Question:** Can I have more than 3 custom objects in one location?
**Answer:** No, you are limited to a maximum of 3 custom objects per location.

**Question:** What happens if I delete a custom object?
**Answer:** Deleting a custom object will permanently remove it, along with all associated records, fields, and workflows.

**Question:** Can I edit the internal name of a custom object?
**Answer:** The internal name and primary field name cannot be edited after the object has been created.

**Question:** What are some examples of when to use a custom object?
**Answer:** Custom objects are ideal for tracking specialized data such as properties, pets, event attendees, or legal cases that go beyond standard objects.

**Question:** How can I customize my SmartLists for Custom Objects?
**Answer:** You can customize your SmartLists by selecting filters, sorting options, and columns. This allows you to tailor the views to display the most relevant and up-to-date information for your team.

**Question:** Can I share SmartLists with my team?
**Answer:** Yes, you can share your SmartLists with team members by providing direct links. You can set permissions for each user, allowing either view-only access or editing rights.

**Question:** Are changes to custom objects reflected across the platform immediately?
**Answer:** Yes, updates to custom objects are instantly applied across navigation menus, filters, reports, and dashboards.

**Question:** Can I rename standard objects like Contacts or Companies?
**Answer:** Yes, you have the ability to rename Contacts, Opportunities, and Companies to your preferred terms, such as Clients, Deals, or Organizations.  

**Question:** What is a "Primary Display Field" and why is it important?
**Answer:** This is the name used to identify specific records within your custom object (e.g., "Pet Name" or "Case Number"). It is the main label you will see when searching for or linking to a specific record.
