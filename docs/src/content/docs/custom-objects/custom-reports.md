**Custom Report**

Welcome to the guide on Custom Reports, a feature designed to give you full control over creating, scheduling, and sharing reports tailored to your business needs. This step-by-step guide will walk you through using the Custom Report Builder, setting up recurring reports, and customizing each report page to suit your style and presentation preferences.

## **Why Custom Reports Matter**

Custom Reports enable you to:

-   **Save Time**: Automate report scheduling and delivery, ensuring consistency.
-   **Enhance Data Presentation**: Choose between live previews or downloadable PDF snapshots.
-   **Drive Better Decisions**: Present key metrics and insights clearly, supporting informed choices.
-   **Improve Professionalism**: Design visually appealing, multi-page reports with custom branding.

### **1\. What is a Custom Report Builder?**

The Custom Report Builder is a powerful tool with a drag-and-drop interface, allowing you to create reports from scratch or convert existing dashboards into a report. Each report page is fully customizable, including multi-page support, page list navigation, and personalized branding options.

### **Key Features:**

-   **Multi-Page Support**: Add multiple pages to cover comprehensive insights.
-   **Drag-and-Drop Widgets**: Easily add charts, tables, and elements to each page.
-   **Page Properties**: Customize colors, logos, headers, and add a professional cover page.
-   **Report Scheduler**: Automate report delivery on a recurring schedule.

### **2\. How to Access and Create a Custom Report**

**1\. Navigate to Custom Reports**:

-   Go to the **Reporting** section in your dashboard.
-   Select **Custom Reports**.

**2\. Create a New Report**:

-   Choose to start from scratch or convert an existing dashboard.
-   Click **Create New Report** to open the report builder.
-   If you already have a dashboard created, you can import it directly to save time and retain its structure.

![](https://assets2.modalsupport.net/1764083818451-1.jpg)

**3\. Add Widgets and Elements**:

-   Use the **drag-and-drop builder** to add charts, tables, and other widgets.
-   Arrange elements as desired, making sure they provide clear, valuable insights.
-   Add additional data or elements like visitor statistics, custom text, or embedded links to enhance the report's value further.

![](https://assets2.modalsupport.net/1764083877672-2.png)

### **3\. How to Customize Report Pages**

**1\. Multi-Page Setup**:

-   Click **Add Page** to create additional pages.
-   Use the **Page List Navigation** to rename, reorder, or delete pages.

**2\. Configure Page Properties**:

-   **Branding**: Add your logo, choose page colors, and configure headers.
-   **Cover Page**: Create a cover page for a professional look.

![](https://assets2.modalsupport.net/1764083896731-3.png)

**3\. Customize Widgets**:

-   Resize and reposition widgets for a clean layout.
-   Adjust data sources as needed to align with your reporting goals.

![](https://assets2.modalsupport.net/1770756402151-image7.png)

![](https://assets2.modalsupport.net/1770756418793-image1.png)

#### **4.  Add Themes to Reports:**

-   Open the Report Builder and click the **+** icon.
-   Go to the **Theme tab** in the left menu.
-   Select a predefined theme or create your own custom theme.
-   Themes apply seamlessly across the entire report or individual widgets.

![](https://assets2.modalsupport.net/1770756442232-image3.png)

**5\. Preview Your Report**:

-   Send a preview to your email to ensure layout and data accuracy.

### **4\. Scheduling and Sending Reports**

**1\. Set Up a Schedule**:

-   After saving your report, select **Schedule**.
-   Choose the frequency (e.g., daily, weekly, or monthly) and set the delivery times.
-   Configure the report to include specific date ranges, such as the last seven days or month-to-date. Add comparisons, such as previous periods, for more insightful analysis.

![](https://assets2.modalsupport.net/1764084154652-7.png)

**2\. Select Recipients**:

-   Add recipients, such as clients, stakeholders, or team members.
-   Ensure that email addresses are correct for successful delivery.

**3\. Date Settings:**

-   Enable the "Use Widget-Level Date Settings" option if you want each widget to follow its own date configuration. Otherwise, disable this option, and all widgets will follow the global date range.

![](https://assets2.modalsupport.net/1764084170889-8.png)

**3\. Delivery Options**:

-   Choose between **live previews** or **PDF snapshots** of the report.
-   Before scheduling, send a test email to confirm the layout and content.

![](https://assets2.modalsupport.net/1764084192310-9.png)

### FAQ

**Question:** What is the Custom Report Builder?
**Answer:** The Custom Report Builder is a fully customizable tool for creating, designing, and scheduling multi-page reports using a drag-and-drop interface.

**Question:** Can I add multiple pages to a report?
**Answer:** Yes, you can add and organize multiple pages, with each page customized to display specific insights.

**Question:** How do I brand my report?
**Answer:** Use the **Page Properties** settings to add a logo, set page colors, configure headers, and create a cover page.

**Question:** Can reports be scheduled for automatic delivery?
**Answer:** Yes, reports can be scheduled for recurring delivery to selected recipients based on your desired frequency.

**Question:** Can I add themes to my reports?
**Answer:** Yes, themes can now be added to reports for a cohesive look. Choose from predefined themes or create a custom theme that applies to the entire report or specific widgets.

**Question:** How can I ensure accurate date ranges in my reports?
**Answer:** Simply enable the "Use Widget-Level Date Settings" option to ensure each widget retains its specific date range when the report is scheduled.

**Question:** Does the custom reports feature provide a template library?
**Answer:** Yes, you can create, edit, and reuse custom report templates with the all-new Custom Reports Template Library.  


**Question:** Can I import an existing dashboard into a custom report?"
**Answer:** Yes, you can import an existing dashboard into the Custom Report Builder to save time and retain the structure of the dashboard in your report.

**Question:** How do I add widgets and data to a report?
**Answer:** Use the drag-and-drop interface to add charts, tables, text, embedded links, or visitor statistics to any report page. Widgets can be rearranged to create a clean, professional layout.

**Question:** What is the difference between a "Live Preview" and a "PDF Snapshot" in the scheduler?
**Answer:** When scheduling a report, you can choose to send a **Live Preview**, which provides an interactive link to the data, or a **PDF Snapshot**, which captures the report exactly as it appears at that moment for a permanent record.
