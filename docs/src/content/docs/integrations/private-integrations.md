Private Integrations enable robust, customized linkages between and third-party applications, allowing for tailored functionality.

**Benefits of Private Integrations**

-   **Simplicity:** Easily generate and manage tokens from your account settings.
-   **Security:** Restrict developer access to specific areas of your account.

**Private Integrations vs. API Keys** Private Integrations offer a secure, advanced alternative to API Keys, featuring:

-   **Enhanced Security:** Limited access for developers.
-   **Modern Technology:** Access to the latest API version (v2.0), supporting more features like webhooks.

**What differentiates Private Integrations from OAuth2 Access Tokens?**

Private Integrations are essentially static, fixed versions of OAuth2 Access Tokens.

**How can I use Private Integrations?**

Private Integration tokens are utilized in the Authorization header, similar to other Access Tokens. For instance, you can use the GET Contact request with the Private Integration Token in the Authorization header.

### **FAQ's**




**Question:** What is a Private Integration?

**Answer:** A Private Integration allows you to securely connect to third-party applications via APIs, offering more control over access and functionality.




**Question:** How are Private Integrations different from API Keys?

**Answer:** Private Integrations provide a more secure and feature-rich alternative to API Keys, with controlled access and support for newer API versions and webhooks.




**Question:** Who can create Private Integrations?

**Answer:** By default, all account admins can create and manage Private Integrations, but permissions can be adjusted at the user level in account settings.




**Question:** Where can I find Private Integrations?

**Answer:** Private Integrations can be managed under the 'Other Settings' section in your account settings. Ensure the feature is enabled in Labs if it's not visible.




**Question:** How do I create a Private Integration?

**Answer:** To create a Private Integration, navigate to settings, select 'Create New Integration,' name it, define permissions, and generate a token to share with trusted developers.




**Question:** How can I ensure my Private Integration token is secure?

**Answer:** Rotate your tokens every 90 days to ensure continued security. You can also expire old tokens immediately once new ones are in place.




**Question:** What should I do if my Private Integration token is compromised?

**Answer:** Immediately rotate the token by following the steps in the settings to prevent unauthorized access. Update the token in any third-party applications using it.




**Question:** Can I edit the permissions of a Private Integration without changing the token?

**Answer:** Yes, you can update the name, description, and permissions of a Private Integration without generating a new token.




**Question:** How do I delete a Private Integration?

**Answer:** If no longer needed, you can delete a Private Integration by selecting 'Delete' from the settings menu for the specific integration.




**Question:** How often should I rotate my Private Integration token?

**Answer:** It’s recommended to rotate your tokens every 90 days to maintain security and prevent unauthorized access.




**Question:** What’s the difference between rotating and expiring a token?

**Answer:** Rotate**: Generates a new token while the old one remains valid for a short overlap period (until updated).  

**Expire**: Immediately invalidates the existing token.




**Question:** Can Private Integrations access only specific areas of my account?

**Answer:** Yes. You can select precise permissions/scopes when creating or editing a Private Integration to restrict developer access to only what is necessary.




**Question:** How can I modify the permissions of an existing Private Integration without generating a new token?

**Answer:** You can change the name, description, and permissions of a Private Integration without generating a new token. To do so, simply update the scopes or permissions that the integration has access to in your account settings. Once updated, click "Update" to save the changes. The existing token will continue to work, and no new token will be created.




**Question:** How do I delete a Private Integration?

**Answer:** Navigate to the integration in settings and select “Delete” to remove it when it’s no longer needed.




**Question:** What is the relationship between Private Integrations and OAuth2 Access Tokens?

**Answer:** Private Integrations are essentially a **static, fixed version of OAuth2 Access Tokens**, providing the same level of modern technology without the need for complex authentication flows.




**Question:** Can I use Private Integrations to set up Webhooks?
**Answer:** Yes. Because Private Integrations utilize API v2.0, they fully support webhook subscriptions. This allows third-party apps to "listen" for events in your account (like a new contact created) in real-time.
